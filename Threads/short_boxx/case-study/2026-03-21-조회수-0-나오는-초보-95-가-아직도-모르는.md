# 조회수 0 나오는 초보 95%가 아직도 모르는

조회수 0 나오는 초보 95%가 아직도 모르는
'3초 멱살잡이' 공식 6가지

유튜브 깡통 차는 원인의 80%는
영상미가 아니라 첫 3초 대본이 구리기 때문입니다.

똑같은 아재의 썰이라도 첫 문장 하나로
조회수 차이가 100배 이상 벌어집니다.

시청자 손가락을 부러뜨릴 듯이 멈춰 세우는 6가지 대본 공식,
이것만 님들 대본에 복붙하세요.

[1. 충격 팩트] + [초보들의 착각 부수기]
사람들이 유튜브는 당연히 이래야 한다고 믿는 걸 박살 내야 합니다.

<예시>
"유튜브는 꾸준히 올리면 터진다?" → "무식하게 매일 올리는 놈들이 제일 먼저 망하는 이유"
"조회수 높이려면 편집을 잘해야 한다?" → "비싼 편집 학원 다닌 수강생이 깡통 찬 이유"

2. [구체적 숫자] + [압도적 결과]
뜬구름 잡는 소리 말고, 내 통장에 찍힌 숫자를 까발려야 신뢰감이 생깁니다.

<예시>
"쇼츠로 돈 버는 법" → "방구석에서 폰 하나로 월 1,200만 원 찍은 현실적인 방법"
"빚 갚는 노하우" → "15년 묵은 빚 10억, 캡컷 어플 하나로 다 갚았습니다"

3. [불안감 유발] + [치명적 손해 암시]
4050 가장들은 '이거 안 보면 당장 내 노후가 박살 나겠다' 싶을 때 클릭합니다.

<예시>
"무자본 부업 추천" → "이 부업, 당신의 퇴직금까지 다 날릴 수 있습니다"
"은퇴 후 현실" → "나이 50 넘어 이 짓거리하면 평생 자식 눈치 봅니다"
