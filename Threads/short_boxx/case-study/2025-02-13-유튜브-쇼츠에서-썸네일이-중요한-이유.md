# ✅ 유튜브 쇼츠에서 썸네일이 중요한 이유

✅ 유튜브 쇼츠에서 썸네일이 중요한 이유

원래 유튜브 쇼츠는 자동 재생이 핵심이라 썸네일을 지정할 필요가 없었어.

그런데 최근 탐색 기능(검색, 추천, 구독 피드 등)에서 쇼츠가 많이 노출되면서, 썸네일도 중요해졌지.

📌 왜 썸네일이 중요해졌을까?
1️⃣ 쇼츠 피드에서는 자동 재생 → 썸네일 영향 적음
2️⃣ 탐색 피드(검색, 추천, 구독)에서는 썸네일을 직접 클릭 → 썸네일 중요
3️⃣ 탐색 피드 유입이 증가 → 100만 돌파 가능성 UP!!

즉, 누군가 썸네일을 클릭했다는 건 그만큼 흥미를 느꼈다는 뜻!
탐색 피드를 통한 유입이 많아지면, 유튜브 알고리즘이 영상을 더 많이 추천해줘.
