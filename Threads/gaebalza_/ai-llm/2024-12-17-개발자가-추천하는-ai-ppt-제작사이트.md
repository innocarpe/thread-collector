# 개발자가 추천하는 "AI PPT" 제작사이트

개발자가 추천하는 "AI PPT" 제작사이트

1. 감마(gamma.app)
텍스트, 파일, URL연동으로도 제작가능
10개 슬라이드까지만 무료
AI텍스트, AI이미지 연동까지 빠르고 좋으나
텍스트 구성 배치가 조금은 아쉬움
2번 일해야하는 경우가 발생할 수 있음
피치가 생기고 사용자가 많이 이탈한 것 같음
⠀
2. 피치(pitch.com)
이미 AI로 제작된 샘플 템플릿이 많이 존재함
샘플은 마케팅, 스타트업, UX등 각 분야마다
존재해 디자인이나 구성 카피하기도 편함
Gpt를 활용해서 프롬포트 구성 후 반영하면 
조금 더 구체적으로 결과값이 산출되는데
텍스트 폰트 구성 배치 레이아웃이 감마보다는
깔끔하고 전문적이여서 좋음
(트랜디한 구성이라 편집할 부분이 적으나 한글폰트 미지원으로 파워포인트로 수정작업필요!)
⠀
댓글 이어서 pptai 

그 외 슬라이드고 등도 있는데 무료사용, 편집 쉬움을 제외하면 미리AI랑, 피치를 주로 볼 것 같음

⠀
3. 미리캔버스 AI(miricanvas.com)
이미 국내에서는 템플릿, 무료이미지 작업으로 많이사용하는지라 국내사용자들이 원하는 레이아웃 이미지등 구성이 간편하고 괜찮음
