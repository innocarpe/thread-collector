# 엑셀 단축키 고민하지마

엑셀 단축키 고민하지마 
상사한테 그만물어봐 이것만 우선 알자

1. F4 : 방금 작업 반복
2. F2 : 마우스 없이 셀 수정
3. F12 : 다른 이름 저장
4. Ctrl+1 : 셀서식, 테투리 한번에 설정
5. Ctrl+A : 표 전체 선택
6. Ctrl+D : 위 셀 내용 복사
7. Ctrl+Z : 되돌리기
8. Ctrl+S : 수시로 저장하기
9. Ctrl+ - : 불필요한 열 행 삭제
10. Ctrl+' : 수식 오류 확인
11. Ctrl+; : 금일 날짜 입력
12. Ctrl+Space : 열 전체 선택
13. Shift+Space : 행 전체 선택
14. Windows+D : 바탕화면바로보기
15. Alt+F4 : 종료
