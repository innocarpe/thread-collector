# 개발자가 이건 정말 괜찮다 느낀

개발자가 이건 정말 괜찮다 느낀 
"AI 숏폼관련 사이트" 추천
⠀
1. Makereels.ai
텍스트, 뉴스, 블로그 url입력시
정말 빠르게 영상, 주요텍스트를 자동으로
추출하여 숏폼으로 생성
(블로그는 N사는 안되는게 아쉬움)
뉴스는 정말 신박한데 저작권 이슈만 없으면
숏폼으로 수익화 괜찮아 보임
⠀
2. Klap.ai
아직 유명하진 않은 것 같은데
위 사이트보다 로딩 속도가 많이 늦으나
내가 원하는 url 입력시 핵심만 추려
영상과 텍스트를 추출 재가공
영상은 생성 안되나 롱폼위주업로드 하는 대상이 숏폼으로 따로 제작할 때 편리할 수 있으나
횟수 유료화, 한국어 속도 느림
