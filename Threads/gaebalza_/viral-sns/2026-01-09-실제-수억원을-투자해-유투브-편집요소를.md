# 실제 수억원을 투자해 유투브 편집요소를

실제 수억원을 투자해 유투브 편집요소를
줄여주는 툴을 활용하여 유투브수익화하는 라이브방송을 시청하게되었는데 
강의플랫폼에서 진행하는 방송이더군요

개발면에서 편의성부분에서 조금은 거슬렸지만
나름 사용자편의에 맞게 잘 제작된 것 같은데
강의팔이, 프로그램팔이로
부정적인식이 많이 깔린 것 같아 안타깝더라구요.

시장을 미리 알고 큰 금액을 투자해
해당 사업까지 확장시키려는 모델을 보면
대단하다는 생각이 드는데

돈이 되는거면 너네만 하면 되는거 아니냐의
의견들이 많아 놀라웠습니다.

해당 툴을 가지고 조금 더 쉽게 부업에 접근하는 개념으로 받아들이면 괜찮을 것 같은데

프로그램을 개발하는 입장에서는
너무 쳐내는 의견들만 있는 것 같아 
다양한 방법들을 풀어주는 케이스로
제가 제작을 한번 해봐야겠어요
