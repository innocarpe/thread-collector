# 트레이딩뷰에서 전략 짜는 걸 보는데 한계가 명확해

트레이딩뷰에서 전략 짜는 걸 보는데 한계가 명확해.

백테스트는 깔끔하게 나와도 실제 브로커 연결하면 문제투성이야.
슬리피지, 수수료, 주문 지연 같은 변수들이 수익을 갉아먹어.

진짜 시스템은 직접 만들어야 해.
API 연결부터 리스크 관리까지 모든 걸 컨트롤할 수 있어야 한다.
