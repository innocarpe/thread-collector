# "형, 전략 하나 완벽하게 만들면 될까요 여러 개 돌리는 게 나을까요"

"형, 전략 하나 완벽하게 만들면 될까요 여러 개 돌리는 게 나을까요"

둘 다 틀렸음.

하나 완벽은 불가능하고, 여러 개 돌려도 상관관계 높으면 하나랑 똑같다.

답은 "상관계수 낮은 3~5개". 
완벽할 필요 없고, 서로 다른 시장에서 작동하면 된다.
