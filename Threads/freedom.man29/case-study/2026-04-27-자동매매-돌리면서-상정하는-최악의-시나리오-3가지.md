# 자동매매 돌리면서 상정하는 최악의 시나리오 3가지

자동매매 돌리면서 상정하는 최악의 시나리오 3가지.

1. 거래소 API 먹통 — 포지션 열어둔 채로.
2. 내 서버가 다운됐는데 집 전기도 나감.
3. 전략이 구조적으로 깨져서 3개월 연속 MDD 갱신.

1번은 거래소별로 API 이중화. 바이낸스 죽으면 OKX에서 헷지 청산.
2번은 AWS + 로컬 이중 배포. 둘 다 죽으면 텔레그램 kill switch로 수동 전량 청산.
3번이 제일 어려움. 이건 기술로 못 막고 포지션 사이징으로 막는 수밖에.

최악을 상상 안 하는 시스템은 최악이 왔을 때 끝남.
