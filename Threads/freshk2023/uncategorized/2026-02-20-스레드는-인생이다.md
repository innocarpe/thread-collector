# 스레드는 인생이다

스레드는 인생이다.

나보다 성공한 사람의 생각을 비난,가르치려 든다
=> 스레드에서 차단 당함
=> 나보다 못한 사람들하고만 어울림.
=> 발전 없이 계속 퇴보함.

배울점이 있는분의 생각을 자주 본다.
=> 생각이 무의식중에 닮게 된다.
=>그 분을 닮게 되어 서로 친구가 된다.

그래서 부자는 더 부자가 된다.
스레드에서도 자연히 부자는 부자끼리 어울린다.
