# 혼자서 SNS 12개 계정 운영하고 있다

혼자서 SNS 12개 계정 운영하고 있다.

매일 콘텐츠 30개 이상 발행. 인건비 0원.

구조는 이렇다.
1. 크론잡이 정해진 시간에 AI 세션을 깨운다
2. AI가 브랜드별 페르소나에 맞춰 콘텐츠 생성
3. 자체 품질 검증 (7.0 미만이면 재생성)
4. API로 자동 발행
5. 실패 시 알림, 성공 시 리포트

처음 이걸 만드는 데 2개월 걸렸다. 지금은 하루 운영에 드는 시간이 15분. 알림 확인하고 가끔 프롬프트 튜닝하는 정도.

근데 여기서 가장 어려웠던 건 기술이 아니라,

각 계정의 "톤"을 잡는 거였다.

같은 AI가 만들어도 "직설적 멘토"와 "캐주얼 실험자"는 완전히 다른 글을 써야 한다. 이걸 프롬프트 하나로 구분하는 게 생각보다 정교한 작업이다.

기술 스택: 크론 + 쉘스크립트 + Claude API + Threads/IG API. 서버 비용 월 0원 (로컬 PC에서 전부 돌림).

1인 사업에서 콘텐츠 생산은 가장 큰 병목이다. 이걸 시스템으로 풀면 게임이 바뀐다.
