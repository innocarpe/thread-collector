# 구독 가격을 정할 때도 Claude를 활용했습니다

구독 가격을 정할 때도 Claude를 활용했습니다.

미국 구독 금액을 기준으로
각 국가의 실질 구매력(PPP) 을 반영해 가격을 다시 계산했고,
실질 데이터를 기반으로 175개국 가격을 자동 산출했습니다.

Apple price tier에 맞게 보정하고
기존 가격·국가별 매출 데이터를 함께 비교한 뒤,
App Store Connect API로 실제 구독 가격까지 반영했습니다.

이제는 국가별 가격을 감으로 정하지 않습니다.
구독 가격 전략도 데이터로 처리합니다.
