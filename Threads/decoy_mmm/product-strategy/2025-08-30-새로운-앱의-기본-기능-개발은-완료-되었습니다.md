# 새로운 앱의 기본 기능 개발은 완료 되었습니다

새로운 앱의 기본 기능 개발은 완료 되었습니다. 대략 4-5일의 시간이 걸렸는데 클로드 코드가 없었더라면 이렇게 짧은 시간 안에 개발을 끝내지 못했을 것 같네요. (생각보다 기능이 많고, 구현 난이도가 있는 앱입니다.)
파이어베이스, 애드몹 광고, 구독 등 기능만 넣으면 출시가 가능해서 다음 주에 업로드를 목표로 진행하고 있습니다.

바이브코딩하면서 느낀 점은
1. 향후 몇 년안에 얼마나 더 발전할지 무섭다.

2. 현재까지는 그래도 개발을 아예 모르면 중간에 어려움이 생긴다. (구조가 좀 복잡해지기 시작하면 클로드코드가 스스로 버그를 고치기 어려워하는 순간이 있음. 이럴 때 개발자가 직접 수정 방향을 제시해주면 확실히 버그를 잘 수정함)

3. 꽤 자주 버그 수정하면서 이미 잘되고 있는 기능까지 수정해버림 ㅠ

4. 디자이너와 협업에서 가이드에 맞춰서 개발해야되는 상황이라면 UI 개발에서 사용하기는 어려울 것 같다.
