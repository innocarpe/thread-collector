# fastlane + App Store Connect API + Claude 조합으로

fastlane + App Store Connect API + Claude 조합으로
앱스토어 관리를 거의 자동화했습니다.

- 바이너리 업로드
- 메타데이터 변경 (앱 이름, 설명, 업데이트 내용, 키워드)
- 구독 가격 변경
- 구독 상품 설명 수정

예전에는 이걸 전부 App Store Connect 웹에서 하나씩 수정했습니다.
25개가 넘는 언어를 지원하다 보니… 솔직히 너무 귀찮았습니다.

이제는 Claude에게
“이번 업데이트 내용 이거야, 반영해줘.”
라고 하면 끝입니다.

결과적으로 메타데이터 최적화를 훨씬 자주 하게 됐습니다.
할 수 있는 상태가 아니라, 안 할 이유가 없는 상태가 된 거죠.

1인 개발자에게 반복 작업 자동화는 생각보다 임팩트가 큽니다.
시간 절약도 크지만, 더 큰 건 심리적 허들이 사라진다는 점입니다.
