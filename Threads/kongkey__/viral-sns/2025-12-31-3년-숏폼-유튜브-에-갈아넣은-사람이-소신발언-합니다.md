# 3년 숏폼(유튜브)에 갈아넣은 사람이 소신발언 합니다

3년 숏폼(유튜브)에 갈아넣은 사람이 소신발언 합니다

1. 시청자는 당신의 일상에 1도 관심 없습니다.
2. '나'를 빼고 철저히 '시청자 이득'만 생각하세요.
3. 벤치마킹은 소재가 아니라 '구조'를 훔치는 겁니다.
(내용까지 베끼면 도둑질이고, 조회수도 안나옵니다)
4. 숏폼 자체에 집착하다 보면 조회수는 따라옵니다.
5. 데이터 분석 없는 업로드는 그냥 노동입니다.
6. 제 계정 들어가서 게시글 몇 개만 봐도 헛수고 안 하실 겁니다.
7. 그러니까 팔로우 하고 정보 받아보세요.
