# 제목 지을 때 그냥 짓지 마세요

제목 지을 때 그냥 짓지 마세요.
제발 '숫자'를 넣으세요

사람의 뇌는 추상적인 것보다 구체적인 숫자에 반응합니다

"숏폼 잘하는 법" (X) -> 너무 뻔함
"숏폼 잘하는 3가지 법칙" (O) -> 정리된 느낌.
"조회수 올리는 법" (X) -> 밋밋함.
"조회수 10배 올리는 법" (O) -> 이득이 명확함.

숫자는 신뢰도를 높이고, 클릭하고 싶게 만듭니다.
특히 '3가지', 'Top 5', '7일 만에' 이런 거 잘 먹힙니다.
제목 학원 다닐 필요 없습니다. 숫자만 잘 써도 반은 갑니다.
