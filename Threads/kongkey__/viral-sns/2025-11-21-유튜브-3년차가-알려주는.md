# 유튜브 3년차가 알려주는

유튜브 3년차가 알려주는
대본 쓸 땐 이정도는 체크해야한다

1. 어떠한 전개 방식으로 대본을 작성하는가?
2. 영상을 시작 하는 후킹멘트는 어떻게 시작하는가?
3. 후킹멘트 뒤 시작할 땐 어떤 전개로 대본을 풀어나가는가?
4. 하나의 대본 안에 몇개의 문장이 있는가?
5. 주로 어미는 어떤 어미들을 사용하는가? (-음, -지, -는데요 등)
6. 부정적인 단어는 어느정도까지 사용하는가? (마약, 자살, 살인, 비속어 등을 직접 사용했는가?)
7. 문장과 문장을 연결할 때 접속사를 사용하였는가? (-게다가 등)
8. 마지막 부분에 CTA(좋아요, 구독 유도)가 있는가?

진짜 이정도만 체크해도
생각보다 좋은 결과를 낼 수 있음
