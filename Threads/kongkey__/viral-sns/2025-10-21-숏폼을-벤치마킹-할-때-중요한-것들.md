# 숏폼을 벤치마킹 할 때 중요한 것들,

숏폼을 벤치마킹 할 때 중요한 것들,

간단하게 ‘대본’의 관점에서만 보더라도

1. 영상을 시작 하는 후킹멘트는 어떻게 시작하는가?
2. 후킹멘트 뒤 시작할 땐 어떤 전개로 대본을 풀어나가는가?
3. 하나의 대본 안에 몇개의 문장이 있는가?
4. 주로 어미는 어떤 어미들을 사용하는가? (-음, -지, -는데요 등)
5. 부정적인 단어는 어느정도까지 사용하는가? (마약, 자살, 살인, 비속어 등을 직접 사용했는가?)
6. 문장과 문장을 연결할 때 접속사를 사용하였는가? (-게다가 등)

이정도가 있다
농담아니라 나만 팔로우하고 따라와도
