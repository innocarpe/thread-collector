# 3년을 유튜브에 갈아넣은 사람이 발견한거

3년을 유튜브에 갈아넣은 사람이 발견한거

유튜브(숏폼)을 포기한 사람들은 공통점이 있음

1. 본인 콘텐츠에 본인 혼자만 재밌어함 (객관적이지 못함)
2. 남의 콘텐츠를 벤치마킹 안함
3. 영상 한두개 올리고 반응 없다고 좌절하다 포기함
4. 글쓰기의 중요성을 모르고 대본 대충 씀
5. 내 콘텐츠 자체를 분석하지 않음

유튜브를 포기한 사람의 95% 이상은
이 5가지 중 3가지 이상에 해당됨
그럼 이걸 어떻게 고치냐고?
내 프로필 와서 글 몇개만 읽어봐
