# 숏폼 하다가 99%가 떨어져 나가는 이유

숏폼 하다가 99%가 떨어져 나가는 이유?
이 5가지 떄문임

1. 메타인지 부족: 본인 영상이 재미없는 줄 모르고 알고리즘 탓만 함
2. 도파민 중독: 영상 만들러 들어갔다가 1시간 동안 남의 영상만 구경함
3. 문해력 부족: 기획, 대본 공부는 안 하고 편집 기술만 파고 있음
4. 실행력 제로: "준비되면 해야지"라며 평생 준비만 함
5. 지속력 부족: 영상 3개 올리고 반응 없다고 바로 접음
