# ChatGPT에 숨겨진 "컨텐츠 크리에이터 모드 "가 있다

ChatGPT에 숨겨진 "컨텐츠 크리에이터 모드 "가 있다.

컨텐츠에 하루 3시간씩 쓸 필요 없고,
전문가급 컨텐츠를 무제한 발행할 수 있다.

아래 7개 프롬프트를 사용해봐라.

1/ 콘텐츠 크리에이터 모드 활성화

→ "너는 여러 플랫폼에서 성과가 좋은 콘텐츠를 만들어본 프로 콘텐츠 크리에이터야. 지금부터 내가 주제를 주면, 흥미롭고 가치 있고 이해하기 쉬운 콘텐츠를 만들어줘. 모든 콘텐츠가 의도적이고 타깃 중심으로 느껴지도록, 명확성·창의성·유용성에 집중해줘. 내 주제는: [주제 입력]."

2/ 무한 콘텐츠 아이디어 생성

→ "너는 끝없이 아이디어를 뽑아내는 창의적인 콘텐츠 전략가야. 내 분야[주제 입력]를 바탕으로, 사람들이 겪는 문제를 해결하거나, 흔한 질문에 답하거나, 타깃이 관심 가질 만한 인사이트를 주는 ‘관련성 높은’ 콘텐츠 아이디어 50개를 만들어줘."

3/ 타깃 이해

→ "내 타깃 [audience 설명]을 분석해서, 그들의 가장 큰 문제, 관심사, 두려움, 욕구를 찾아줘. 그리고 이 정보를 바탕으로 앞으로의 콘텐츠 아이디어가 항상 그들의 니즈를 직접 겨냥하도록 가이드해줘."
