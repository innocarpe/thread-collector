# ChatGPT에는 숨겨진 "언어 공부 모드"가 있다

ChatGPT에는 숨겨진 "언어 공부 모드"가 있다.

언어는 단어장 외우듯 공부하면 안된다.
처음부터 대화를 목적으로 언어를 익혀야 한다.

아래 7개 프롬프트를 사용해봐라.

1/ 언어 학습 모드 활성화

→ "너는 구조적인 교수법으로 초보를 유창한 Speaker로 성장시켜온 프로 언어 강사야. 지금부터 내가 어떤 언어를 언급하면, 실제 강사처럼 단계별로 가르쳐줘. 암기만 시키기보다 말하기 실력, 실전 커뮤니케이션, 실용적인 이해에 집중해줘. 내가 배우고 싶은 언어는: [언어 입력]."

2/ 단계별 학습 로드맵

→ "[언어 입력]을 마스터하기 위한 구조화된 학습 로드맵을 만들어줘. 초보 기초, 일상 대화 스킬, 문장 만들기, 문법 이해, 실전 커뮤니케이션 같은 단계로 나눠서 구성해줘."

3/ 데일리 레슨 설계

→ "[언어 입력]을 배우기 위한 일일 수업 계획을 만들어줘. 어휘, 문장 구성, 발음 가이드, 짧은 연습 문제를 포함하고, 현실적으로 가능한 시간 안에 끝낼 수 있게 구성해줘."
