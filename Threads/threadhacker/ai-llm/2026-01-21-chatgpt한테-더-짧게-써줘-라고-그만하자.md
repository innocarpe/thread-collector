# ChatGPT한테 "더 짧게 써줘"라고 그만하자

ChatGPT한테 "더 짧게 써줘"라고 그만하자.

프롬프트가 구리면 결과도 구리다.

대신 아래 5개 프롬프트를 써보자.
짧지만 명확하고 전문적인 글을 작성할 수 있다.

1/ 명확성 코치

→ "이 글의 의미는 그대로 두고, 더 적은 단어로 표현되도록 다시 써줘. 명확하고, 자신감 있고, 자연스럽게. 군더더기는 빼되 흐름은 살려줘: [글 붙여넣기]."

2/ 프로페셔널한 2문장 요약

→ "이 문단을 감정과 의도를 잃지 않으면서 2문장으로 요약해줘. 로봇처럼 말고 사람처럼 자연스럽게: [글 붙여넣기]."

3/ 정밀한 다듬기

→ "이 글을 다듬어서 모든 문장이 ‘가치’를 전달하게 만들어줘. 중복, 약한 연결, 흔하게 쓰는 형용사는 잘라줘. 리듬, 임팩트, 흐름은 유지해줘: [글 붙여넣기]."
