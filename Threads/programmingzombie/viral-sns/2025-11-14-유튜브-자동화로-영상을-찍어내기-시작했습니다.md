# 유튜브 자동화로 영상을 찍어내기 시작했습니다

유튜브 자동화로 영상을 찍어내기 시작했습니다.
컨셉 생성 > 콘텐츠 생성 > 자동 업로드 과정을 모두 자동화 하였고, 모든 내용은 중복 콘텐츠 생성을 방지하며 콘텐츠 생성을 하게 하였습니다.

이제, 알아서 콘텐츠 생성을 시작하는것과 영상을 언제 공개할 것인지 정도만 처리하면 정말 손하나 까딱하지 않고 알아서 굴러가는 채널이 될 것 같습니다.

앞으로 어떤 결과가 있을지 궁금하네요 ㅎㅎ 하나의 재미있는 실험을 진행중입니다.
