# (채용 계획 전혀 없음 주의)

(채용 계획 전혀 없음 주의)
그냥 상상이지만 만약 제가 동료를 찾게 된다면, 나는 어떤 사람을 원할까? 라고 곰곰히 생각해보니 이런 생각이 들어서 기록차 작성해봅니다 ㅎㅎ

1. 실전 경험 (3년차 이상): 단순히 연차가 아닌, 문제 해결의 깊이가 어느정도 있는 분
2. A to Z 운영 경험: 기획부터 배포, 운영까지 하나의 프로젝트 사이클을 온전히 책임져 본 경험
3. 꾸준함의 증명: Github 1일 1커밋이나 기술 블로그 등, 스스로의 성장을 꾸준히 기록해 온 끈기
4. AI Native: AI 도구를 내 몸처럼 다루며, 코딩의 속도와 효율을 극대화하는 '바이브 코딩'이 가능한 분
5.  확장성 중심의 사고: 하나의 기능에 매몰되는 장인정신보다는, 서비스의 확장과 비즈니스 임팩트를 먼저 고려하는 유연함
6. 비즈니스 마인드: 기술 그 자체보다, 기술을 통해 얻을 수 있는 성과와 수익에 대한 건강한 욕심이 있는 분
