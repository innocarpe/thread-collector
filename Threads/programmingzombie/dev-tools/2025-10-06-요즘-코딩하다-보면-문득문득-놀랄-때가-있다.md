# 요즘 코딩하다 보면 문득문득 놀랄 때가 있다

요즘 코딩하다 보면 문득문득 놀랄 때가 있다.
터미널 창을 열고 "git pull 해줘"라고 입력하는 순간, 놀라서 내용을 다시 지웠다. 아, 여긴 그냥 터미널이었지. 다시 git pull을 입력했다.
참 신기하다. 나도 모르게 이렇게 자연스럽게 대화하듯 명령하고 있었다니. 정말 사소한 영역까지, 의식조차 하지 못한 채 코딩을 위임하고 있는 내 모습을 발견했다.
