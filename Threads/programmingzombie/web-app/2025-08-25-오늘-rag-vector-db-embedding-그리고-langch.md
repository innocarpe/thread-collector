# 오늘 RAG, Vector DB, Embedding, 그리고 LangChain을 활용한 서비스를 만들고 있습니다

오늘 RAG, Vector DB, Embedding, 그리고 LangChain을 활용한 서비스를 만들고 있습니다. 이 기술들에 대해서는 개념적으로는 알고 있었지만, 실제로 내 서비스에 적용해보는 것은 이번이 처음입니다.

이전 같았다면 학습하는 데만 일주일, 그리고 설계와 실제 서비스로 구현하는 데는 최소 2주 이상이 필요했을 것입니다.

그런데 놀랍게도 AI의 도움으로 하루 만에 학습부터 핵심 기능 구현까지 모든 과정을 진행할 수 있었습니다. 이론적 지식을 실제 동작하는 서비스로 빠르게 연결할 수 있다는 점이 정말 인상적입니다.

최근에는 새로운 웹서비스 아이디어가 떠올라 도메인 2개를 추가로 구입했습니다. 이렇게 빠르게 아이디어를 현실로 만들 수 있는 환경이 조성되니, 개발자로서 더없이 흥미진진하고 즐거운 시대를 살고 있다는 생각이 듭니다.

기술의 발전이 개발자의 생산성과 창의성을 얼마나 크게 향상시킬 수 있는지 몸소 체감하고 있습니다.
