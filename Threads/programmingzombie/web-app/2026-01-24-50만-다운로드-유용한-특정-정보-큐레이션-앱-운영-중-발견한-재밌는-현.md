# 50만 다운로드 유용한 특정 정보 큐레이션 앱 운영 중 발견한 재밌는 현상

50만 다운로드 유용한 특정 정보 큐레이션 앱 운영 중 발견한 재밌는 현상.

1. 앱이 좀 심심해 보여서 "니 입맛대로 골라봐" 하고 '커스텀 기능'을 넣어줌.
2. 업데이트는 많이들 했는데, 실제 사용률은 1%도 안 됨.
3. 결론: 사용자는 '설정의 자유'보다 '떠먹여 주는 편리함'을 압도적으로 선호함.

출시 전에는 "와, 기능 넣어주면 이것저것 세팅해서 엄청 잘 쓰겠지?" 하고 상상의 나래를 펼쳤는데 보기 좋게 빗나감.

사용자는 알아서 잘 떠먹여 주는 걸 제일 좋아하는 건가? 오늘도 데이터 보며 한 수 배웠음.

NORMAL - 일반 내가 제공해주는 큐레이션 콘텐츠 클릭
CUSTOM - 사용자가 직접 커스텀한 기능 콘텐츠 클릭

(결론) CUSTOM은 없었다.
(참고) 
- 해당 정보 수집은 신규 업데이트 기능으로 함께 넣음.
- 커스텀이라고 해봐야 거의 딸깍 수준으로 쉽게 수집되게 만듬.
