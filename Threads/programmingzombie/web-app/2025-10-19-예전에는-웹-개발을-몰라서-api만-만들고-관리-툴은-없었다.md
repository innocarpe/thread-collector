# 예전에는 웹 개발을 몰라서 API만 만들고 관리 툴은 없었다

예전에는 웹 개발을 몰라서 API만 만들고 관리 툴은 없었다. Postman으로 일일이 API를 호출해 데이터를 넣는 식의 원시적인 작업이 꽤나 많은 시간을 잡아먹고 있었다. 한눈에 볼 수 있는 대시보드도 없어 비효율의 연속이었다.
지금은 대부분을 어드민으로 구축했다. 모든 것이 한눈에 정리되어 있고, 자동화 기능으로 버튼 한 번이면 콘텐츠 생성과 마케팅이 해결된다.
여전히 웹 개발은 못 한다. 하지만 AI가 있다면 무엇이든 가능하다.
