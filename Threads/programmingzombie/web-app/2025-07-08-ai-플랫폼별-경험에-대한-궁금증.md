# AI 플랫폼별 경험에 대한 궁금증

AI 플랫폼별 경험에 대한 궁금증

최근 AI 개발 도구들을 사용해보면서 흥미로운 패턴을 발견했습니다. 웹 개발에서는 AI가 정말 "알아서 잘 딱 깔끔하고 센스있게" 해주는 반면, 네이티브 개발에서는 조금 다른 느낌을 받고 있어요.

웹 개발에서의 AI 경험
SaaS 서비스를 만들 때 웹에 대한 지식이 거의 없는 상태였음에도 불구하고, AI가 제가 상상했던 것 이상으로 매끄럽게 구현해줬습니다. 복잡한 로직이나 UI/UX까지도 자연스럽게 처리해주더라고요.

네이티브 개발에서의 차이점
반면 안드로이드 개발의 경우, 제가 이미 많은 경험과 지식을 가지고 있어서 개발 자체는 수월하다고 느끼지만, 완전 초보 상태에서 AI에게 맡기기에는 웹 개발보다 난이도가 있을 것 같다는 생각이 듭니다.

궁금한 점들
Flutter나 React Native 같은 크로스 플랫폼에서는 어떤 경험을 하셨나요?
다른 분들도 플랫폼별로 AI의 성능 차이를 느끼셨는지 궁금합니다.
다양한 바이브 코딩 경험이 궁금하네요

디자인도 생각해봤을때 웹쪽이 정말 화려하고 멋진걸 잘 만들어내는 반면, 네이티브 작업을 할 땐, 물론 제가 하는것보다야 훨씬 나은 디자인을 만들어주지만 상대적으로 웹과 비교하면 훨씬 퀄리티가 떨어진다는 느낌을 받았습니다. 🤔

그렇다면 화려하고 멋진 앱 서비스를 위해 "Flutter, React Native로 한번 도전해봐야하나"라는 생각이 드네요
