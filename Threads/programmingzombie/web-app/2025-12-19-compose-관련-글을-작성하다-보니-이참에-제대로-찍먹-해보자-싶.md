# Compose 관련 글을 작성하다 보니 '이참에 제대로 찍먹 해보자' 싶어 Claude Code를 이용해 기존 Base Template을 마이그...

Compose 관련 글을 작성하다 보니 '이참에 제대로 찍먹 해보자' 싶어 Claude Code를 이용해 기존 Base Template을 마이그레이션하고 신규 앱 제작에 도입해 봤습니다.

결론부터 말하자면 아주 만족스럽습니다. 예전에 테스트했을 때는 기존 네이티브(XML/UIKit) 방식이 Flutter나 React Native에 비해 AI가 짜주는 UI 퀄리티가 현저히 떨어져서 사용하지 않았었거든요. 그런데 Compose로 시도해 보니 Flutter와 유사한 수준으로 세련된 UI를 곧잘 만들어냅니다. 앞으로 순수 Android 프로젝트는 Compose를 적극적으로 도입하게 될 것 같습니다.
