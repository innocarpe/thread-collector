# 요즘 나는 어드민 페이지 만드는 일에 푹 빠져있다

요즘 나는 어드민 페이지 만드는 일에 푹 빠져있다.

8년이 넘는 시간 동안 API 구축과 서버 운영, 앱 개발에는 많은 경험을 쌓아 어느 정도 자신 있게 할 수 있게 되었다. 하지만 정작 매번 피하고 싶었던 업무가 바로 웹 개발이었다. 

가장 큰 이유는 단순했다. 잘 몰랐기 때문이다. 여기에 '웹 개발로는 돈 벌기 어렵다'는 막연한 선입견도 한몫했다. 그런데 이제 웹 개발을 다룰 수 있게 되면서 정말 많은 것들이 달라졌다.

가장 큰 변화는 업무 효율성이다. 예전에는 각종 지표를 분석하기 위해 수작업으로 데이터를 뒤지고 정리하는 노가다성 작업을 반복했다. 지금은 이 모든 것을 어드민 페이지로 구축해 한눈에 볼 수 있게 만들었다.

새로운 앱을 만들 때마다 API를 직접 호출해서 데이터를 세팅하던 번거로운 작업도 이제는 어드민을 통해 손쉽게 처리하고 있다. 클릭 몇 번으로 예전에 코드를 직접 건드려야 했던 일들이 해결된다.

더 흥미로운 건 사고의 확장이다. 이제는 AI를 적용해서 이 모든 과정을 자동화할 수 있을지도 모른다는 생각까지 하게 되었다. 어드민 페이지가 단순한 관리 도구를 넘어서 지능형 업무 파트너가 될 수도 있다는 상상을 하고 있다.

정말로 1인 개발자가 할 수 있는 범위가 크게 넓어졌다. 예전에는 팀이 있어야 가능할 것 같았던 일들을 지금은 혼자서도 하나씩 처리해나가고 있다. 나 같은 1인 개발자에게는 정말 좋은 세상이 아닌가 싶다.
