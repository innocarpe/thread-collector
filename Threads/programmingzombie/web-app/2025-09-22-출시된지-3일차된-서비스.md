# 출시된지 3일차된 서비스

출시된지 3일차된 서비스
마켓을 유심히 살펴보다 꿀빨고 있는 서비스들을 몇개 발견하고 경쟁으로 참전.
3일차 지표인데, 생각보다 괜찮은듯

1. 초기 부터 높은 eCPM
2. 아직 광고가 거의 나가지 않았음에도 자연유입 꽤 됨

광고 제대로 들어가면 꽤나 성장이 예상되는 서비스로 추측됨. (물론 아닐 수도 있음)

당시 동시에 출시한 3개의 앱은 핵심이 유사한 서비스이기에, 하나의 지표가 괜찮게 나오면 다른 앱들도 비슷한 성장이 될 확률이 높음. 아이디어가 없다면 꾸준히 습관적으로 시장을 관찰하는것 또한 많은 인사이트를 얻을 수 있음.

이 서비스는 마케팅에 당분간 꾸준히 투자할 예정이고, 불과 이틀전 마켓을 유심히 살피다 꿀빨고 있는 서비스를 다수 발견함. 2차 경쟁자로 참전하러 이번에는 안드로이드 앱으로 진입 해보려고 개발 진행중임. 이번에도 마찬가지로 다수의 앱을 병행으로 제작할 예정.
