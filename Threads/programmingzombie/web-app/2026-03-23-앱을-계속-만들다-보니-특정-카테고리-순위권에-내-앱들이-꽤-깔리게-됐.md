# 앱을 계속 만들다 보니, 특정 카테고리 순위권에 내 앱들이 꽤 깔리게 됐다

앱을 계속 만들다 보니, 특정 카테고리 순위권에 내 앱들이 꽤 깔리게 됐다.
그러다 보면 자연스럽게 보인다. 비슷한 카테고리, 비슷한 순위의 앱들이 대략 얼마나 벌고 있는지.

이런 마켓 감각이 생기는 것 자체가 거대한 동기부여가 된다. 어디에 자리를 잡으면 얼마짜리 자리인지 보이기 시작하니까.

세상에서 아주 작은 부분일 뿐인 앱 마켓 안에서도, 엄청난 돈이 흐르고 있다.
