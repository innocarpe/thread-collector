# 클로드 코드 데스크탑 앱 업데이트가 있었네요

클로드 코드 데스크탑 앱 업데이트가 있었네요.

1. 서버 프리뷰 - 데스크탑 앱 안에서 개발 서버를 실행하고 앱을 바로 미리볼 수 있습니다. 콘솔 로그를 읽고 에러를 잡아가며 자동으로 반복 수정해 줍니다.

2. 로컬 코드 리뷰 - 코드 푸시 전에 Claude가 인라인으로 버그와 이슈를 코멘트해 줍니다. 본격적인 코드 리뷰 전에 미리 문제를 걸러낼 수 있습니다.

3. PR 모니터링 - PR을 올리면 백그라운드에서 CI를 추적하면서 실패 시 자동 수정, 통과 시 자동 머지까지 처리해 줍니다. Claude가 알아서 관리하는 동안 다음 작업에 집중할 수 있습니다.

4. 세션 이동 - CLI에서 작업하던 세션을 데스크탑 앱으로, 다시 웹이나 모바일로 자유롭게 이어서 작업할 수 있습니다.

터미널 환경이 어색한 분들도 앱으로 쉽게 접근할 수 있게 한 것은 물론, 개발 워크플로우 전반의 생산성을 끌어올리려는 의도가 분명한 업데이트입니다. 개발자와 비개발자 구분 없이 누구나 코딩을 할 수 있게 하려는 노력이 돋보입니다.

지금과 같은 속도로 계속 발전해 나간다면, 짧은 소견으로는 2~3년 이내에 개발이라는 영역이 누구나 마음만 먹으면 도전할 수 있는 분야가 될 것 같습니다.(사실 생각해보니 지금도 그러한 것 같습니다)
