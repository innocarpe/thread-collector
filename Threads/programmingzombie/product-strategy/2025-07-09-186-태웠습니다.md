# $186 태웠습니다

$186 태웠습니다. Flutter, React Native, Android 중 AI 코딩의 승자는?

하나의 동일한 앱 기획서를 바탕으로 Flutter, React Native, Android 각각 별도 프로젝트를 생성하여 Claude Code의 Plan Mode를 활용해 MVP를 제작했습니다.

실험 조건:
- 동일한 상세 기획서를 각 프로젝트 폴더에 복사
- 빈 프로젝트에서 시작 (Android는 추가로 기존 템플릿으로도 테스트)
- 동일한 프롬프트로 Claude Code에 개발 요청

🏆 결과 순위
1위 Flutter - 압도적 속도와 완성도
2위 React Native - 준수한 성능과 품질
3위 Android - 예상 외로 아쉬운 결과

상세 분석은 댓글에서 확인하세요 👇

상세 분석 결과

🥇 Flutter
제작시간: 약 5분, 압도적으로 빠른 속도
디자인: 당장 프로덕션에 사용 가능한 수준의 UI/UX
완성도: 기획서 요구사항을 거의 완벽하게 구현, 약간의 보완만 필요

🥈 React Native
제작시간: 약 8분, 비교적 빠른 개발 속도
디자인: Flutter 대비 약간 아쉽지만 충분히 괜찮은 수준
완성도: 초기 실행 오류가 있었으나 수정 후 실용적인 수준

🥉 Android
제작시간: 약 15분, 다른 플랫폼 대비 2~3배 소요
디자인: 즉시 사용하기 어려운 수준의 UI
완성도: Gradle 버전 이슈 등으로 인한 빈번한 오류, 기획서 반영도 30% 미만

추가로, Android의 경우 기존에 잘 활용하던 템플릿 코드가 있어서 빈 프로젝트가 아닌, 많은 것들이 구현되어있거나 유용한 유틸 클래스가 존재하는 템플릿 프로젝트에서 동일한 프롬프트로 실행시켜봤으나 결과는 마찬가지였습니다.

결론)
현재 Claude Code는 Flutter > React Native > Android 순으로 최적화 되어있는 것으로 보이며, 플랫폼에 대한 지식이 동일한 조건에서의 MVP 제작이 목적이라면 Flutter를 추천드립니다.

주의)
무조건 위의 실험만으로 Flutter가 좋다라고 단정지을순 없습니다. 하지만 제가 한 실험 기준, Claude Code와 함께 했을때 "알잘딱깔센"은 Flutter가 1위 였습니다.
