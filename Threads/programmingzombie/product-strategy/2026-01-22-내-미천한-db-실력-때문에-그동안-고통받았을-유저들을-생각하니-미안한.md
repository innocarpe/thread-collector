# 내 미천한 DB 실력 때문에 그동안 고통받았을 유저들을 생각하니 미안한 마음이 든다

내 미천한 DB 실력 때문에 그동안 고통받았을 유저들을 생각하니 미안한 마음이 든다. 

예전에는 빨랐던 서비스가 데이터가 쌓이면서 느려졌는데, API 응답이 11초나 걸리는 줄도 모르고 방치하고 있었다. 

뒤늦게나마 이를 인지하고 바이브 DB를 통해 최적화한 결과, 속도를 66배 단축하는 데 성공했다.
