# 오늘은 Claude Code에게 동시에 7개의 프로젝트를 돌려보고 있습니다

오늘은 Claude Code에게 동시에 7개의 프로젝트를 돌려보고 있습니다.

1. 각 앱들의 컨셉과 기능에 대한 기획서를 최대한 상세히 작성한다.
2. 작성한 기획서의 내용을 각 프로젝트에 md 파일로 넣어두고 Cladue Code에게 참고하여 작업 플랜을 세우라고 Plan Mode로 실행.
3. 도출된 계획 리스트를 검토하고 작업 스타트.
4. 잠시 쉬다온다.
5. 작업이 완료된 순으로 실행시켜보며 수정할 부분이나 미흡한 부분이 있는지 검토한다.
6. 검토 과정중 더 필요한 작업이 있거나 오류가 있는 경우 상세하게 명령을 전달하고 다시 스타트.
7. 적절한 수준의 서비스가 완성될때 까지 5, 6번을 반복한다.
8. 가끔 AI 가 해결하지 못하는 부분이나 이상하게 만든 코드는 직접 수정

한번에 7개의 프로젝트를 돌리는건 처음인데 어떤 결과물이 나올지 흥미진진합니다.

기타사항
- 7개를 동시에 돌리면 Opus 제한이 빨리 찾아온다.
- 당연히 토큰 소비도 엄청 빨리 된다.
