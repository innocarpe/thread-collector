# 오늘 Claude Code 토큰이 부족해서 Max $200 플랜을 하나 더 결제했다

오늘 Claude Code 토큰이 부족해서 Max $200 플랜을 하나 더 결제했다. 이제 Max $200짜리를 두 개 돌리는 셈이다.

문득 이런 생각이 들었다. 창업이나 회사 운영을 간접적으로라도 경험해보고 싶다면, Claude Code Max $200이 꽤 괜찮은 입문 코스가 되지 않을까. 따지고 보면 월급 $200을 주고 똑똑하지만 손이 많이 가는 직원 한 명을 고용하는 것과 다를 게 없는셈이다.

월급은 매달 꼬박꼬박 나간다. 아무것도 안 시키면 그대로 월급 루팡을 모시는 꼴이지만, 어떤 방향성을 가지고 어떻게 부리느냐에 따라 $200이 $2,000의 수익으로 돌아올 수도 있다.

$200을 줬으면 그 이상을 벌어와야 한다. 방향을 잡아주고, 일을 시키고, 관리하는 건 온전히 내 몫이다. 내버려 두는 순간, 나는 매달 $400씩 월급을 주면서 아무것도 안 하는 천재 두 명을 모시고 있는 셈이 된다.
