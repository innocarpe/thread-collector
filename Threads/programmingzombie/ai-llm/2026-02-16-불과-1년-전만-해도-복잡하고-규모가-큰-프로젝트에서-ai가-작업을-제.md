# 불과 1년 전만 해도, 복잡하고 규모가 큰 프로젝트에서 AI가 작업을 제대로 수행하려면 관련 클래스나 메서드를 직접 찾아서 참조로 넘겨줘야 했다

불과 1년 전만 해도, 복잡하고 규모가 큰 프로젝트에서 AI가 작업을 제대로 수행하려면 관련 클래스나 메서드를 직접 찾아서 참조로 넘겨줘야 했다. 그래야 좀 더 맥락을 정확히 이해하고 원하는 결과물을 잘 만들어냈다.

지금은 다르다. 구체적인 클래스명이나 범위를 지정하지 않아도, 자연어 명령만으로 AI가 알아서 필요한 클래스를 찾아내고 참조해서 작업에 반영한다.

오히려 이제는 내가 직접 참조할 클래스를 떠올리고 찾는 시간이 더 오래 걸린다. AI에게 정확한 컨텍스트를 넘겨주려는 인간의 노력이 되레 병목이 되는 셈이다.

AI가 발전할수록, 사람이 개입하는 구간이 병목이 되어가고 있다.
