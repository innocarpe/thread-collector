# Claude Code 아직 멍청함이 해결이 안된 것 같은데 임시 방편으로 Opus 4

Claude Code 아직 멍청함이 해결이 안된 것 같은데 임시 방편으로 Opus 4.1 -> 4.0 으로 다운그레이드 하여 사용하면 성능이 다시 좋아졌다고 하는 제보가 보입니다. 

다운그레이드 하는 명령어는 이렇게 입력하면 되는 것 같고
/model claude-opus-4-20250514

의견이 분분하긴한데, 최근 퍼포먼스가 확연하게 떨어진것 같아서 Opus 4.0 으로 다운그레이드 하여 사용해보려고 합니다.
