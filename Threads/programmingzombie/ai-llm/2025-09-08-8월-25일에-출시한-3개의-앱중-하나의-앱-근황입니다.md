# 8월 25일에 출시한 3개의 앱중 하나의 앱 근황입니다

8월 25일에 출시한 3개의 앱중 하나의 앱 근황입니다. 해당 앱은 ChatGPT API 래핑하여 뾰족한 문제를 해결해주는 서비스. GPT에게 사용자 데이터를 공유한 덕분에 GPT-5를 하루 100만 토큰까지 무료로 사용할 수 있는데, 생각보다 유저들의 API 요청이 많아서 일일 무료 한도를 초과하며 과금이 시작됐네요.

GPT-5로는 비용 부담이 커서 GPT-5 mini로 다운그레이드했습니다. 서비스 품질은 떨어지겠지만 수익성 측면에서는 개선될 거라 봅니다. 

결국 서비스 운영에서 지속 가능하게 하기 위해 가장 중요한 건 '수익 - 비용 = 순수익'을 만드는 것이더라고요. eCPM 개선 노력과 함께 사용량에 따라 API 모델도 유동적으로 조절하면서 지속 가능한 서비스로 만들어가려 합니다.
