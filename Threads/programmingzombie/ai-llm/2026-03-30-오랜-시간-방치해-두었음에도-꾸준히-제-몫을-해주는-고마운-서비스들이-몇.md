오랜 시간 방치해 두었음에도 꾸준히 제 몫을 해주는 고마운 서비스들이 몇 개 있습니다. 특별히 기름칠을 하지 않아도 잘 굴러가기도 했고, 솔직히 귀찮은 마음도 커서 그대로 두었죠.

그러다 최근 Claude Code 토큰이 좀 남길래, 예전 같으면 리소스 문제로 엄두도 못 냈을 대규모 업데이트를 시도해 봤습니다. 놀랍게도 기획부터 출시까지 단 몇 시간 만에 모든 작업이 끝났습니다.

앞으로는 우선순위에서 밀려 구석에 먼지 쌓여가던 예전 프로젝트들도 훨씬 더 쉽게, 다시 반짝이게 만들어 줄 수 있을 것 같습니다.
