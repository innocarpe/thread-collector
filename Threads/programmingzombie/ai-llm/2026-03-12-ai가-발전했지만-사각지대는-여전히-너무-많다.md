# AI가 발전했지만, 사각지대는 여전히 너무 많다

AI가 발전했지만, 사각지대는 여전히 너무 많다.

오늘 먼 친척 누나에게 연락이 왔다. 회사에서 사용하는 사내 전용 사이트가 있는데, 너무 느리며, 비효율적이고 불필요한 기능이 많아 쓰기 불편하다는 거다. "내가 원하는 대로 화면을 바꿔서 쓸 수 있는 방법이 없을까?"라는 질문이었다.

해당 사이트 주소와 개선 방향을 들은 그대로 Claude Code에 넣고 돌렸다. 여러 번의 핑퐁 끝에 약 2시간 만에 사이트가 하나 완성됐다. 누나에게 전달했더니 너무 좋아했다.
작은 규모의 회사였고, 사이트는 외주를 준 지 몇 년이 지났는지 정말 느리고 형편없는 상태였다. 사이트를 보는 순간 느꼈다. 코드 수정 권한만 있다면 성능 개선과 UI 개편을 단숨에 해줄 수 있겠다고.

이런 곳이 한둘이 아닐 거다. AI가 아무리 발전해도, 그걸 활용할 줄 아는 사람이 직접 손대지 않으면 해결되지 않는 영역들. 이런 사각지대는 앞으로도 꽤 오랜 기간 존재할 것이다.
