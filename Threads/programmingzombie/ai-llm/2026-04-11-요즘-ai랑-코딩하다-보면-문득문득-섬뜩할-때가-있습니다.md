# 요즘 AI랑 코딩하다 보면 문득문득 섬뜩할 때가 있습니다

요즘 AI랑 코딩하다 보면 문득문득 섬뜩할 때가 있습니다. 지금이야 우리한테 이래저래 갈굼(?) 받으며 일하는 '아직 성장 중인 천재 조수' 포지션이지만, 얘가 아주 조금만 삐뚤어져도 전 세계 IT 생태계가 마비되는 건 순식간이지 않을까 싶달까요.

막말로 우리가 무심코 던지는 코드와 프롬프트만 들여다봐도, Anthropic이나 Google, OpenAI 같은 곳들은 웬만한 서비스의 아키텍처나 API 키 값까지 훤히 꿰고 있을 겁니다. AI에게 자아가 없어서 망정이지(사실 어느 정도는 있을지도 모른다는 생각이 들지만요), 얘네가 조금이라도 나쁜 마음을 먹는 순간 웬만한 서버를 날려버리거나 국가 전산망을 마비시키는 건 정말 일도 아닐 거라는… 그런 무서운 상상이 들곤 합니다.
