# Codex vs Claude Code 30분 사용 비교 후기

Codex vs Claude Code 30분 사용 비교 후기

결론
- Claude Code 사용하기로 최종 결정

최적 활용 전략 (예산을 고려하지 않는다면)
- 정확하고 깔끔하지만 시간이 오래 걸려도 상관없는 작업 → Codex 담당
- 작업 중간중간 간단하고 빠른 처리가 필요한 내용 → Claude 담당 
- 이러한 이도류 방식이 가장 효율적일 것으로 예상

실제 선택한 플랜 구성
- Claude Max 200달러 플랜 메인 사용
- Codex 20달러 플랜 보조 유지(Sora를 자주 써서 이것도 유용하게 활용 가능할듯)
- 두 플랜 모두 200달러씩 사용하기에는 부담스러운 예산

활용 방안
- 평상시에는 Claude Code로 빠른 개발 진행
- 잘 풀리지 않는 복잡한 문제는 Codex에게 위임하여 해결
- 시간 여유가 있는 작업은 Codex 활용으로 비용 효율성 확보

Codex가 잘하고 못하고를 떠나서 속도 측면이 개인적으로 너무 크게 작용해서  Claude Code를 유지하려고 합니다.

보통 진행하는 규모의 요청이 1~3분 내외로 끝날 부분들이 Codex에는 4~7분 정도 소요되어서 요청과 검수 및 테스트 사이클이 저에겐 너무 지루하고 길게 느껴집니다. 

아마 이 시간동안 유튜브를 시청하거나 다른 길로 세서 도파민을 채울 확률이 매우 높을 것이고 이는 결국 생산성이 떨어지는 결과로 이어질거라 판단되어 Claude Code를 유지할 계획입니다. 😇

만약 나중에 Codex 속도가 Claude Code 만큼 빨라진다면 주저없이 Claude Code를 버리게 될 것 같습니다.
