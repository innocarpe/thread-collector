# 요즘 Claude Code가 엄청 빨라진 것 같은 느낌이 든다

요즘 Claude Code가 엄청 빨라진 것 같은 느낌이 든다. 예전에는 분석하고 코드를 생성하는 동안 기다리는 시간이 길어서 다른 데로 새기 일쑤였는데, 이제는 명령을 내리자마자 결과물이 나오는 것처럼 느껴진다. 마치 "넌 닥치고 명령이나 해"라고 외치는 것 같달까.

사실 Claude Code 자체가 획기적으로 빨라진 건 아니다. 비결은 단순하다. 프로젝트를 최소 3개 이상 병행으로 돌리는 것이다. A 프로젝트에 명령을 던져놓고 B로 넘어가고, B 결과를 확인하는 사이 C를 세팅하면 기다리는 체감 시간 자체가 사라진다. 빨라진 건 Claude Code가 아니라 내 작업 방식인 셈이다.
