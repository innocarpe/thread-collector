생각보다 빨리 리밋에 걸려버리는군요
리밋에 걸리기까지 써본 경험은

좋았던점
1. AI 품질은 괜찮다 (일단 웹 기준)
- UI 깔끔하게 잘 만들어주며 한번에 요청을 잘 알아듣는 느낌
2. 무료이다
3. 커서 쓸때와 마찬가지로 롤백 기능이 편리하게 잘 되어있다.

안 좋았던 점
1. Claude Code 보단 느리다.
2. Limit이 생각보다 빨리 도달한다.
3. 사용자가 많아서 그런건지(?) 중간중간 요청을 씹어먹거나 에러가 발생하는 경우가 잦다.
