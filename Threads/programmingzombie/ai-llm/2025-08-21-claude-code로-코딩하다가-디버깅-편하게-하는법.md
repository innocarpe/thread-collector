# Claude Code로 코딩하다가 디버깅 편하게 하는법

Claude Code로 코딩하다가 디버깅 편하게 하는법

1. Claude Code에게 문제가 되는 부분에 로그를 찍어 달라고 한다.
2. 앱이든 서버든 Claude Code에게 직접 실행하고 문제를 파악해달라고 한다.
3. 문제가 되는 지점으로 접근한다.
4. 잠시 기다리면 Claude Code가 문제를 파악하고 알아서 수정해준다.
