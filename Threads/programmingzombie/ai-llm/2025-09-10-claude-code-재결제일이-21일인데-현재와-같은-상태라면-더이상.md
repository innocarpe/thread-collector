# Claude Code 재결제일이 21일인데, 현재와 같은 상태라면 더이상의 구독은 연장하지 않고 다른 대안을 찾아보려합니다

Claude Code 재결제일이 21일인데, 현재와 같은 상태라면 더이상의 구독은 연장하지 않고 다른 대안을 찾아보려합니다.

최근 많이 겪은 불편한 경험들
1. 성능이 정말 많이 떨어짐을 느낀다.
2. 쓸데없이 사용되지 않는 코드들 생성하는 경우 많음.
3. 쓸데없이 로그를 지나치게 많이 찍는다.
4. 테스트 코드 또한 정말 유의미한 검증을 해주지 않는다.(구멍 많음)
5. 간단한 기능을 요청했으나 오버스펙으로 만드는 경우가 많음.
6. 성능 이슈가 꽤나 오랜시간 지속되었음에도 불구하고 해결해주는 능력이 떨어진다.
7. 대충 말하면 잘 해결 해줬는데, 이제는 상세히 적어줘도 놓치는 경우가 많음.

다시 성능이 좋아진다면 돌아오겠지만, 최근 2~3주간 클코와 함께한 바이브코딩 경험은 너무나 불편함의 연속이었습니다.

바이브 코딩을 통해 만들어내는 속도보다, 이를 검증하고 문제를 찾아내는데 들어가는 시간이 대폭 상승하면서 생산성이 많이 떨어진 느낌입니다. 😩
