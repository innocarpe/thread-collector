# 이제 폰만 가지고 있으면 어디에서나 코딩을 할 수 있게 됐네요

이제 폰만 가지고 있으면 어디에서나 코딩을 할 수 있게 됐네요.

오늘 Anthropic이 Claude Code Channels라는 걸 공개했는데요.
한 마디로 설명하면, 내 폰의 텔레그램이나 디스코드에서 Claude Code한테 직접 일을 시킬 수 있는 기능입니다.

집이나 사무실 컴퓨터에서 Claude Code를 켜놓으면, 밖에서 폰으로 "이 버그 고쳐줘" 하고 메시지 보내면 Claude가 알아서 작업하고, 끝나면 다시 폰으로 결과를 보내줍니다.
카카오톡으로 친구한테 부탁하듯이, 텔레그램, 디스코드로 AI한테 코딩을 시키는 거죠.

여기서 진짜 무서운 건 이겁니다.

컴퓨터를 안 끄고 Claude Code를 계속 켜두면, 24시간 내 명령을 기다리고 있는 개발자가 한 명 생기는 겁니다. 이동 중에도, 밥 먹으면서도, 아이 재우면서도 "이거 해놔" 하면 나 대신 일을 해주는거죠.

1인 개발자 입장에서 이건 사실상 직원 한 명 고용한 효과입니다.

아직 초기 버전이라 아쉬운 점도 있긴 합니다. (특정 작업은 컴퓨터로 가서 직접 승인 버튼을 눌러야 하는 경우가 있다고함). 완전한 원격 조종까진 아직 한 발 남았지만, 방향성만큼은 확실한 것 같습니다.

설정 방법이 궁금하신 분들은 공식 문서 참고하세요.
