# ClawdBot은 터미널이고 브라우저건 그냥 컴퓨터를 다 쓸 수 있는 녀석입니다

ClawdBot은 터미널이고 브라우저건 그냥 컴퓨터를 다 쓸 수 있는 녀석입니다.

전에 atlas같은 llm+브라우저에서 한번 이야기했던 것 처럼 얘한테는 절대로 내 계정이나 권한을 주면 안됩니다. 사실 clawdbot이 바이럴된건 메일이건 메신저건 자기계정 다 열어두고 한 케이스인데 이건 초기라 좋은 케이스만 공유되어서 그렇지 앞으로 엄청난 사고들이 발생할겁니다. 대표적으로 터미널 권한이 있는 가끔 맛이가면 claude code는 rm -rf ~/ 명령어를 날려서 1초만에 맥북 초기화시켜버리기도 해요. 

제가 세팅할 방법이면서 추천하는 방법은 아예 새 라즈베리파이건 홈서버건 맥미니건 새로운 pc로 포맷해서 만들고, 구글 계정부터 모든 계정을 새로 세팅해줘야합니다. 마치 신규 직원 입사한 것 처럼요. 

똑똑한 비서 하나 들어왔다고 공인인증서고 내 메일이고 메신저고 다 설치/로그인된 하드에 비트코인도 조금 저장되어있는 노트북을 주진 않잖아요? 그거보다도 위험합니다.

현재 제가 시도해보려는 케이스도 하나 공유드리자면 github계정도 하나 만들어주고 pr만 올릴 수 있게 하려고 합니다 오픈소스처럼요. 그리고 로컬에 claude code 설치해주고 로그인 해두고 github에 이슈 등록되면 그거 이 친구 지메일 계정으로 메일 받게 해서 claude code로 지지고 복고 해서 수정해서 pr을 올리게 할겁니다. 

저는 이걸 반영할지 말지만 결정할거에요. 오픈소스 메인테이너처럼요.

이런식으로 본인 피씨와 계정으로 알아서 할 일들을 나에게 전달해주는 식으로 하나하나 구성하시면 보안이나 치명적인 문제 없이 잘 굴려보실 수 있을 걸로 생각합니다. 진짜 사람 직원 하나 들어왔고 pc하나 지급해준다고 생각하고 만들어보세요! 다만 이렇게해도 저 명령어 날려서 피씨 포맷시킬 수는 있음. 직원들도 가끔 이런사람들 있잖아요 감수해야죠

추가로 해보려는 케이스는 이런 이런 주제 볼만한 유튜브 영상 찾아서 내 메신저로 보내줘 같은 것들 조금 시도해보려고 합니다.

홈서버나 라즈베리파이 같은 기존 홈서버운영 경험이 있으신 분들은 사실 맥미니 필요 없습니다. 전 마침 하나 갖고 있어서 써보려는건데 다 api콜이라 로컬 llm돌리실거아니면 자체 성능이 중요하지는 않아요.

다만 맥미니는 가성비있는 홈서버로 용도로 원래 좀 유명했고 브라우저나 엑셀같은 사용자용 응용프로그램들을 잘 돌릴 수 있다보니 지금 가장 fit한 녀석이라고 볼 수 있죠 ㅎㅎ

또한 사용자의 개발환경과 거의 일치한 환경에서 개발을 시킬 수가 있어서 장점이 많아보여요. 뭐 예를들면 ios 앱 빌드를 시킨다거나 등등 클라우드 환경에서 구축하기 좀 빡센 것들을 공유할 수 있어서 세팅 코스트가 굉장히 낮을듯 합니다.
