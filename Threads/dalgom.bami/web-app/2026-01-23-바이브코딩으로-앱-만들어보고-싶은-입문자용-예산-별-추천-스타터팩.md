# 바이브코딩으로 앱 만들어보고 싶은 입문자용 예산 별 추천 스타터팩

바이브코딩으로 앱 만들어보고 싶은 입문자용 예산 별 추천 스타터팩

우선 이건 공통
- 안드로이드 개발자 계정($25, 최초 1회 한번)

1. 정말 최소만 쓰고 싶다
- AI 코딩툴 Antigravity(무료)
- ios 과감히 포기해서 연 $100 아낌 + 맥북 없어도 됨
- 서버 없는 앱 개발해서 서버 비용도 0원

2. 대중적인 스택으로 기본은 하고 싶다
- AI 코딩툴 커서($20/월)
- ios 개발자 계정($100/년), 맥북으로 하면 좋긴 하지만 사람들이 잘 모르는게 맥북 없이도 할 수 있음. expo로 개발해서 development build로 하면 원격빌드로 ios앱도 만들 수 있음. 단 테스트를 위해 아이폰 하나쯤은 있는게 좋음. expo(rn)는 대중적인 스택이라 힙스터 기술도 아님

3. 난 복잡한거 싫고 돈 많다
- replit, rork, createanything 등으로 best practice 따라가며 개발
- ios 개발자 계정($100/년)

(주의) 개발 지식 없이 서버 필요한 앱은 바이브코더라면 개발 감 잡기 전에는 어지간하면 손도 안대는 것을 추천. 특히 첫 앱이면 더

[개발자용 스택 추천]
- 개발자면 claude code가 현재 무난하게 가장 좋은 선택. $100 플랜정도부터 시작해서 부족하면 늘리면 될 듯

서버가 들어갈 시
- supabase (어지간한 트래픽 생기기전엔 사실상 무료)
- cloudflare r2(이미지, 영상 등 용량 큰 파일 저장용 스토리지). supabase storage는 총 1기가 제한이라 금방 과금 되니 비추천
- render, railway 가장 작은 인스턴스 월 $5~7정도 함. supabase edge function잘 쓰면 사실 서버 필요 없을 수도 있으나 있으면 배치잡 등 돌리기에 편함.
- modal: serverless로 gpu 인스턴스 띄울 수 있음. 개발자 경험 편하고 잘 만들어놨음. gpu필요한 기능 싸게 굴리고 싶을 때 활용
- gcp 첫 credit 활용
