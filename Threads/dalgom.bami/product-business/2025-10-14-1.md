# 고양이 책집사

1. 고양이 책집사
타이머 + 독서 기록앱 느낌
고양이의 책집사가 되어서 고양이가 책을 읽는걸 도와주는 역할인 컨셉
책 골라서 읽기 시작할때 시작 누르고 종료누르면 자동 기록 됨
읽을 책 목록 미리 등록해두기 가능

장점) 캐릭터가 귀여움. 전체적으로 깔끔하게 필요한 기능만 있음. 실제 내 독서 시간을 기록할 수 있음. 약간 키우기류 게임 같은 느낌
단점) 아직 출시된지 얼마 안되어서 주요 기능 외에 사이드 기능들이 어설픈 것들이 몇몇 있음. 꾸미기 등 게이미페이케이션 요소가 조금 있었으면 좋겠음

2. 북적북적
국내 도서앱 중에서는 제일 사용하는 사람이 많은 것 같음. 앱 이름이 검색 키워드에 나름 유의미하게 잡힐 정도니까 꽤 인지도 있는 앱인듯
퀄리티가 좋고 sns에 공유할만한 view가 멋지게 구성되어있음

장점) 전체적인 앱 퀄리티 굉장히 좋음. sns 공유하기에 좋은 화면 많음(읽은 책 높게 쌓아서 볼 수 있는 화면 있음). 캐릭터 수집 요소 귀엽게 잘 만든 것 같음
단점) 퀄리티 좋고 유저도 많은 앱이다보니 어쨌건 기능도 많고 복잡한 ui 싫어하는 사람은 취향 아닐 수 있음. pro기능은 월/년구독만 있고 lifetime plan은 없음
