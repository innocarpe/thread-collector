# 1인개발자가 처음에는 대부분 수익화, 고객획득에 실패하는 이유

1인개발자가 처음에는 대부분 수익화, 고객획득에 실패하는 이유

능력치가 0~10까지 점수로 있다고 했을 때,
개발자들은 개발 능력은 5 이상으로 준수할테지만
기획, 디자인, UX, 마케팅, 영업, 재무 등의 능력에서는 1 이하, 심하면 0에 가까운 경우가 많아서라고 생각해. 

그리고 하나의 비즈니스가 돌아가려면 이 모든게 곱하기로 영향을 준다고 생각하는데
예를들어 계산해보면 이런식인거지 5(개발) x 2 x 0.2 x 0.4 x 0 x 0 = 0 

개발에 힘을 조금 빼더라도 3 x 1.2 x 1 x 1 x 1 x 1= 3.6으로 나는 이 값이 최소 0만 아니라면 어느정도의 PMF검증의 준비는 되었다고 생각해.

개발자들은 보통 큰 기업에서 커리어를 쌓고 있는 경우가 많고 큰 기업은 개인이 1개의 영역에서 10에 도달하는 것을 지향하는걸 원해. 개발자면 개발에 파고드는 것을 선호하지 여러 영역에 관여하는 것은 큰 조직이 원하는 인재가 아닐 확률이 커.
