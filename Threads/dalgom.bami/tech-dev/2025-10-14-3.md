# Quollect

3. Quollect
문장 수집에 최적화된 앱. 스레드에서 자주 소통해온 @simon.dsgn 님이 만든 앱
확실히 디자이너 베이스이신 분이 만들어서 그런지 앱 디자인 퀄리티가 좋음

장점) 문장 수집이 메인 기능이라 다른 잡다한 기능이 필요 없고 문장을 수집하고 관리하는 게 목적이라면 최고의 선택. 깔끔한 디자인과 UX. pro의 경우 평생 이용권 있음. 
단점) 문장 수집보다 독서 기록이나 목록 관리 등이 목표라면 아무래도 메인 기능을 제끼고 사용하는거라 반쪽짜리 사용이 될 수 있음. sns에 공유할만한 화면이 아직 뚜렷하게 없음.
