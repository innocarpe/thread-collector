# 그럼 어떻게 대응하면 좋을까

그럼 어떻게 대응하면 좋을까? 

나는 현재 운영중인 서비스가 모든 챗봇에서 잘 노출되고 있고 실제로 챗봇을 통한 유입도 상당히 많은 편이라고 생각한다.

- 기본적인 크롤러빌리티를 신경써줘야한다. Google Search console, Bing Web Master 등 주요 검색엔진에 홈화면 뿐만 아니라 주요 컨텐츠가 다 잘 등록되어야한다.(이 부분이 한 줄로 적었지만 꽤 방대하고 할게 진짜 많다)
- Schema를 적극 지원해 봇이 이해할 수 있게 해줘야한다.
- llms.txt 도입을 검토해본다.(아직 얼리어답터 기술이지만 claude가 꽤 적극적)
- GPT 봇들에 대해서 노출이 필요한 사이트 컨텐츠를 적극적으로 열어줘야한다.
- 사이트 렌더링 성능(lighthouse) 점수 높이는 것만 seo가 아니다. seo영역에서 사실 저건 치열한 경쟁 속에서 마지막 고도화 역할이지 훨씬 더 중요한 요소가 많다.
