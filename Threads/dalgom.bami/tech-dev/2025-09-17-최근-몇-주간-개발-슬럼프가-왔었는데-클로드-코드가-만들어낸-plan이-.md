# 최근 몇 주간 개발 슬럼프가 왔었는데 클로드 코드가 만들어낸 plan이 형편없어서 그냥 내가 손으로 직접 짠

최근 몇 주간 개발 슬럼프가 왔었는데 클로드 코드가 만들어낸 plan이 형편없어서 그냥 내가 손으로 직접 짠 코드가 다시 많아졌고 오히려 클로드 코드가 이상하게 잘 되던걸 만져대서 작업하던걸 다 롤백시킨 안좋은 경험이 많았다.

개발 속도가 점점 떨어지고 오히려 클로드 코드를 쓰는게 더 안좋다고 생각하고 있었는데 실제로 클로드코드에 성능 이슈가 있다고 anthropic에서 공식 발표도 하고 레딧에서 욕도 많이 먹고 복구중이라고 하는데 믿음이 많이 떨어진 상태였다. 

아마 최근 투자 유치를 위해 오버스펙으로 서비스 해주다가 투자 유치 후 비용 절감에 들어간 것 같았다.

다만 1.0.88로 다운그레이드 하면 괜찮다는 이야기가 많이 들려서 구독을 다시 살리고 며칠간 다운그레이드하고 써봤는데 몇 주간 헤메던 때 보다는 훨씬 낫긴 하다..

하지만 이 버전도 결국 서버랑 통신하는 것이기 때문에 언제 또 성능을 조용히 낮출지 몰라서 이번 구독 기간까지만 지켜보고 결정하려 함.
