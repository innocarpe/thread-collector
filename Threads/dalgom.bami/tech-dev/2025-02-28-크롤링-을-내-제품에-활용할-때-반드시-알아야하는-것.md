# 크롤링 을 내 제품에 활용할 때 반드시 알아야하는 것

크롤링 을 내 제품에 활용할 때 반드시 알아야하는 것

미국에서도 완전 관대한 건 아니지만 우리나라는 관련 대법원 판례도 있고 소송도 은근히 빈번히 발생하는 등 미국이나 글로벌 보다 더 엄격하게 신경써야 해. 가끔 아이디어 상담 받으러 오는 분들중에 네이버나 어디사이트 데이터를 이용해서 이런저런걸 만든다는 분들이있는데 깜짝놀라서 설명드리는 내용이야.

1. 사이트도메인/robots.txt 접속해볼것
2. 위 텍스트에 해도되는지 안되는지 사이트 주인이 다 써둠. 볼 줄 모르겠으면 저거 복사해서 gpt에넣고 이거 아무나 크롤링해도 되는거야? 라고 물어볼 것
3. 네이버는 기본적으로 모두 금지임
4. 개인적인 목적으로 수집하는건 딱히 막지 않는 것 같고 판례도 없음.
5. 이 데이터를 본인 비즈니스에 누가봐도 티나게 활용하는 순간 소송 가능
6. 인터넷에 있다고 다 내 사업에 써도 되는게 아님. 공식 api를 제공해주거나 크롤링 허용된 곳 아니면 기본적으론 하면 안된다는걸 꼭 명심
