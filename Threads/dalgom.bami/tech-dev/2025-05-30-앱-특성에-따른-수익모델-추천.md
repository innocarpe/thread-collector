# 앱 특성에 따른 수익모델 추천

앱 특성에 따른 수익모델 추천

유저 한 명당 월 평균 3~5시간 정도는 유저가 앱에 시간을 써줄 카테고리다
→ 애드몹등 인앱 광고 모델 + 구독형(광고제거+프리미엄기능)으로 최적화

유저가 꾸준히(매일,매주,매달) 쓸 앱이긴 한데 앱에서 빠르게 유저가 원하는 것을 얻고 나가기 때문에 체류시간은 상당히 짧다
→ 광고x, 구독 앱. 첫 세션에 hard paywall(결제 안하면 사용 못 함)로 구독 유도. 또는 무료체험 n일+구독 플랜 이용

유저가 진짜 가끔 필요하지만 엄청 필요한 기능이긴 하다(지불 의사 큼)
→ 웹으로 서비스하는게 더 좋을 수 있음. 웹에서 1회성 결제로 제공
→ 앱에서도 1회성 결제로 할 수 있긴 하지만, 이러면 마케팅 플랜 설계를 정말 잘해야함. 
1명 앱 설치 시키는데에 비용이 웹 유저 확보에 비해 훨씬 비싸고 구현도 더 복잡하기 때문

실물 제품 큐레이션 형태의 앱이다
→ 아마존, 쿠팡 어필리에이트 활용

(플랫폼 비즈니스는 제외함. 혼자선 하지마세요..)
