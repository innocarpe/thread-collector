# 검색엔진의 순위를 결정하는 요소는 컨텐츠,테크니컬 요소 등등 정말 다양해

검색엔진의 순위를 결정하는 요소는 컨텐츠,테크니컬 요소 등등 정말 다양해. 

알려진 정확한 공식이 있지도 않고 매달 업데이트 되고 구글은 이걸 서치콘솔에 항상 공지해줘. 여튼 수년간 쌓인 여러 지식들이 있고 이에 따라서 최적화 하는 방법들을 보통 강의에서 알려줄거야. 

자 이중에서 혼자 달성하기 힘든게 바로 백링크 획득이야.

백링크는 다른 사이트가 내 글을 보고 맘에 들어서 링크를 자기 사이트 본문에 출처 등으로 달아주는 거야 html <a>태그를 이용해. 

스레드로 치면 인용,리포스트 기능으로 샤라웃 해주는거라고 보면 되겠다.

이 백링크의 양과 질이 굉장히 중요한 요소로 알려져있어. 구글은 구글 창업자가 구글을 처음 만들 때 본인들이 쓴 논문인 PageRank라는 백링크 기반에서 시작한 검색엔진이거든. 최근엔 백링크의 점수 비중을 줄이고는 있다고 하더라

나 혼자하기는 어렵겠지?
