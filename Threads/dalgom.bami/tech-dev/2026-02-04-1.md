# 센서타워 들어가서 매출 순위에서 생산성, 유틸리티, 사진 앱 등을 쭉 구경한다

1. 센서타워 들어가서 매출 순위에서 생산성, 유틸리티, 사진 앱 등을 쭉 구경한다.
2. 설치해서 써본다. 레퍼런스가 온보딩 + 페이월 + 무료체험 유료구독을 유도하는 앱이면 좋다.
3. 우리가 노릴 앱은 뭔가를 올리거나 입력하면 뭔가 처리해서 짠하고 주는 앱이다. 플랫폼 이런거 절대쳐다보지도 말 것. 예를들면 사진올리면 배경제거해주는 앱 같은 연속성이 없는 하나의 기능. 비싼 llm기능 쓰지말것. 로그인넣지마
4. 이 구독까지해서 유료로 파는 기능을 나는 무료 앱으로 만들어. 대신 이 기능 한번 수행할때마다 애드몹 전면광고를 보여주도록 해. 안드로이드만 할거임(ios 개발자계정 비싸니 비추. 안드로이드 $25)
5. 출시하고 유료광고 돌려. 예산은 10만원. 
6. 유료광고는 google ads의 앱설치 캠페인이 진짜 제일 쉬움. 설치당 50원세팅하고 일 예산 만원정도. 국가는 모든 국가.
7.  이러면 방글라데시인도파키스탄 형님들이 진짜로 설치당 몇십원에 설치되고 앱도 씀.
