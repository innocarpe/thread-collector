# 앱 스크린샷 준비

1. 앱 스크린샷 준비
2. 스샷 sora2에 넣고 앱 소구점 넣고 홍보영상만들어달라고함. 이 영상만보고도 스토어에 앱이 있다는건 인지가능해야함
3. 틱톡계정생성
4. 틱톡 프로필에 앱 제목 넣고 스토어에서 검색하라고 유도문구 작성
5. 영상올리기
6. 올리면 홍보하기(promote) 기능 틱톡에 있음. 1~2만원 예산도 가능. 인앱결제로 긁어짐.
7. 영상조회수, 타겟국가 자동(한국계정은 보통 한국만타겟됨) 또는 특정국가타겟(그쪽 나라에 웜업안한 틱톡계정이면 노출 안될 수 있음)

끝. 효율은 안좋으나 사업자, 큰 시드머니 없이 가능하다는 점이 장점. 영상 잘만들수록 효율 up

세상에서 제일 빠르게 딸깍, 소액(1~2만원) 유료 앱 마케팅 경험해보는 방법
