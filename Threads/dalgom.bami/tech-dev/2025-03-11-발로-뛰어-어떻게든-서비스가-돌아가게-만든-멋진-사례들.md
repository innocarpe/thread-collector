# 발로 뛰어 어떻게든 서비스가 돌아가게 만든 멋진 사례들

발로 뛰어 어떻게든 서비스가 돌아가게 만든 멋진 사례들

1️⃣ 토스는 초기에 송금요청이 들어오면 직원이 직접 은행에 요청해 송금처리를 해줬다. 당시 써봤을 때 실시간이 아니고 당일에 되긴 됐어서 만족했음. 공인인증서 지옥에 가고싶지않았거든

2️⃣ 김캐디도 초반엔 예약이 들어오면 상담원이 직접 스크린골프장에 전화를 걸어 예약해줬다. 확정 시 앱에서 상태를 비동기적으로 바꿔줬음

3️⃣ 명함 관리 앱으로 시작한 리멤버는 ocr기술이 당시엔 오차가 좀 있는편이라 사람이 직접 눈으로 보고 입력해줬었다. 택배로 명함 수백장보내면 다 등록해주고 그랬다 😇😇

4️⃣ 배달의 민족도 가게 디비등록을 전단지 주워서 직접 해서 콜드스타트를 해결했다. 전화위주에서 지금같은 온라인주문으로 바꿀 때 앱에서 주문들어오면 배민 직원이 가게에 직접 전화해 배달접수해줬다. 

검증되기전에 모든걸 개발하려고 하는건 아닌지 생각해봐. 생각보다 사람이 직접했을 때 퀄리티와 고객 경험을 놓치지않고 잘돌아가는 경우가 많아😊
