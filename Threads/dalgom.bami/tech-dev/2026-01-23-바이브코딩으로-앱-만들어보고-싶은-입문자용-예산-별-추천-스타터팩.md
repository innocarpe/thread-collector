# 바이브코딩으로 앱 만들어보고 싶은 입문자용 예산 별 추천 스타터팩

바이브코딩으로 앱 만들어보고 싶은 입문자용 예산 별 추천 스타터팩

우선 이건 공통
- 안드로이드 개발자 계정($25, 최초 1회 한번)

1. 정말 최소만 쓰고 싶다
- AI 코딩툴 Antigravity(무료)
- ios 과감히 포기해서 연 $100 아낌 + 맥북 없어도 됨
- 서버 없는 앱 개발해서 서버 비용도 0원

2. 대중적인 스택으로 기본은 하고 싶다
- AI 코딩툴 커서($20/월)
- ios 개발자 계정($100/년), 맥북으로 하면 좋긴 하지만 사람들이 잘 모르는게 맥북 없이도 할 수 있음. expo로 개발해서 development build로 하면 원격빌드로 ios앱도 만들 수 있음. 단 테스트를 위해 아이폰 하나쯤은 있는게 좋음. expo(rn)는 대중적인 스택이라 힙스터 기술도 아님

3. 난 복잡한거 싫고 돈 많다
- replit, rork, createanything 등으로 best practice 따라가며 개발
- ios 개발자 계정($100/년)
