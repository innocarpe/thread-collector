# AI 챗봇이 웹검색 기능을 제공해주면서 SEO는 이제 중요하지 않다는 글이 자주 보인다

AI 챗봇이 웹검색 기능을 제공해주면서 SEO는 이제 중요하지 않다는 글이 자주 보인다

근데 조금만 생각해봐도 웹검색은 LLM의 컨텍스트에 RAG로 상위 검색결과를 넣어주는건데 SEO가 중요하지 않을 수가 없다. 오히려 첫 페이지에 못들어가면 큰일나는 상황인거라고 생각한다.

ChatGPT는 웹검색 시, 임베딩을 이용한 문서 찾기와 bing 검색을 보조로 사용 중이며
Perplexity는 Bing 검색결과, 그리고 자체 PerplexityBot을 이용한 커스텀 검색 엔진을 구축해 혼합해 사용 중인 것으로 보인다.

Claude는 최근 지원하기 시작헀는데, 관련해서 분석한 아티클들을 보면 Brave search와 85% 이상 일치한다고 하는걸 보니 brave search의 api + 본인의 database를 쓰는 것으로 보인다. 

SEO가 이제 중요하지 않다고 생각한다면 GEO랑 헷갈리고 있는건 아닌지 확인해볼 필요가 있다.(Generative Engine Optimization)
