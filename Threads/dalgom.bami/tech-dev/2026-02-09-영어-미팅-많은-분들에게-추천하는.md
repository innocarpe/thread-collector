# 영어 미팅 많은 분들에게 추천하는

영어 미팅 많은 분들에게 추천하는 
회의 멘붕 방지 서비스 4가지를 소개합니다

1. otter.ai: 미팅 중 실시간 받아쓰기
- 상대방이 빠르게 말해도 텍스트로 바로 떠서, 못 알아들은 부분 눈으로 바로 확인
- 끝나고 AI가 요약, 액션아이템 자동으로 뽑아줌. 회의록 개꿀

2. Fireflies.ai: 미팅 녹음/분석 자동화
- 누가 뭐라고 했는지 화자구분되어서 누가 한말인지 바로 찾을 수 있음
- 키워드랑 날짜별 과거 미팅 검색이 돼서 10초만에 찾을 수 있음

3. grammarly.com: 채팅, 후속 이메일 실시간 교정
- 미팅 중 채팅 영어로 칠 때 실시간으로 문법 잡아줘서 수치사 막아줌
- 팔로업 메일 쓸 때 톤을 비즈니스 톤에 맞게 바꿔줘서 든든함

4. trysmooth.ai: 실시간 미팅 AI 어시스턴트
- 상대가 말하는 것 한글로 번역되고 말문이 막힐 때 다음 할 말 AI가 제안해줌
- 미팅 끝나면 내 영어 피드백 리포트가 바로 와서 복습하며 실력 향상 가능
