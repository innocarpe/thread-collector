# 제대로 당해봐야 내 진짜 경험이 된다

제대로 당해봐야 내 진짜 경험이 된다

react state 10단 prop drilling으로 고통 한 번 받아봐야 zustand가 반갑고
rest api 데이터 관리하다가 캐싱에서 꼬여서 버그 여러번 내봐야 react-query가 반갑고
aws에 최소사양 db, ec2 최소로 띄웠는데도 월 몇만원씩 타봐야 supabase가 반갑다

테이블에 인덱스 정확히 안걸었다가 데이터 쌓였을 때 응답속도 느려져서 서버 터져봐야 index의 소중함을 알게 되고
트래픽 몰려서 운영 서버가 크게 장애나고 식은땀 흘리며 복구해봐야 메시지큐의 소중함을 알게된다

좋은 엔지니어는 이 기술을 상황에 맞게 지금 왜 쓰는지 잘 알고 있어야 된다고 생각한다
그리고 가장 빨리 배우는 법은 한 번 호되게 당해보고 그 기억을 잊지 않는 것이다 😂
