# 한국 국적의 1인 개발자여서 답답한 점이 생각보다 많은 것 같다

한국 국적의 1인 개발자여서 답답한 점이 생각보다 많은 것 같다. 
그 중에 크리티컬 한 것들 몇가지를 공유해보자면..

1. 스트라이프 못 씀 (제일 크리티컬 함 😢): 수많은 인디해커 관련 SaaS툴이 stripe와 연계되어 돌아가는게 많은데 한국에서 쓰려면 미국 법인을 만들거나 stripe atlas같은걸 이용해서 우회해야해서 상당히 복잡해짐.

2. 틱톡, 릴스, 숏츠 계정 만들어서 영상 올리면 초반에 한국 계정 위주로 노출 됨: 미국에서 만든 계정들은 처음부터 바로 미국 고객에게 도달 가능한 반면 한국에서 만든 계정은 불가능. vpn을 쓰면 우회할 수 있긴한데 숏폼 플랫폼이 vpn을 탐지해서 계정 자체가 쉐도우 밴 당하기도 함.

3. 미국 인디 개발자는 유료 구독 앱 만들 때 사업자 등록 의무 없음(수익 높아지기 전까지)
한국은 단 한건의 결제라도 시키려면 무조건 개인 사업자 등록 필요

언젠간 해결되긴 하겠지만 미국 국적 동업자 한 명만 있어도 많은게 편해지겠다 싶음
