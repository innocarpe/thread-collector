# React native vs Flutter 1인개발기준 뭐가 좋을까

React native vs Flutter 1인개발기준 뭐가 좋을까?

어쩌다보니 둘 다 꽤 많이 쓰게됐고 출시도 많이 했어. Rn은 github star 300개정도 받은 오픈소스라이브러리도 만든 적이 있고 현재도 사이드로 개발중. 플러터는 회사 앱 출시에 활용 중이야

우선 결론만 말하자면 2025년 둘 다 아주 좋은 선택이 되었어. 근데 장단점이 조금 다른데 취향따라 본인 상황에 맞는걸 선택하면 될 것 같아😆 난 둘다 호

React native
장점
1. 리액트 해 본 사람 바로 적응 가능
2. 과거 단점이었던 expo가 정말 좋아짐. 개발환경 쾌적해짐 
3. 수많은 라이브러리 생태계

단점
1. 수많은 라이브러리 생태계: 뭘골라야할지 애매할 때가 있고 폭탄라이브러리를 고르면 빌드 지옥될 수 있다
2. 플러터 대비 이런저런 라이브러리를 더 많이 install해서 dependency 관리 해줘야함
3. 갠적으로 애니메이션적용은 플러터 난이도가 쉽더라 rn이안되는건 아님
