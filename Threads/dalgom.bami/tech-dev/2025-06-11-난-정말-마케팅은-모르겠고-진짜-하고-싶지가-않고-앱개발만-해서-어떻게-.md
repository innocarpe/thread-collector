# 난 정말 마케팅은 모르겠고 진짜 하고 싶지가 않고 앱개발만 해서 어떻게 안될까

난 정말 마케팅은 모르겠고 진짜 하고 싶지가 않고 앱개발만 해서 어떻게 안될까? 
라고 생각이 드는 개발자를 위한 마케팅 전략

우선 나는 Paid와 여러 sns,커뮤니티 마케팅 채널을 조합해서 활용하는 것을 더 선호하고 성격이 급해서 성향에 맞지 않아 사용하지 않는 방법이지만 이 방법은 어떤 개발자건 사용자 획득을 할 수 있을거라고 자신함

이 전략이 잘 맞는 사람
- 앱 개발 빠르게 할 자신 있음
- mvp 코드에 장인 정신 없음
- 이 앱을 통해 당장 돈벌어야하는거 아님(본업 있음)

스레드에서 다루기엔 너무 긴 내용이긴 해서 추후에 뉴스레터로 한 번 다시 정리하긴 할건데 일단 간단한 방법을 먼저 공유할테니 힌트로 얻어갔으면 해!

(자세히 보기👇)

1. 앱 아이디어 일단 쭉 노트에 정리하기
2. 해당 아이디어의 핵심 검색 키워드 추정해보기(가급적 영어)
3. apple search ads에서 검색 키워드 별 Popularity 확인
4. 적당한 핵심 키워드를 찾아내고 볼륨과 popularity가 어느정도 된다 싶으면 개발 시작
5. 2주 개발, 2주 디버깅+앱 스토어 심사 올리기
6. 앱스토어 페이지에 핵심 키워드를 타겟한 ASO 빡세게 잘 해두기. 이 부분은 GPT가 잘 도와줄 수 있음. 스샷, 제목, 부제목, 키워드 엄청나게 중요함
7. 출시 후 방치해 둠
8. 다음 앱 개발 시작
9. 6개월정도 지켜보며 애플 스토어 리포트에서 월간 다운로드 300 넘는 앱 나올때까지 계속 반복
10. 월간 다운로드 300 넘는 앱이 생기면 그때부터 앱 기능 고도화, 수익 모델(구독,광고 등) 진행
11. ASO로 효과를 보려면 최소 6개월은 걸리기 때문에 여러 앱을 뿌려두며 되는 놈만 키우는 전략
12. 매달 하나씩 출시 목표
