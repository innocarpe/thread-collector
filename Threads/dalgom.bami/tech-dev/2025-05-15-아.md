놀랍게도 스타트업이면 최대 10억원 어치 크레딧을 받을 수 있어
AI LLM Wrapper 스타트업 창업을 노리고 있다면 적극 활용해봐.

생성형 AI 경쟁이 치열해지면서
아마존, 마소(Azure), 구글(GCP)가 서로 경쟁하면서 공격적으로 AI 스타트업을 육성 / 지원하고 있는데 잘 모르고 API 비용 그냥 내는 팀이 많더라고..??

아마존 = Amazon bedrock에 claude 있음, 최대 30만달러
Azure = OpenAI api 있음, 최대 15만달러
구글(GCP) = Gemini 모델들 있음, 최대 30만달러

어지간한 상용 LLM API는 다 제공하고 있고, 잘 조합해서 쓰면 다 받으면 75만달러고 10억정도의 금액이야.
