# 나노 바나나 API 비용 분석

나노 바나나 API 비용 분석

정식 명칭은 gemini-2.5-flash-preview-image

입력 비용은 gemini가 별도 토큰화를 통해 1024x1024 이미지를 넣더라도 높게 측정되지않아서 진짜 100만토큰 텍스트 프롬프트를 줄게 아니라면 무시해도 괜찮은 수준

출력 비용은 1024x1024(=1,290토큰) 한 장에 $0.39(=54원)

나노 바나나는 기존 이미지 모델들에 비해서 일관성 유지가 뛰어나기 때문에 충분히 AI 제품화하기 좋은 퀄리티이고 이런 시도가 많이 있을 것으로 예상됨

다만 한 장당 54원은 gemini 플레이그라운드에서 만들어보면서 비용 청구안될 땐 별로 체감이 안될테지만 실제 사용자에게 제공하면서 마케팅비용과 서버 운영 비용, 마진까지 고려했을 때 장당 평균 100~200원은 받아야 비즈니스가 정상적으로 돌아갈 수 있을 것 같음. 

ex)월 9,900원을 받아도 사용자가 한달에 200장 이상 만들면 적자임
