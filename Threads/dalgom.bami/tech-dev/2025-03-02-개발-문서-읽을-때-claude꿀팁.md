# 개발 문서 읽을 때 claude꿀팁

개발 문서 읽을 때 claude꿀팁

개발하다보면 공식문서 읽을 일이 정말 많은데 보통은 영어로 되어있어도 개발자들은 보통 슥슥 읽기엔 문제 없지만 가끔 정말 중요한 페이지나 어려운 개념의 문서가 나올 때가 있어. 잘못파악하고 개발하면 나중에 크게 고쳐야하거나 진짜 버그투성이 될 수 있어서 나는 아래 방법으로 문서를 한 번 더 읽고 있어.

예를들면 react-query 공식문서의 important defaults 페이지는 이 라이브러리를 쓴다면 진짜 정확히 알고 있어야하는데 이런 페이지들을 대충보다보면 나중에 큰 문제가 되기도 해. 

방법은 거창하게 써둔거에 비해 별거 아니긴한데 한 번 같이 살펴보자👇
