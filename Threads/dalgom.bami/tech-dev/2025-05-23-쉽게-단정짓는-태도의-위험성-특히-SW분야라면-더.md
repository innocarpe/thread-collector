# 쉽게 단정짓는 태도의 위험성, 특히 SW분야라면 더

쉽게 단정짓는 태도의 위험성, 특히 SW분야라면 더

7년전에 갑자기 부서에서 혼자 빠르게 앱을 만들어야 하는 상황이 생겨서 expo + react-native를 처음 써봤었다. 

당시엔 expo도 react-native도 완전 초창기라서 조금 미숙한 기술이었는데, 그래도 어느정도 android framework지식이 있고 react를 할 수 있는 개발자라면 UI를 만드는 공수를 많이 줄여줄 수 있어서 해볼만 하다고 생각했던 것 같다. 

근데 막상 개발해보니까 진짜 빌드할때마다 스트레스를 너무 많이 받았고 깨져서 ui만드는데에 아낀 시간을 여기에 다 날렸었다. 당시에 reddit에는 expo는 특히 구리고 느리고 빌드도 깨지고 쓰레기 같다는 글이 매일 도배되었었다. 

이 때 너무 화나서 RN 차트 라이브러리를 직접 만들어서 오픈소스로 배포하기도 했다.
