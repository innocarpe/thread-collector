# 최근 애플 앱스토어쪽 변화가 심상치 않은 것 같아

최근 애플 앱스토어쪽 변화가 심상치 않은 것 같아

일단 앱을 AI로 찍어낼 수 있다보니까 ASO관련 로직을 계속 변경하는 것 같아 그래서 멀쩡히 잘 트래픽이 나오던 앱들이 트래픽을 다 잃기도 하는 것 같고

심사 자체도 훨씬 깐깐해진게 느껴져. 비슷한 앱이 너무 많은 경우 독창성이 없으면 심사 리젝먹이는 빈도가 정말 많이 늘어난 것 같아. 아무래도 카피캣 앱이 AI의 힘으로 쏟아져 들어오고 있기 때문이 아닐까 싶네.

앱스토어에 첫 출시하면 1주정도 트래픽 밀어주던 것도 있는데 이거 없어졌다는 이야기도 많이 나오는 중.

플레이스토어는 이걸 비공개테스트12명 해야된다는 강한 정책을 걸어서 막고 있어서 좀 덜깐깐한 느낌이 들어. 확실히 법인이나 이미 앱을 운영하고 있는 경우 시원시원하게 앱 출시, 심사를 해주는 느낌

앱 업데이트도 ai 심사가 걸리면 30분안에 되는 빈도도 더 높아졌고 한번 화이트리스트에 올라가면 쿨하게 승인해주는 편. 법인에게 특히 관대한 것 같음
