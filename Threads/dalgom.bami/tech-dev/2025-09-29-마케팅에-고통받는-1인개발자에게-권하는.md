# 마케팅에 고통받는 1인개발자에게 권하는

마케팅에 고통받는 1인개발자에게 권하는
바이브코딩시대 0 to 1 마케팅 플레이북

1. 아이디어 구상
2. 내 니치타겟 유저들에게 조회수 터진 숏폼이면서, 내가 구상하는 앱이 자연스럽게 1~2초라도 등장가능한 숏폼 찾기(이미 다른서비스 등장중인 숏폼이면 더 쉬움)
3. 바이브코딩으로 진짜 동작하는것처럼 보이기만하는 데모 프로토타입 앱 with 더미데이터 준비(하루 이상투자x, expo go로 하면 앱 빌드시간조차 필요없음)
4. 터진 숏폼 구성 최대한 비슷하게 만들고 내 프로토타입 앱 1~2초라도 자연스럽게 등장시킴
5. 터질때까지 반복, 터트리기전까지 앱개발x, 아이디어를 바꿔도 좋음
6. 터지면 앱개발시작. 출시 후 터진포맷 아주 약간씩만 변주 주면서 무한 생산
7. 제일 잘터진 숏폼을 paid광고소재로 씀(cpi진짜 엄청 낮게 잘나옴)
