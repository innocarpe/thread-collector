# 새로 생긴 스레드 DM 기능으로 드디어 가능해진 방법

새로 생긴 스레드 DM 기능으로 드디어 가능해진 방법

1인 개발자 입장에서 스레드는 잠재고객 확보에 정말 최적화된 플랫폼이라고 생각하는데 이번에 DM기능이 생겨서 더 제대로 활용해볼 수 있게 된 것 같아. 

우선 잠재고객을 확보하는데에 정말 효율적이고 유명한 방법인 리드마그넷을 활용하는 방법이 있지. 보통 무료 PDF나 공들여 만든 자료 등을 미끼(?)로 걸고 댓글에 이메일등을 남겨서 글을 받아가는 사람이 댓글,리포스트 같은 글을 부스팅 해주는 좋은 일을 해주고 바이럴 시켜주는 대신 자료를 주는 모종의 거래로 돌아가는 방법이야.

근데 이게 이메일을 남기면 너무 찝찝하잖아 개인정보 유출도 걱정되고. 그래서 인스타나 링크드인에서는 보통 그냥 "댓글" 이라고만 달게 하고 dm으로 보내주고 그랬는데 스레드에선 어려웠지.
