# 요새 AI가 앱이건 웹이건 뚝딱 잘만드는 것 처럼 보이는 이유

요새 AI가 앱이건 웹이건 뚝딱 잘만드는 것 처럼 보이는 이유

기존 프로젝트가 아닌  깨끗한 제로베이스에서부터 뭔가를 만들기 때문이라고 생각해.
그래서 보기에는 와 진짜 다만들었다 끝났다! 라고 보이지만

원래 개발 해본사람들은 다 알텐데 처음에 백지상태에서 초기 mvp까지 만드는 것 까지는 쉬운 편이야. 
문제는 그 뒤로 여러 요구사항에 맞춰서 수정하고 유지보수해나가는 것이고, 대부분의 개발자는 이런 과정에 큰 의사결정을 해야하기 때문에 기업에서 고용해서 쓰는거지.

프로토타입 뚝딱 만드는 것 까진 AI가 어떻게어떻게 할 수 있다고 보는데 이건 원래도 개발자가 금방하던 영역이고 중요한건 서비스라면 유지보수를 할 줄 알아야하는데 이건 cursor끼고 내가 짠 코드를 보면서 검토하는 것도 쉽지 않은데 개발 모르는 사람이 운영중인 서비스를 만드는 것은 적어도 현 시점에는 불가능하다고 생각해.
