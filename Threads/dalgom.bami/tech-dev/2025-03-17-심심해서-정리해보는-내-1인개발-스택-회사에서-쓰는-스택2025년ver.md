# 심심해서 정리해보는 내 1인개발 스택 / 회사에서 쓰는 스택(2025년ver)

심심해서 정리해보는 내 1인개발 스택 / 회사에서 쓰는 스택(2025년ver)

매년 바뀌긴 하지만 한 번 정리해보면 재밌을 것 같아서 😊

1인개발 
App: Flutter, React Native
Web: React(react router or next.js)
Database & Server: Supabase로 백엔드 모두 해결(최근 업뎃으로 Message queue도 가능해짐)

회사(시리즈 A 스타트업)
App: Flutter
Web: React(Next.js(SEO프로덕트) / Remix(백오피스))
Database: RDS(aurora)
Server: Nest.js
ORM: Prisma
Edge function: GCP cloud run
MQ: SQS

1인개발쪽 스택은 supabase가 날로 성장하면서 점점 더 심플해졌고 이제 supabase 하나로 백엔드 전체 구성이 가능해진 것 같아.

회사쪽 스택은 마음에 안드는 부분도 많은데 legacy땜에 천천히 정리해나갈 예정
