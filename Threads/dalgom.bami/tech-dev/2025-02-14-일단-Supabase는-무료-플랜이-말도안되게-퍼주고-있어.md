# 일단 Supabase는 무료 플랜이 말도안되게 퍼주고 있어

일단 Supabase는 무료 플랜이 말도안되게 퍼주고 있어. 나는 MAU 3만짜리 앱을 서버비 0원으로 굴리고 있어. supabase덕이지.

1인개발이나 mvp로 쓰기엔 정말 최고야.
하지만 이걸로 큰 프로젝트를 진행하고 싶지는 않아. 트랜잭션이 많이 필요하거나 네트워크 여정을 줄이거나 보안관련 코드를보강하려면 진짜 귀찮아 지거든. supabase는 디비를 row level security로 쉽게 프론트가 접근할 수 있게 해줬지만 이걸로 생기는 헛점도 상당히 많아져.
