# 주니어 개발자 입장에서 LLM AI의 등장은 사실 자연재해에 가깝다고 생각했다(반전있음)

주니어 개발자 입장에서 LLM AI의 등장은 사실 자연재해에 가깝다고 생각했다(반전있음)

아마 이미 많이들 들은 것 처럼 시니어 개발자들의 업무효율 개선에 엄청난 역할을 하고 있고, 사실 이것보다 더 심각하다고 생각한 것은 너무 쉽게 문제가 해결되어 버리는 것이다.

내 생각에 주니어 개발자가 가져야할 가장 중요한 역량은 간단한 문제부터 차근차근 스스로 공식 문서를 찾아 읽고, 적당한 키워드로 검색을 하고, 소스코드를 읽으며 사고하는 과정을 반복해 점점 더 복잡한 문제를 해결할 수 있게 되어야 하고, 지금까지는 사실 어찌됐건 일이 되게 하려면 억지로라도 해야했으니 많은 개발자들이 그렇게 성장해왔다. 

근데 LLM AI는 문제를 너무 쉽게 해결해줘버린다. 소위말하는 "딸깍" 하면 해결이 된다.

그리고 이 경험은 유혹이 너무 강력하다. (계속)
