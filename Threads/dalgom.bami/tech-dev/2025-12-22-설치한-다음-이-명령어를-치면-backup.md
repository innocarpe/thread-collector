# 설치한 다음 이 명령어를 치면 backup

설치한 다음 이 명령어를 치면 backup.sql이라는 파일이 pc에 생길거에요. 이게 바로 여러분들 db를 전체 백업한 생명줄 파일입니다. 이걸 매일 실행해서 파일명 오늘날짜로 바꾸고 차곡차곡 쌓으세요. 그럼 위기때 마찬가지로 개발자 헬프 요청 가능합니다

[] 부분은 여러분이 채워주셔야해요. db비번(계정비번 아니고 db비번입니다)은 supabase가입할때 만들었을거고 project_ref는 세팅들어가면 project_id라고 있을거에요.

supabase db dump --db-url "postgresql://postgres:[PASSWORD]@db.[PROJECT_REF].supabase.co:5432/postgres" -f backup.sql

ex)supabase db dump --db-url "postgresql://postgres:123412341234@db.dkjuansgfsyvp.supabase.co:5432/postgres" -f backup.sql
