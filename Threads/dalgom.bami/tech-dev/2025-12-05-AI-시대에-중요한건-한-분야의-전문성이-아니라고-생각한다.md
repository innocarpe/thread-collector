# AI 시대에 중요한건 한 분야의 전문성이 아니라고 생각한다

AI 시대에 중요한건 한 분야의 전문성이 아니라고 생각한다. 도메인 지식이 있냐 없냐가 훨씬 중요하다고 생각한다. 

이 분들은 개발만 초보지 이미 여러 앱과 서비스를 런칭해본 도메인 전문가다. 화면이 어떻게 흘러가야할지 서버 응답속도는 얼마 안에 와야할지 필수 기능은 뭐가 초반에 있어야할지 개발 지식은 없지만 뭐가 잘 되고 있고 안되고 있고는 정확하게 알고있다. 

그럼 이건 AI가 해결해 줄 수 있다. 

당연히 트래픽 몰리거나 해서 서버 터질 포인트나 보안 허술해서 개인정보 유출에 대한 위험 등등 허술한 점도 보인다. 근데 이건 유저 트랙션이 있는 경우에 한참뒤에 벌어질 일이다. 트랙션이 있으면 하지 말라고 해도 개발 공부해서 다 대응할거고 문제 터지면 열심히 배워가며 고치면 될 일이다. 왜 뭐 한다는 사람들 기를 그렇게 죽여놓는지..
