# 스타트업 팀빌딩 됐고, 아이템 정해졌으면 이것부터 해야함

스타트업 팀빌딩 됐고, 아이템 정해졌으면 이것부터 해야함 
진짜 하길 잘했다라고 생각하게 될 거임

1. 도메인 구입
2. 도메인 google search console, 네이버 서치어드바이저에 등록
3. 랜딩페이지 적당히 구현(별 내용 없어도 됨), 단 홈에서 아래 만들 컨텐츠 접근 가능한 링크는 있어야함.
4. (중요) 이 도메인 아래에 포스팅 매일 1개씩 발행. 포스팅은 우리가 풀려는 문제 + 잠재고객이 좋아할만한 정보성 글이면 좋음(ai써도 됨, 단 내 고유 의견이나 경험 들어가야함)
5. 발행한 컨텐츠 search console에서 강제 인덱싱 요청 하나하나 진행
6. {도메인}/robots.txt 그냥 모든 agent allow all로 처리(gpt한테 만들어달라해)
7. 발행한 컨텐츠 팀 또는 팀원 sns계정에 간단한 멘트 + 컨텐츠 링크로 구성해서 꾸준히 게시(인스타,스레드,트위터,핀터레스트,페이스북그룹,단톡방)

타겟 니치만 정해졌다면 제품 없어도 할 수 있음
