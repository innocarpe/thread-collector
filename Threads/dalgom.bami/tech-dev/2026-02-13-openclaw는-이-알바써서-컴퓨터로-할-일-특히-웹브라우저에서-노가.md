# openclaw는 이 알바써서 컴퓨터로 할 일, 특히 웹브라우저에서 노가다성으로 해야할 일을 하기에 최적화된 친구다

openclaw는 이 알바써서 컴퓨터로 할 일, 특히 웹브라우저에서 노가다성으로 해야할 일을 하기에 최적화된 친구다. 웹사이트들은 사람한텐 어떤 서비스건 아주 친절하게 활짝열려있기 때문이다. 그리고 서비스 공급자 입장에서 openclaw는 실제 사람과 기술적으로 구분할 수가 없다. 실제 내 컴퓨터와 내 ip로 사람처럼 브라우징을 하기때문이다. 이 브라우징 기술을 특히 굉장히 잘만들어둔 프로젝트라 생각한다.

보안은 큰 구멍이 있는 것 맞다. 권한이 말도안되게 큰 만큼 보안은 정말 취약하다.

그래서 아예 포맷하고 새 맥미니, 새 구글계정, 새 애플계정으로 세팅해서 쓰면 된다. 이 컴퓨터의 모든파일을 공개드라이브에 올리고 세상사람들이 다 봐도 아무타격이 없을 정도로 운영하면 된다. 개인적인파일,정보,api키 두지말자

그리고 나도 모르게 코인채굴갑자기 하고있을 확률 굉장히 크니 프로세스 관리자는 자주봐주자.. 😉

Openclaw도 따지고보면 api wrapper아니야? 왜이렇게 호들갑이야 라는 의견이나 그냥 크론잡해둔거랑 똑같은거아님? 같은 의견들이 있는데 openclaw가 갖는 최고의 차별점이자 강점은 실제 사람 처럼 웹을 돌아다닐 수 있고 클로드 코드 조차 사람처럼 터미널에서 자기가 알아서 쓸 수 있다는 점이다.

크롤링이나 자동화 봇, make, n8n 같은걸로 자동화시스템 구축을 해본사람은 금방 공감할 수 있을텐데 사람이 웹에서 하면 금방하는데 이 자동화를 하려면 진짜 개고생을 해야할 확률이 크다. 만드려는 기능이 공식 api가 없으면 봇을 만들어야하고, 봇탐지기술이 발전해있어서 좀 쓰다보면 차단당하고 막히고 제대로 되지않는다.

알바고용해서 사람이 하게하면 그냥 쉽게 할 일들을 봇으로 하려면 개고생을 해야한다.
