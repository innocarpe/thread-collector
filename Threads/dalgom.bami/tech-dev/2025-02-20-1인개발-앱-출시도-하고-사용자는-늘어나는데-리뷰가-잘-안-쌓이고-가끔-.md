# 1인개발 앱 출시도 하고 사용자는 늘어나는데 리뷰가 잘 안 쌓이고 가끔 부정 리뷰만 달린다면

1인개발 앱 출시도 하고 사용자는 늘어나는데 리뷰가 잘 안 쌓이고 가끔 부정 리뷰만 달린다면? 대부분 이걸 앱 안에 세팅 안해뒀더라

앱스토어, 플레이스토어 내 앱 페이지는 내 서비스의 랜딩페이지야.
여기까지 힘들게 데려왔는데 설치할지말지가 여기있는 정보로 마음이 휙휙 바뀌기도 해.
저번에 앱 설명과 스크린샷의 중요성을 이야기했었는데 그에 못지않게 중요한게 바로 사용자 리뷰야. 사실 더 중요할 수도 있어.

그런데 잘 생각해보면 우리가 앱을 쓰다가 갑자기 스토어에 가서 와 정말 좋다 리뷰 남겨야지 하는 케이스는 거의 없단말이야.(자발적으로 와서 리뷰 달아주시는 분들 사랑합니다)
화가나면 가서 리뷰를 남기는게 일반적일거고

근데 다른 앱들 보면 앱 리뷰가 몇천 몇만개가 쌓여있는걸 보면서 어떻게 했는지 의문이 들었었어. 

그리고 이 앱 리뷰를 남기게 하는 것도 어느정도 설계가 필요하고 꽤 앱 유지에 중요한 요소더라고, 바로 해볼 수 있는 액션 아이템 알려줄게.
