# 우선 mixpanel기준으로 설명할게 amplitude도 정확히 같은분석이 가능하고 sdk구조도 거의 같아

우선 mixpanel기준으로 설명할게 amplitude도 정확히 같은분석이 가능하고 sdk구조도 거의 같아.

1️⃣ 일단 앱이건 웹에 sdk설치하고 사이트가입, 둘 다 트래픽이 적을 땐 무료 플랜이 넉넉해
2️⃣ 앱에 처음 사용자가 들어왔을 때 부터 내 앱의 핵심 기능까지의 여정을 머리에 그려봐. 그리고 거쳐야하는 단계를 쭉 써봐. 예를들어 지브리스타일 사진 메이커 앱이라 치면, 앱설치->세션시작->온보딩 조회->로그인->사진 업로드-> 프롬프트입력-> 전송클릭-> 결과 조회
이렇게 구성할 수 있겠지? 
3️⃣ 자 이제 저 이벤트가 발생하는 시점에 sdk를 이용해 이벤트를 전송해주게 코드 구현 해. 그럼 events탭에 실시간으로 이벤트들이 저 플로우대로 하면 찍히는걸 볼 수 있을거야.
4️⃣ 리포트로가서 funnel을 골라. Select step부분에 세션시작부터 위 순서대로 쭉 골라

그럼 짠하고 본문 스샷 처럼 퍼널이 나오고 어디서 사용자가 크게 drop되었는지 확인할 수 있어 😊
