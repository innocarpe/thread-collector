# 지금 시점에 생성형 AI 기반 스타트업을 계획중이라면 해볼만한 전략

지금 시점에 생성형 AI 기반 스타트업을 계획중이라면 해볼만한 전략

바로 Microsoft Launcher와 같은 초기 스타트업 지원 프로그램을 이용해 API를 공짜로 사용하면서 서비스를 성장 시키는 방법이 있어

부트스트래핑 방식으로 폭풍성장하고 있는 릴리스 AI도 이 프로그램의 도움을 크게 받은걸로 뉴스레터에서 읽은 것 같아

단계에 따라서 최대 15만달러의 open api 사용 크레딧을 받을 수 있고 매달 api 사용료와 동일한 매출을 발생시킬 수만 있다면 1차적으로 성공이라고 생각해.

왜냐면 현재 추세는 성능은 올라가면서 api 사용료가 1년에 10배씩은 싸지고 있는 추세이고 당분간 지속 될 것 같거든

저런 크레딧 프로그램들은 모든 클라우드 서비스가 제공하는데 일반 개인이 받기는 어렵지만 시드투자를 받은 스타트업이거나 pre A 정도 단계라면 그렇게 어렵지 않게 지원해서 받을 수 있어
