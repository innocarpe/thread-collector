# Cursor등의 LLM 도구는 정말 훌륭한 도구다

Cursor등의 LLM 도구는 정말 훌륭한 도구다. 퍼포먼스 향상에 분명 도움이 된다.
하지만 개발자라면 문제 해결에는 쓰지 않는 것을 권한다.

특히 이거 에러났는데 왜 안되냐 고쳐줘, 이런 기능 구현해줘 같은 단순한 형태로는 정말 지양해야한다. 

추천하는 사용 방법은 내가 머리속으로 알고있는 만큼만 구현시키는 것이다.
내가 손이 부족해서 타이핑 하기 귀찮은 코드을 대신 치도록 부려야지 나도 모르는걸 만들게 하면 절대 안된다. 

저번 주에 소프트웨어 마에스트로 선발 면접에서 면접관으로 참여할 기회가 있어서 이런 질문들을 많이 했다. 

놀랍게도 대부분 주니어 입장에서 LLM을 올바르게 사용하는 방법을 잘 고민하고 실제로 잘 사용하고 있더라. 하지만 힘이 없으니 온라인에서는 주니어들 다 끝났다 망했다 하면서 두들겨맞고 있는 것 같았다. 

사실 요새 젊은놈들 끝났다는 인류의 유구한 역사다. stackoverflow에서 보고 해결하는 것도 죽어라 욕먹던 시절도 있었다.
