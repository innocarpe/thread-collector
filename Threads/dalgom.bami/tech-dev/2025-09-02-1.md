# 아래 프롬프트로 나노바나나용 스크립트 생성

1. 

아래 프롬프트로 나노바나나용 스크립트 생성. (ChatGPT & Claude & Gemini 중 아무거나 본인이 쓰는걸로 이용)

사이트가 없으면 앱 설명을 사이트 주소 부분에 자세히 써주면 됨. 사이트가 있으면 주소 복붙해두면 알아서 사이트 잘 읽어서 이해함

```````
사이트 주소: https://www.calai.app/

[주요 목표]
이 사이트의 앱을 보고, 페이스북 고품질의 광고 이미지를 만들거야. 광고 이미지를 생성형 ai로 만들기 위해. 이 광고 이미지를 묘사하는 이미지 프롬프트를 영문으로 상세하게 작성해줘. 앱 스크린샷이 화면에 나오면서 광고 문구가 자연스럽게 배치되면 돼. 

[주의 사항]
- 앱 스토어, 플레이스토어 다운로드 버튼 등은 포함하지마.
- 집중해야할 주요 소구점: 먹으려는 음식 사진만 찍으면 칼로리가 ai로 자동으로 측정됨
- using attached image 1 as a phone screen 프롬프트를 마지막에 추가해
```````
