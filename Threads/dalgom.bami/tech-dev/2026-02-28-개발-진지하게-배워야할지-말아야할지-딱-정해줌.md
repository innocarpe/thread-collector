# 개발 진지하게 배워야할지 말아야할지 딱 정해줌

개발 진지하게 배워야할지 말아야할지 딱 정해줌

랜딩페이지, mvp,시연용 프로토타입, 나 혼자 쓸 유용한 제품 정도만 만들어서 쓰면 됨 → 바이브코딩툴로 개발지식 없이 써도 충분함. lovable, createanything, rork, 안티그래비티, 커서 정도로 충분

만들어서 유의미한 수익을 타인에게 서비스로(앱,웹) 벌고싶다(외주x) → 바이브코딩 찍먹 해서 흥미, 동기부여 정도만 얻고 바로 개발 배우기시작하는건을 추천. 부트캠프건 인강이건 책이건 ai에게 물어보건 배울 길이 과거대비 정말 널려있음. 바이브코딩을 배우는것보다 훨씬 쉬운 길이라고 장담할 수 있음.

단, 시장, 마케팅의 감각이 본인에게 있는 경우 서버 기능없는 앱으로 한다면 개발지식없이 수익화까진 해볼 수 있으나 고점에 한계가 있음
