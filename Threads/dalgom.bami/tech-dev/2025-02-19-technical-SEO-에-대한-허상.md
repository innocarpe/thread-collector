# technical SEO 에 대한 허상

technical SEO 에 대한 허상
검색엔진 유입 한달 0 클릭에서 50만 클릭까지 직접 키워오며 느낀 점

개발자 입장에서 SEO를 공부하다보면 technical seo에 대한 내용이 아무래도 빨리 와닿고 신경쓰게 돼. 특히 lighthouse같은걸 이용하면 점수가 딱딱 나오니 내 코드가 얼마나 좋구나 잘 하고 있구나 피드백이 있으니 만족스럽지. 

하지만 SEO를 제대로 해본 사람들은 다들 알겠지만 컨텐츠에 비하면 technical seo는 비중이 낮고 후순위야.

technical seo는 컨텐츠가 동일한 질의 컨텐츠와 백링크 품질일 때 경쟁사를 이길 수 있게 마무리해주는 역할이지 컨텐츠가 구리면 pagespeed에서 올100을 맞아도 검색 유입은 0이라는걸 기억해야해.

이 점수에 개발자가 집착하게 되면 무슨 일 까지 생기냐면 Google analytics나 mixpanel같은 3rd party 툴을 연동하게 되면 성능이 팍팍 떨어지는데 이걸 거부하거나, (이어서 계속)
