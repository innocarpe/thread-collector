# 유튜브에서 적당한 영상 찾기

1. 유튜브에서 적당한 영상 찾기
2. ai.studio.google.com 접속
3. Chat들어가서 우측에 Run settings중에 Gemini 2.5 pro 선택되어있는지 체크
4. 영상 주소 복붙한다음 추가로 아래 프롬프트 입력

"나는 문장 1000개를 모아서 통암기하는 식으로 영어 공부를 하고 있는 한국인이야.
이 영상에 나오는 영어 문장 중 외울만한 네이티브가 자주 쓸 법하고 자연스러운 문장을 최대한 뽑아서 정리해주고, 문장 / 뜻 이렇게만 출력해.
영상에 나오는 문장을 그대로 쓰고, 문법이 틀린경우만 교정하고 가급적 원문을 유지해"

5. 여기까지만 해도 되고 "더 뽑아줘" 하면 더 얻어낼 수 있음😊
