나는 어쩌다보니 개발도 서비스 영역에 필요한 모든 기술을 다 조금씩이라도 배우게 되었고, 마케팅, 기획, UX 등등 포지션 특성상 새로 배워야 할 일이 많았는데 

내 기술이 되는 시점이 결국 언제인가 생각해봤어.

개발일 땐 새로 나온 기술의 공식 문서를 보면서 Get Started 섹션을 내 손으로 직접 코드를 쳐보면서 결과를 확인 했을 때,

마케팅일 땐 해당 방법론과 소재대로 광고 채널에 캠페인을 세팅 했을 때,

UX/기획에선 세워본 가설을 일단 사용자에게 배포해보고 A/B테스트로 결과를 눈으로 확인했을 때,

등등....

수년간 머리속으로 공부할 땐 진짜 잘 이해되지 않았던 것들이 결국 내 손으로 직접 한 번 해보니까 바로 이해되는 경험을 정말 많이 했어.

어떤 분야건 요새는 정보가 없어서 못 배우는 시대는 아닌 것 같아. 초심자용 튜토리얼이 유튜브, 블로그 등에 깔려있고 가장 쉬운 것 부터 차근차근 하다보면 어느새 여러 영역을 다 잘 다루는 나를 발견하게 될거야!
