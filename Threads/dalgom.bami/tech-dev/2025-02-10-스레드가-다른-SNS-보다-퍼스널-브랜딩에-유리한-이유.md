# 스레드가 다른 SNS 보다 퍼스널 브랜딩에 유리한 이유

스레드가 다른 SNS 보다 퍼스널 브랜딩에 유리한 이유

지금 구글에 가서 [스레드아이디] site:threads.net 라고 검색해봐. 
나를 예시로 들면 lets.abtest site:threads.net 이 되겠네.

내 프로필과 "작성한 글"들이 대부분 구글에 검색 노출 되고 있는 걸 확인할 수 있어. 이걸 SEO에서는 인덱싱 되었다 라고 하는데, 어찌됐건 특정 목적으로 구글에서 검색하고 있는 사람들에게 내 글이 노출될 확률이 생겼다라고 이해하면 돼.

사실 구글에 등록하는 것 자체는 어렵지 않지만 귀찮은 일이거든, 상위 노출되기도 어렵고.
일반적인 글들을 순위에서 이기긴 어렵겠지만, niche 키워드나, 롱텀 키워드(ex. 앱 개발을 잘하는 법 flutter로, 같은 검색량은 낮지만 구체적이면서 긴 키워드)에서는 내 글이 노출되어서 사람들이 볼 확률이 있다는거지! 

(인스타그램은 주로 프로필 페이지 정도만 인덱싱해주는 것 같아)
