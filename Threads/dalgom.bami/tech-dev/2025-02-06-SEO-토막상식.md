# <SEO 토막상식>

<SEO 토막상식>
백링크란?
1. 구글은 창업자의 논문 PageRank에서부터 시작 됨
2. 다른 사이트에서 이 컨텐츠에 대한 링크가 있다면, 일종의 샤라웃(Shout out)으로 인정해 줌. 
3. 아무리 인정받는 사이트여도 샤라웃을 너무 많이하면 샤라웃 당 가치는 떨어짐
4. 누군지도 모르는 사이트가 샤라웃 해봤자 영향 없음
5. 우리와 연관 있으면서, 인정받는 사이트에서 주는 백링크가 최고
6. 인정 받는 사이트 = Domain Authority가 높은 사이트, 오래 될 수록 유리하고, 좋은 컨텐츠를 많이 갖고 있을 수록, 백링크 획득을 통한 점수가 높을 수록 Domain Authority가 높다고 볼 수 있음

결론 = 좋은 백링크를 많이 보유할 수록 구글 상위검색에 올라갈 확률이 높다.
