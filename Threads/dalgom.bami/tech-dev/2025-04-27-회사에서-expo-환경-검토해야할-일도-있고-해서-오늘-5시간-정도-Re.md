# 회사에서 expo 환경 검토해야할 일도 있고 해서 오늘 5시간 정도 React native 프로젝트를 새로 

회사에서 expo 환경 검토해야할 일도 있고 해서 오늘 5시간 정도 React native 프로젝트를 새로 만들어서 작업해봤어

내가 필요해서 만든 유튜브 요약 앱인데 UI쪽은 괜찮게 나온 것 같고 LLM api 응답 속도 때문에 좀 고민이 되는 부분이 있어서 스토어 출시는 좀 미뤄야할 것 같아. 

앱 디자인 + UI 구현은 커서 + claude 3.7로 했고 style 관련 코드는 내가 직접한건 하나도 없는데도 결과물이 꽤 만족스러워서 공유함.

expo 많이 좋아졌더라 8년전이랑 확실히 다르네.. 🤣

하루만에 다 만들 수 있을 것 같아서 시작했는데 역시 해보면서 배우는게 많은듯 해. 
실제 요약이 많이 필요한 30분~1시간 짜리에서 너무 오래 걸려서 결국 batch job + push notification으로 만들거나 LLM 성능이 더 빨라지길 기다리거나 해야할듯!

그래도 UI 코딩에 쓰는 시간이 거의 없어지다보니까 효율이 정말 많이 올라간 느낌
