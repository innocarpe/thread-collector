# Moltbot(구 clawdbot) 관련 몇가지

Moltbot(구 clawdbot) 관련 몇가지

1. 이제 두 달 정도 된 프로젝트라 보안 이슈 있는거 맞긴함. 아무리 로컬 모델을 써도 외부 연결 등을 위해 순수 로컬에서만 구동되는 식은 아니다보니 관련 결함들 있고 실제 injection공격에 취약한 부분 "아직은" 있음

2. 엄청나게 활발한 오픈소스임. 벌써 컨트리뷰터가 315명이고 미친듯한 속도로 성장중 github star그래프만 봐도 엄청남. 조금 지나면 엄청 크리티컬한 보안이슈는 잡히긴 할 것으로 보임

3. 본인 데스크탑에 설치해서 쓰는건 강력 비추. 개발자여도 비추. env에 담긴 중요 token이나 가상화폐탈취가능성 꽤 높음. 가급적 해커가 진짜 들어왔다고 쳐도 아무것도 가져갈게 없는 pc를 세팅하고 계정도 새 계정으로 해서 장난감 용도로 써보는 것 정도만 아직 추천

4. 바이럴된 사례들이 좋아보이지만 opus 4.5를 안쓰면 좀 별로인듯하고 이거 쓰면 대화당 가격 너무 비싸서 솔직히 쓰기 어려움

5. 로컬모델들의 발전이 무섭긴 하나 api를 활용하는 고급모델에 비해 대화 성능도 부족할 뿐만아니라 속도가 굉장히 답답할 수 있음. 아직은 직원처럼 부리기엔 조금 부족하다는 생각이 들기는 함

6. 이거 쓰려고 쓴 글이긴 한데 이정도로 바이럴되고 관심있는게 검증되었으니 곧 Anthropic이 Claudebot 이렇게 만들어서 곧 출시 할듯(사실 얼마전 출시된 claude co-work에 몇개 기능만 더 붙어도 moltbot임). 아님 OpenAI에서 내거나.. 일반사용자는 이런것들 출시되면 이걸 쓰는게 안전할듯하고 개발자들은 그냥 갖고놀 수 있는 재밌는 장난감 생겼다 + 사실 막상 사면 별로 쓸데는 없지만 괜히 자꾸 언급되니 갖고 싶은 맥미니를 지르는 좋은 명분이 생겼다 정도
