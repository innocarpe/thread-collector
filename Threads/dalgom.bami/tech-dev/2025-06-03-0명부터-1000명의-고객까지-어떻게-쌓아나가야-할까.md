# 0명부터 1000명의 고객까지 어떻게 쌓아나가야 할까

0명부터 1000명의 고객까지 어떻게 쌓아나가야 할까?

멘토링이나 컨설팅을 할 때 가장 많이 받는 질문이고, 다들 가장 어려워 하는 부분이 바로 이 고객이 한 명도 없을 때 어떻게 서비스를 궤도에 올리느냐인 것 같아

사실 나는 이 부분이 제일 자신 있기도 하고 강점이 있는 것도 맞는데 이게 어떤 뚜렷한 공식이 있는게 아니라서 나도 할때마다 참 쉽지 않은 것 같음

보통은 명쾌한 솔루션을 원하는 것 같아. 
예를들면 뭐 어디에 어떻게 홍보해서 쭉쭉 데려온다 같은..

근데 여러 프로젝트를 했던걸 돌아보면 보통 이 첫 0명부터 고객을 쌓아나가는 과정은 보통은 엄청 지저분하고 지루한 일의 반복이었던 것 같아

커뮤니티 활동을 자연스럽게 해서 홍보를 하거나, 
타겟 고객을 하나하나 설득해서 사용해보게 하거나, 
paid로 데려온 고객을 어떻게든 남겨서 고객으로 만들거나, 
실제 오프라인에서 고객들이 모여있는 곳에 찾아가거나,
양질의 무료 자료 배포로 리드를 적극 수집하거나,
