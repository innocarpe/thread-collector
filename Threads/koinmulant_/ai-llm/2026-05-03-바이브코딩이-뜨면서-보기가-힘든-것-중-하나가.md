# 바이브코딩이 뜨면서 보기가 힘든 것 중 하나가

바이브코딩이 뜨면서 보기가 힘든 것 중 하나가...

기존 전문가들이 망할 것처럼 말하는 사람들이다

프로그래머가 사라질 것처럼 말하고,
디자이너가 필요 없는 것처럼 말한다.

근데 그런 사람들이 딸깍으로 만들었다는 프로덕트를 보면
디자인이고 구조고 전부 개판이었다.
UI UX 디자인은.... 뭐 말할 것도 없다

1페이지로 끝낼 수 있는 디자인을
아무 고민 없이 AI에게 맡겨서
여러 개로 나눠진 경우도 너무 많고,

CTA버튼이나 헤더를 스티키 안 시키는 건 기본이다.
(UX의 가장 기본인데 AI가 생각보다 자주 놓친다)

홍보용 디자인을 자동으로 해준다는
플랫폼의 디자인이 개똥 같은 경우도 봤다
(그게 이상한 줄 모르는게 더 이상하다...)

제발 기존 업에 있는 사람들을 무시하진 말자

AI로 쉬운 접근은 가능해도,
모두가 같은 AI를 쓰기에
바이브코딩은 전부 똑같은 디자인과
비슷한 기능으로 수렴되고 있다.

그런 가운데 정말 전문가가 필요 없을까?
한번 생각해 볼 문제다
