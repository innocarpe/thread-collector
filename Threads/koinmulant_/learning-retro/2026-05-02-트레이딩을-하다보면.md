# 트레이딩을 하다보면

트레이딩을 하다보면

뭔가 완벽한 기법과 결과물을 바라는 사람이 많다
완벽한 손절, 완벽한 익절.
그런데 그건 불가능하다.

트레이딩은 공부만 한다고 되는게 아니라
실전이 반드시 동반 되야한다.

차트패턴만 공부하기보다
좋은 지표만 백날 찾기보다
ICT 기법 같은 뭔가 있어보이는 느낌의
기법을 찾고 공부하는 것보다

빨리 지표든 기법이든 확정하고 정리 한 뒤
자기만의 익절 손절 라인을 체득시켜서
승률을 높이고 손익비를 높이는게 정답이다.
