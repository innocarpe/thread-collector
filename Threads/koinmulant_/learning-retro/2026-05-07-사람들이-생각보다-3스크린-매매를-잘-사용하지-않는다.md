# 사람들이 생각보다 3스크린 매매를 잘 사용하지 않는다

사람들이 생각보다 3스크린 매매를 잘 사용하지 않는다.

3스크린 매매는 아주 단순하다.

1분 5분 15분
1시간 4시간 일봉 등

서로 점점 더 작아지는 타임프레임을 보고
장기가 상승 중일 때, 중기와 단기가 조정 받는 모습을 파악해서
진입하는 방법이다.

이게 추세 매매의 가장 기본이기도 하다.
솔직히 ICT 같은 기법 보다
3스크린 매매에 맞춰 추세 매매 하는게 더 확률이 높다.

클래식이 왜 클래식인지.
왜 수 많은 펀드매니저들과 트레이더들이
아직도 이걸 쓰는지 그걸 아는게 좋다.

좋은 기법이라서 살아남았다기 보다는
이게 기본이라서 필요한거다.

적어도 단타를 칠때 15분이 계속 떨어지고 있는데 롱을 치거나,
15분이 계속 오르고 있는데 숏을 치는 짓만 안해도
적어도 청산을 경험하진 않는다.
