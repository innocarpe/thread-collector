# 유튜브 누적수익별 4단계 과정

유튜브 누적수익별 4단계 과정
(조회수 수익만 / PPL 광고 수익 제외)

1️⃣ 1억 미만
: 누구나 가능 / 운에 따라 천차만별
2️⃣ 1~5억 사이
: 감각 생겨야만 가능/여기서 많이 걸러짐
3️⃣ 5~10억 사이
: 변화 성공해야만 가능/기존 노력 안통함
4️⃣ 10억 이상
: 싸이코들만 가능/유튜브가 놀이터 됨

저 같은 4단계 이상 유튜버들은
다 똑같이 말하고 다닙니다.
결국엔 싸이코가 승리하는 바닥이라고.
