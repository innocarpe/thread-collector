# 네이버 블로그 반자동화

네이버 블로그 반자동화
내가 하는 방식 그대로 알려줌

1. 키워드 → 퍼플렉시티로 뽑는다
2. 대본 → 클로드코드한테 시킨다
3. 이미지 → 나노바나나로 뽑는다
4. 발행 → 워드프레스 자동 업로드

여기서 사람 손 가는 건
1번 키워드 고르는 거
3번 이미지 확인하는 거

나머지는 자동이다

이걸로 하루 6개 올린다
수동으로 6개 쓰면 하루 종일인데
이 방식이면 1시간이다

북마크 해둬라
