## 1. 개요

**@ai_margin_** 은 고졸 조선소 출신 비개발자로, AI 바이브코딩을 활용해 혼자 다음 규모의 자동화 사업을 구축한 1인 사업자다.

### 5개월 만에 구축한 규모 (2026-04-01 기준 본인 직접 공개)

| 채널 | 규모 |
|------|------|
| 스레드 계정 | 21개 자동+반자동 포스팅 운영 |
| 블로그 | 6개 자동 발행 운영 |
| 인스타그램 | 본계정 팔로워 20만+, 부계정 20만+, 3번째 계정 1만+ |
| 노트북 | 10대 각 역할별 상시 가동 |
| 전자책 | 출시 2일 만에 400명, 최종 930명+ 다운로드 (무료) |
| 스레드 팔로워 | 본 계정 1.7만 |
| 스레드 주간 도달 | 300만+ |
| 인스타 월간 조회수 | 계정 1개 기준 5000만 |

### 배경

- 코딩 전공 아님, 비개발자
- AI 이전에 제휴마케팅 포함 다양한 수익화 시도 경험
- 바이브코딩(말로 프로그램 만들기)으로 프로그램 20개+ 직접 제작
- LG, CJ 등 대기업 인스타 광고문의 4회+ 수신
- 마누스AI(Meta 운영)에서 유료 협업 문의 수신

## 3. 시스템별 상세 분석

### 3-1. 스레드 자동 포스팅 시스템

**목적:** 스레드 계정 21개를 자동+반자동으로 운영, 쿠팡 수익 자동 발생

**실제 작동 흐름 (본인 공개):**

```
① 바이브코딩으로 내 계정 글 긁어서 저장
② 벤치마킹 계정 글 긁어서 저장
③ AI가 분석해서 대본 추천 (조회수 기반 패턴 분석)
④ AI 리라이팅 → 초안 생성
⑤ 쿠팡 파트너스 링크 자동 교체
⑥ 예약 포스팅 → 자동 발행
```

**구체적 수치:**
- 하루 스레드 7개 올리는데 직접 쓰는 건 2개
- 나머지 5개: 소재 고르기(5분) + AI 초안(3분) + 검수수정(5분) + 예약(1분) = 글 1개 15분
- 한 주치 콘텐츠를 1시간이면 완성
- 주간 도달 300만+ 달성

**핵심 포인트:**
- 처음엔 6단계짜리 복잡한 시스템 만들었다가 전부 갈아엎고 4단계로 단순화
- "복잡한 게 잘하는 게 아니라 단순한 게 진짜 실력"

---

### 3-2. 블로그 자동 발행 시스템

**목적:** 네이버 블로그 6개 운영, SEO 트래픽 기반 쿠팡/광고 수익

**실제 작동 흐름 (본인 공개):**

```
① 퍼플렉시티로 키워드 뽑기 (사람이 1번만 개입)
② 클로드코드한테 "내일 올릴 블로그 6개 써놔" 시키기
③ 나노바나나(제미나이 기반)로 이미지 생성 (확인 1번만 개입)
④ 워드프레스 자동 업로드
⑤ 새벽에 자동 발행
```

**구체적 수치:**
- 자는 동안 12개 생성
- 수동으로 6개 쓰면 하루 종일 → 이 방식이면 1시간
- 클로드코드 퀄리티 80점 → 나머지 20점은 10분 손봄

**노트북 역할 분배 (10대):**
1. 스레드 자동 수집
2. 블로그 자동 수집
3. 스레드 자동 포스팅
4. 블로그 자동 발행
5. 데이터 자동 수집 컨트롤 등

---

### 3-3. 인스타그램 릴스 자동화

**목적:** 팔로워 성장 → 대기업 광고 수익

**실제 작동 흐름 (본인 공개):**

```
① 클로드한테 "건강 릴스 주제 10개 뽑아줘"
② "30초짜리 대본 써줘 건강 친구톤으로"
③ 나노바나나(제미나이)로 이미지/영상 소스 생성
④ 캡컷 자동자막 + BGM 마무리
⑤ 업로드
```

**구체적 수치:**
- 하루 릴스 2~3개 뽑음
- 직접 촬영 없음
- 본계정: 2개월 반 만에 팔로워 20만+
- 부계정: 팔로워 20만+
- 3번째 계정: 1만+ (시작한 지 얼마 안 됨)
- 월간 조회수: 1개 계정 기준 5000만
- 대기업 광고문의: CJ, LG 포함 4회+

### 3-4. 유튜브 대본 자동 추출기

**목적:** 경쟁자 분석, 콘텐츠 리서치 자동화

**실제 작동 흐름 (본인 공개):**

```
① 클로드코드 켠다
② "유튜브 URL 넣으면 자막 추출해서 대본 정리해줘" 지시
③ URL 입력
④ 자막 자동 추출 → 핵심 요약 + 대본 구조 출력
⑤ 대본 구조 / 훅 패턴 / CTA 방식 뽑아서 내 것으로 만들기
```

**구체적 수치:**
- 자기 전에 시켜서 아침에 완성
- 기존에는 외주 맡기면 건당 5만원 → 지금은 0원
- 경쟁자 유튜브 50개 분석에 사용
- 리서치 3일치를 한마디로 대체

### 3-5. 벤치마킹 자동 수집기

**목적:** 경쟁 계정 콘텐츠 매일 자동 분석

**실제 작동 흐름:**

```
① 바이브코딩으로 경쟁 계정 긁어서 저장
② AI가 패턴 분석
③ 내 계정 콘텐츠 방향에 반영
```

---

### 3-6. AI 쇼츠 자동화 (재개 중)

**목적:** 유튜브 쇼츠 자동 생성

**툴:** 리모션(Remotion) 사용
- 대본만 넣으면 영상 출력
- 클로드코드 + 리모션 조합으로 영상 제작

---

### 3-7. 하루 콘텐츠 루틴 시스템 (종합)

본인이 공개한 하루 운영 루틴:

**글 (스레드/블로그):**
```
1. 클로드코드 → "오늘 AI 뉴스 5개 요약해줘"
2. 스레드 초안 3개 자동 생성
3. 검수 2분 → 발행
```

**영상 (인스타/쇼츠):**
```
1. 대본 → 클로드
2. 이미지 → 나노바나나
3. 편집 → 캡컷 자동자막
```

**결과:** 한 세트 30분, 하루 글 7개 + 영상 1개 발행

## 4. 셋업 가이드 (직접 따라하기)

순서: **수동 10번 → 패턴 발견 → 코드로 자동화**

| 도구 | 비용 | 설치 방법 |
|------|------|----------|
| 안티그래비티 (Claude Desktop 앱) | 무료 | anthropic.com에서 다운로드 |
| 클로드 Pro | $20/월 (약 3만원) | claude.ai 구독 |

**총합: 월 7만원**

### Step 2. 첫 번째 자동화 만들기 (1시간 코스)

본인이 직접 제시한 1시간 코스:

**준비 (5분)**
- 클로드코드(Claude Code) 가입 → 터미널 열고 로그인

**첫 제작 (25분)**
```
안티그래비티(또는 클로드코드) 켜고 아래 입력:
"RSS 뉴스 수집기 만들어줘. 
관심 키워드를 내가 지정하면 
그 키워드 관련 뉴스를 매일 자동으로 모아줘"
```
→ 에러 나면 에러 메시지 그대로 복붙해서 붙여넣기 (에러 복붙이 핵심)
→ 스크린샷 찍어서 붙여넣으면 텍스트 복붙보다 에러 해결 2배 빠름

**두 번째 제작 (25분)**
```
"수집된 뉴스로 블로그 초안 만들어줘"
```
→ 1번에서 모은 뉴스를 자동으로 글로 변환

**확인 (5분)**
→ 다음 날 아침에 결과물 확인

---

### Step 3. 블로그 자동화 구축

**본인 실제 방식:**

1. **키워드 추출**
   - 퍼플렉시티 열고: `"[내 분야]에서 이번주 가장 핫한 이슈 5개 알려줘"`
   - 그중 1개 선택

2. **대본 생성**
   - 클로드코드에 입력: `"[키워드]로 네이버 블로그 글 써줘. SEO 최적화해서."`
   - 또는 잠자기 전에: `"내일 올릴 블로그 6개 써놔"`

3. **이미지 생성**
   - 나노바나나(제미나이 기반) 또는 구글AI스튜디오에서 이미지 생성

4. **자동 업로드**
   - 클로드코드한테: `"워드프레스 자동 업로드 스크립트 만들어줘"`
   - 완성된 스크립트를 노트북에서 상시 실행

---

### Step 4. 스레드 자동 포스팅 구축

1. **수집기 만들기**
   ```
   클로드코드에 입력:
   "스레드에서 인기글 자동 수집하는 코드 만들어줘.
   특정 계정을 지정하면 그 계정 글을 긁어서 저장해줘"
   ```

2. **리라이팅 + 쿠팡 링크 교체**
   ```
   "수집된 글을 AI가 리라이팅하고,
   본문에 관련 쿠팡 파트너스 링크를 자동으로 삽입해줘"
   ```

3. **예약 포스팅**
   ```
   "스레드 API 또는 자동화로
   특정 시간에 자동 포스팅하는 시스템 만들어줘"
   ```

4. **전체 파이프라인을 4단계로 단순화:**
   - 인기글 수집 → AI 리라이팅 → 쿠팡 링크 자동 교체 → 예약 포스팅

---

### Step 5. 인스타 릴스 자동화 구축

1. **주제 생성**
   ```
   클로드 입력: "[분야] 릴스 주제 10개 뽑아줘"
   ```

2. **대본 작성**
   ```
   "30초짜리 릴스 대본 써줘. [분야] 친구톤으로."
   ```

3. **이미지/영상 소스**
   - 나노바나나(제미나이) 사용

4. **편집**
   - 캡컷 → 자동자막 + BGM 추가

5. **업로드**
   - 직접 업로드 또는 자동화 연결

---

## 6. 수익 구조

### 수익 채널 1: 쿠팡 파트너스 (자동 발생)

- **구조:** 스레드 글에 쿠팡 링크 자동 삽입 → 클릭 시 수익 발생
- **방식:** 스레드 21개 계정에서 인기글 수집 후 AI 리라이팅 + 쿠팡 링크 자동 교체
- **특징:** 시스템 구동 중 수동 개입 없이 수익 자동 발생

### 수익 채널 2: 인스타그램 광고 수익

- **구조:** AI 릴스 자동화로 팔로워 성장 → 대기업 광고 문의 → 광고비 수령
- **실적:** 2개월 반 만에 LG, CJ 등 대기업 광고 문의 4회+
- **계정 수:** 본계정 20만, 부계정 20만, 3번째 계정 1만+ 팔로워

### 수익 채널 3: 블로그 광고/제휴 수익

- **구조:** 블로그 6개 자동 운영 → SEO 트래픽 → 광고 수익
- **언급:** 직접적 수익 수치 미공개, 시스템 자동 발행 중 수익 발생 언급

### 수익 채널 4: 외부 협업 수익

- **구조:** AI 계정 성장 → 외부 기업에서 협업 문의
- **실적:** 마누스AI(Meta 운영)에서 유료 협업 문의 수신

### 수익 채널 5: 스레드 운영 대행 (언급)

- **구조:** 스레드 대행 월 800만원 가능성 언급
- "구매자가 만족할 만한 매출 내주면 대행비는 상관없다" 언급

### 전체 수익 구조 요약

```
콘텐츠 자동 생산 (AI)
    ↓

┌──────────────────────────────────┐

└──────────────────────────────────┘
```

## 7. 핵심 원칙

### 원칙 1: 수동 먼저, 자동화는 그 다음

> "자동화 먼저 하려는 사람은 100% 실패한다. 내가 증명했다."
> "수동으로 10번 → 패턴 발견 → 그다음 코드"
> "블로그도, 스레드도, 수동으로 하지 않고 자동화부터 만들었다가 3주를 날렸다"

이 순서를 틀리면 뭘 자동화하는지도 모른 채 시스템을 만들게 된다.

### 원칙 2: 복잡한 게 잘하는 게 아니다

> "처음엔 6단계짜리 복잡한 시스템 만들었는데 전부 갈아엎고 4단계로 줄였음."
> "복잡한 게 잘하는 게 아니라 단순한 게 진짜 실력임."

### 원칙 3: 공부가 아니라 실전

> "AI 강의 10개 사놓고 하나도 안 켠 사람, 그 돈으로 클로드 3개월 쓰는 게 100배 낫다."
> "강의 보는 시간에 만드는 사람이 이긴다."
> "프롬프트 강의 10개 듣는 것보다 거기 프롬프트 1개 써보는 게 정답이다."

### 원칙 4: 에러는 복붙이 정답

> "에러 나면 복붙. 그게 전부임."
> "에러 스크린샷 찍어서 붙여넣으면 텍스트 복붙보다 에러 해결 2배 빠름."

코딩 모르는 사람이 AI 자동화를 구축하는 실질적 방법: 에러가 나면 에러를 그대로 AI에게 보여주기.

### 원칙 5: 한 번에 다 하려면 포기한다

> "하나씩 추가하면 되는 거고, 한 번에 다 깔면 100% 포기함."

툴도, 자동화도 한 번에 전부 구축하려 하면 실패. 하나씩.

### 원칙 6: 잘하는 것보다 안 하는 것

> "1인 사업하면서 제일 크게 배운 것: '잘하는 것'보다 '안 하는 것'이 중요하다."
> "예전엔 콘텐츠도 만들고, 디자인도 하고, CS도 하고, 코딩도 했다."
> "지금은 콘텐츠 방향만 잡고 나머지는 전부 AI한테 넘겼다."

### 원칙 7: AI는 도구다

> "AI는 도구다. 도구로 돈 버는 게 아니라 도구로 시간을 산다."
> "그 시간으로 뭘 하느냐가 돈이 되는 거다."

### 원칙 8: 계산기부터 시작하라

> "안티그래비티 처음이면 '계산기 만들어줘'부터 시작해."
> "5분 만에 돌아가는 프로그램을 내가 만들었다는 경험이 생겨야 다음 단계로 넘어간다."
> "나도 첫 프로그램은 계산기였음. 지금은 노트북 10대 돌림."

### 원칙 9: 포기를 안 해서지 머리가 좋아서가 아니다

> "고졸 조선소 출신이 지금까지 살아남은 거? 포기를 안 해서지 머리가 좋아서가 아니다."
> "첫 번째는 무조건 노답임. 근데 그 노답을 만들어본 사람만 두 번째를 만들 수 있음."
