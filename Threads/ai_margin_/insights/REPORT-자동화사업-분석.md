## 4. 핵심 자동화 파이프라인 상세

### 4-1. 스레드 자동 포스팅 시스템 (21개 계정)

```
인기글 자동 수집 → AI 리라이팅 → 쿠팡 링크 교체 → 예약 포스팅
```

- 바이브코딩으로 구축
- 처음 6단계 → 4단계로 단순화
- 하루 50개 자동 포스팅
- 노트북 여러 대에 분산 실행

### 4-2. 블로그 자동 운영 (6개)

```
소재 수집 (RSS/뉴스) → AI 글 생성 → 자동 발행 (새벽)
```

- 주 40개+ 자동 발행
- 수동으로 직접 블로그를 운영해본 후 자동화함 (이 순서가 핵심)

### 4-3. 인스타 릴스 자동화 (2개 계정)

```
콘텐츠 소재 → 대본 생성 → 리모션으로 영상 자동 생성 → 업로드
```

- 2.5개월 만에 대기업 광고 문의 4건 (CJ, LG 등)
- 본계정 + 부계정 각 20만 팔로워 달성

### 4-4. 유튜브 자동 대본 추출기

```
유튜브 링크 입력 → 자막 자동 추출 → 핵심 요약 + 대본 구조화
```

- 기존 외주비: 건당 5만원 → 현재 0원
- 클로드코드로 30분 만에 제작

### 4-5. 벤치마킹 자동 수집기

```
경쟁 계정 목록 설정 → 매일 자동 수집 → 분석 데이터 축적
```

- 인스타 경쟁 계정 매일 분석

## 8. @ai_margin_ 의 핵심 철학 (재현 시 반드시 지킬 것)

### "수동 먼저, 자동화 나중"
> "자동화 먼저 하려는 사람은 100% 실패한다"
> "수동 10번 → 패턴 파악 → 자동화. 이 순서 틀리면 망함."

### "단순하게"
> "처음엔 6단계짜리 복잡한 시스템 만들었는데 전부 갈아엎고 4단계로 줄였음"
> "복잡한 게 잘하는 게 아니라 단순한 게 진짜 실력임"

### "공부보다 실행"
> "AI 강의 10개 사놓고 하나도 안 켠 사람 — 안티그래비티 켜고 '계산기 만들어줘' 한마디 쳤으면 벌써 3개 만들었음"

### "규모로 전환율 커버"
> "수동으로 100명 중 10명 결제 vs 자동으로 1만명 중 100명 결제 — 자동화가 낫다"
> "1만명 돌릴 시스템 짜면 그다음에 결제율만 올리면 된다"

### "코딩 = AI에게 시키기"
> "make/n8n은 제공하는 것만 됨, 코딩은 상상하는 것이 다 됨"
> "에러 나면 복붙. 그게 전부임."

- [ ] 리모션(Remotion) 세팅
