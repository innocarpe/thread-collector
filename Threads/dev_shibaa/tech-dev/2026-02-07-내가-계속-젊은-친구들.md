# 내가 계속 젊은 친구들

내가 계속 젊은 친구들
바이브코딩해서 돈 쓸어모으라고 하는 이유는,
코딩 이쪽은 '표절' 이런 개념이 없기 때문.

개발자들은 잘 만들어 놓은 기능을
'오픈소스' 형식으로 그냥 온라인에 올려놓는데,
그거 그냥 끌어 쓰면 기능 하나가 뚝딱 완성됨.

게다가 AI 코딩을 하는 에디터 '커서' 조차도
기존의 코드 에디터(VS코드)를 그냥 포크(끌어와서 수정)한 거고

커서 대항마로 불렸던 윈드서프도
VS코드를 포크한 것.

그리고 심지어 안티그래비티도
윈드서프를 포크한 것.
즉, VS코드를 포크한 윈드서프를 포크한 게 안티그래비티.

그러니까 머리 빨리 돌아가는 친구들은
불필요한 표절 시비나 창작 어쩌고
이런 데에 시간낭비 안 해도 되고

그냥 매일 1개 씩만 기존 서비스들 카피치고
레벨 올린다 생각하고 무한공장 돌리면 된다는 거임.

딱 30일만 죽었다 생각하고
공장 돌리면 성과 나옴.
