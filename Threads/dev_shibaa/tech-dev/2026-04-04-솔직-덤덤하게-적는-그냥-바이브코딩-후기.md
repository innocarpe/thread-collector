# 솔직 + 덤덤하게 적는 그냥 '바이브코딩' 후기

솔직 + 덤덤하게 적는 그냥 '바이브코딩' 후기

요즘 매일같이 신기술 나왔다면서
호들갑 떠는 계정들 많은데,
그런 거 싫어해서
그냥 본인이 한 것만 적겠음.

1. 최근까지도 그냥 외주 원툴로
웹사이트 제작 등을 했음.

하면서도 계속 느끼지만
바이브 코딩 좀만 깊게 파고,
고퀄 작업물들 계속 찾다보면 눈높이는 높아짐.

의무교육 무난하게 잘 받은 수준이라면
몸값 500만+ 이상 높이는 건 안 어려운듯..
거기에 영업력 있으면 기본 천만 원 대인 것 같고

근데 가끔씩 순서 거꾸로 하는 사람들이 있음.
2~3년 정도 시장에서 뺑뺑이 돌듯.

2. 암튼 외주만 하다보니 사이드로 돈이 안 돎.
매일 몸만 갈리는 느낌이 들어서

AI 콘텐츠 + 캐릭터 챗 느낌으로 웹 서비스 만들어서
결제 달음(승인을 안 받아서 간편결제도 없음)

마케팅도 그냥 스레드 계정 2개를
오픈클로에 끼워서 자동화 돌림.

아래는 해당 자동화 계정 오늘 트래픽 나온 거.

(이어짐)

자동화 스레드 계정 2개 끼워서
자동사냥으로 많으면 하루 24개씩 결제 나옴.

근데 바이럴은 결국 편차가 있어서
메타 광고를 돌려서 좀 더 얹어 가려고

단가 높이고 + 상품 추가하고 + 퍼널 만들고
작업 끝내서 오픈클로한테 던져놓음.

매출 얼마나 나올지 모르지만
일단 간편결제 붙는 순간부터 월 천 생각하고 있음.

3. 이제 기존 외주 건들도 좀 끝나가는 시기라,
새로 외주 받거나 새상품 오픈해서
좀 진득하게 꿀 빨려고 생각중.

사실 웹사이트 하나 만들어서
비슷하게 찍어내는 거 기획중인데

적당히 인스타 광고 떄려보고 결과 봐야 아는 거고
많이 나오면 월에 5천 이상 생각중인데
그건 아무도 모르는거고 그냥 해봐야 아름..

그렇다고 기존에 했던 커스텀 웹/SaaS 제작을 안 한다는 건 아니고
계속해서 고퀄 뽑아내는 작업은 이어갈려고 생각중.

암튼 나이 젊은 2030은 바이브코딩 하세요.
방구석에서도 성실하기만 하면 되고
