# 이런 사이트 1초만에 만드는 법

이런 사이트 1초만에 만드는 법

1. 바이브 코딩을 시작한다.
2. Spline에 접속한다.
3. 원하는 애니메이션을 클릭하고 URL을 복사한다.
4. 프롬프트에 URL을 넣고 풀스크린 히어로를 만들어달라고 한다.
5. 인터렉션 자동화 해달라고 요청한다.
6. 만능시바를 팔로우하고 꿀팁들을 얻는다.
