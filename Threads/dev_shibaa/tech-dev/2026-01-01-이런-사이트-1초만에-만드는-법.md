# 이런 사이트 1초만에 만드는 법

이런 사이트 1초만에 만드는 법

1. 바이브 코딩을 시작한다.
2. React Bits에 접속한다.
3. 원하는 애니메이션 및 컴포넌트를 클릭한다.
4. 소스코드를 복붙해서 프롬프트에 넣고 커스터마이징한다
5. 리엑트로 구현하지 않는다면 트랜스파일링 해달라고 한다.
6. 만능시바를 팔로우하고 꿀팁들을 얻는다.
