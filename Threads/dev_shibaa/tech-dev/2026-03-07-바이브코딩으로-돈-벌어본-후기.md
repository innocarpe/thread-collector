# 바이브코딩으로 돈 벌어본 후기

바이브코딩으로 돈 벌어본 후기

1. 밤낮 없이 웹사이트 만들고 단기간 월 천 찍음.
2. 근데, 월천 이후에 유지 보수(간단 수정)가 꽤 들어감.
3. 더 많이 벌기 시작하니, 혼자서는 무리라고 판단.
4. 지인/수강생분들 헬프 침. 이번 주 인건비만 200 넘김.
5. 도저히 안되겠다 싶어 서비스 운영 시작. 3일 제작하고 런칭.
6. 바로 결제 나오고 스레드 자동화 돌림.
7. 에이전시는 고정 수익 내고, 서비스 병행중.

답 다 줬다.

3월 첫째 주인데 인건비 실화.
개발 외주 그만 받으려고 갠톡도 닫음.

조만간 재정비해서 크몽 고단가 뚫을 예정.

토스한 외주 등은 카페에서 찾아볼 수 있음. 이상.
