# 카카오톡으로 결제한 ChatGPT Pro 제대로 쓰는 법

카카오톡으로 결제한 ChatGPT Pro 제대로 쓰는 법

1. 클로드봇을 위해 서버를 구매
현재 기준 가장 저렴한 서버 - 국내는 iwinv, 해외는 ultahost
*한국 IP 기준, 초저가인 카페24는 이미 재고 소진
*각각 2만 원대 초반/후반에 구매 가능

2. 클로드봇을 설치하고 GPT를 연결(auth0)

3. 클로드봇한테 스레드 자동화, 코딩, 스케쥴 관리 등 명령

결과 확인.
