# 실례지만 아직도 노션 쓰시는 분

실례지만 아직도 노션 쓰시는 분.. 없죠?

이 서비스는 각종 플랫폼 매출 내역부터
운영하는 네이버 카페 통계,
투두 리스트, 캘린더까지

자동으로 데이터를 갱신하며
AI가 모든 세팅을 도와줍니다.

누워서 '녹음' 버튼을 딸깍 하시면
AI가 내용을 정리해서 관리합니다.

원래는 PC 버전이 메인이지만
스레드 특성상 모바일 버전으로 올려봅니다.

직접 만든 것이며,
데이터는 보안을 위해 Mock Data로 처리했습니다.

실제론 저의 실시간 데이터가 표시됩니다.

바이브코딩하면 만드실 수 있습니다^^
