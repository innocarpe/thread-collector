# 양산형 유튜버들이 안 알려주는 꿀팁들 야금야금 잘 빼먹고 있음

양산형 유튜버들이 안 알려주는 꿀팁들 야금야금 잘 빼먹고 있음?

이거 알려줘도 괜찮냐
왜 알려주냐

이런 질문이 있어서 적어봄

1.꿀팁인 척 적어놓고 단톡방으로 유도한 뒤 전자책팔이 (정작 진짜 꿀팁은 안 풀어줌) ---> 여기에 개빡침 

2.어차피 알려줘도 재사용컨텐츠 정지라는 리스크가 있음

3.근데 진짜로 알려줘도 결국 계속 해왔던 사람 이기기 힘든 현실

4.유튜브는 서로 경쟁하는 구도가 아님 어떤 의미에서 보면 서로 경쟁이 맞는데 이거 다 알려주고 진입자들 많아져도 끊임없이 새로운 걸 연구하고 개발해서 만들어내야 함

5.나중에 쓰레드에서 풀었던 걸로 나도 전자책팔이 할까 고민중임 근데 팔로우하고 보는 사람들은 살 필요없음 거의 99%는 쓰레드로 다 풀어버릴거임

내가 아직 전자책팔이들이나 유튜브 고수처럼 월 천만원 이렇게 달성은 못했지만 올해안에 달성 할 수 있을 것 같음

참고로 나도 최소한 채널 50개는 말아먹고 시간 갈아넣으며 얻은 인사이트임
