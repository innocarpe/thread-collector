# 25년에 미장으로 돈 버신 분들 주목

25년에 미장으로 돈 버신 분들 주목.

이제 슬슬 양도세 (국세20%+지방세2%=22%) 내실 준비하고 계시죠?
아마 수백~수천만원 내실 분들도 계실텐데 아래 내용 참고하세요.

1. 양도세 얼마 내야함? => 증권앱에서 "양도세" 검색하면 조회 가능.
2. 분할납부 가능함? => YES. 단, 양도세 1천만원 초과 시에만 무이자 분납 가능 (7/31까지).
 - 양도세 1천~2천 : 1천만원 초과분 분납 가능
 - 양도세 2천 초과 : 전체 세액의 50% 까지 분납 가능
3. 카드 무이자할부 가능? => YES. 카드사별로 2~5개월 무이자 할부 운영. 심지어 지방세까지 할부 적용 가능함.
4. 카드 결제 시 수수료 있으니 (신용 0.7%, 체크 0.4%) 카드 혜택 고려해서 납부 방식 결정해야 함.
5. 카드로 결제한 양도세는 소득공제 대상은 아니지만 실적에 포함될 수 있음
 - 신한 Air platinum
 - 국민 마일리지가온카드
 - 국민 아시아나올림카드 등등
