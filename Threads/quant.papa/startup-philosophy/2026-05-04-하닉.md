# 하닉

하닉? 삼전? 비트코인? 샌디스크? 무턱대고 사지 마세요.

에버랜드는 사람 없을 때 가려고 그렇게 노력하시면서,
왜 주식은 사람 많을때 기어코 들어가려고 하시는지?

일단 매수버튼에서 손가락 내려놓고 생각 좀만 더 해보세요.

1. 나는 왜 쌀 때 안 사다가 잔뜩 오르니까 갑자기 사려고 하는가?
2. 남들도 사서 돈 버니까 배 아프고 FOMO가 와버린 건가?
3. 만약 산다면 나만의 익손절가가 존재하긴 하나?
4. 지수 최고점에서 현금을 다 태우는 위험한 짓을 하고 있진 않은가?

목적지도 모르는 급행열차에 소중한 자산을 태우는 우를 범하지 맙시다.
원칙도 없이 불나방처럼 따라가다가, 시장 흔들리면 정신 못 차려요.
"나만의 원칙" 없이는 함부로 시장에 돈을 태우지 마세요.
상승장에서는 현금 비중을 늘리고, 하락장에서 현금을 쓰세요.

제 원칙은 아래와 같습니다.

- 1원칙 : 현금이 30%(하락장)~50%(상승장) 확보되어 있는가?
- 2원칙 : 현 자산이 -30%가 되었을 때 대응할 전략이 준비되어 있는가?
- 3원칙 : 불나방 붙기 전에 싸게 사서 기다렸다가 불나방에게 잘 넘기고 있는가?

저는 하락론자가 아닙니다.
다만 자산을 안전하게 불릴 방법을 찾고 실천하려는 사람입니다.
