# 58900% 공감되는 글

58900% 공감되는 글.

평단은 진짜 전혀 중요하지 않음.
걍 평단은 보지도 말고 무시하는게 좋음. (수익률 체크 NO)

왜냐면 평단을 보게 되면 인간 심리 상,
- 사야할 때 못 사고,
- 팔아야할 때 못 파는 경우가 자주 생김. 

Case1. 추가상승여력이 있는데 익절 구간일때 : 불타기로 포지션 늘려야하는데 평단보다 비싸게 사기 싫어서 못 사고 그저 수익률에 만족하고 있음

Case2. 재료 소멸로 끝났는데 손해 구간일때 : 과감하게 손절해야 하는데 쓸데없이 물타기만 계속 해서 더 물리게 됨

절대로 평단을 기준으로 매수/매도 결정이 이뤄지면 안됨.
평단과 상관없이, 종목의 포텐셜과 시장의 흐름을 기준으로 이뤄져야 함.
