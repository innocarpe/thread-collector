# 오늘 글은 1000만원의 가치가 있다

오늘 글은 1000만원의 가치가 있다.

의식주 외에
사람들이 자발적으로 돈을 쓰는 이유는
거의 두 가지로 좁혀진다.

1.시간을 줄여주는 것
2. 재미를 주는 것

시간은 돈보다 귀하다

바쁜 사람들은 더 이상 직접 찾지 않는다.
귀찮음을 대신해주는 사람에게 돈이 간다

마트 대신 
쿠팡을 쓰고, 

긴 드라마 대신에
요약 리뷰 영상을 본다

이게 핵심이다.

콘텐츠든, 제품이든, 서비스든
“얼마나 시간을 아껴주느냐”가 가치다.

부자들은 돈의 ‘흐름’을 본다

“뭐가 유행이지?“보다
“사람들이 어떤 불편을 느끼고,
어떤 지점에서 돈을 쓰는가”를 본다.

부자가 되고 싶다면, 흐름을 읽어라

“시간을 아껴주는 것”
“생각할 필요 없게 해주는 것”
“선택을 덜어주는 것”
여기에 돈이 흐른다.

결국, 시간의 가치를 설계하는 사람이 돈을 번다.
