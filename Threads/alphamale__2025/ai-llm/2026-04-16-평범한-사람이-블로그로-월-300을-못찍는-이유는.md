# 평범한 사람이 블로그로 월 300을 못찍는 이유는

평범한 사람이 블로그로 월 300을 못찍는 이유는
스스로 주제를 생각하기 때문이다
앞으로 이렇게 기획해라

하트누르고 댓글로

콘텐츠 기획 단계에서는 제미나이(Gemini)나 클로드(Claude)를 먼저 쓰는 걸 추천

Gemini는 Google의 AI 어시스턴트로 글쓰기, 계획, 브레인스토밍, 초안 작성에 활용할 수 있고, Claude는 긴 문장 정리, 구조화, 분석형 글쓰기, 대본 초안 작성에 특히 강한 편이다

블로그 제목 후보 만들기, 유튜브 스크립트 초안 정리, 인스타 릴스 훅 문구 아이디어 뽑기까지 전부 활용도가 높다

활용하기 좋은 부분

1. 블로그 제목 10개 뽑기
2. 글 목차 구조 정리
3. 유튜브 대본 초안 작성
4. 인스타 캡션/훅 문구 아이디어 생성
5. 기존 글을 더 읽기 쉽게 다듬기
