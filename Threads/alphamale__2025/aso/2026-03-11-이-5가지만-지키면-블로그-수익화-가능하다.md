# 이 5가지만 지키면 블로그 수익화 가능하다

이 5가지만 지키면 블로그 수익화 가능하다

1. 네이버 대신 애드센스 블로그로 시작

2. 제목은 메인 키워드 + 연관 키워드 조합

3. 트래픽이 쏠리는 키워드를 잡아야 한다

4. 모은 트래픽을 전환시키면 바로 수익화다
