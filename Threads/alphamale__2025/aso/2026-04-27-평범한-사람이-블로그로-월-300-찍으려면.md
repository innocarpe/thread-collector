# 평범한 사람이 블로그로 월 300 찍으려면

평범한 사람이 블로그로 월 300 찍으려면 
글 쓸 때 이 구조는 무조건 알아야 한다 

리포하고 댓글로

1. 검색 의도 기반 글 작성

사용자가 왜 이 키워드를 검색했는지 먼저 파악해야 한다
단순 정보 탐색인지, 비교인지, 해결책 탐색인지에 따라
글 구조가 달라져야 한다

2. H1 / H2 / H3 구조 명확화

구글은 글의 주제 구조를 명확하게 이해하는 콘텐츠를 선호함
하나의 핵심 주제를 H1로 잡고, 세부 하위 주제를 H2
세부 설명을 H3로 정리하면 검색 엔진과 사용자 모두 읽기 쉬워진다

3. 내부 링크와 체류 시간 설계

관련 글을 자연스럽게 연결해 사용자가
한 페이지에서 끝나지 않도록 만들어야 한다
단순 클릭보다 세션 깊이와 체류시간이 중요하다
