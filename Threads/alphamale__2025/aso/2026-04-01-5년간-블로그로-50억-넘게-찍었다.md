# 5년간 블로그로 50억 넘게 찍었다

5년간 블로그로 50억 넘게 찍었다
딱 한 달만 이렇게 해라

하트누르고 댓글로

1주차 (1~7일) : 기초 세팅 + 즉시 신청

1. 네이버 아이디 3개 생성
2. 블로그스팟 5개 생성, 글 12개 발행
3. 승인용 글 작성 → 애드센스 신청
4. 네이버 블로그 3개 개설 → 애드포스트 신청
5. 네이버 블로그 1개: 쇼핑커넥트 즉시 운영 시작 
6. 황금비서 / 로워드 / 블랙키위 가입 후
키워드 20개 리스트 작성

2주차 (8~14일) : 콘텐츠 생산 + 백링크

1. 네이버 블로그 홈판용 글 3개 발행
2. 네이버 블로그 쇼핑커넥트 연계 글 3개 발행
3. 블로그스팟 수익형 글 5개 작성후
네이버 블로그에 리라이팅(백링크) 
4. 네이버 지식인 답변 30개 후
내 블로그 백링크 삽입

3,4주차 (15일~30일) : 본격적인 수익화

1. 애드포스트 승인 확인 + 첫 수익 확인 
2. 구글 애드센스 승인 확인
3. 쇼핑커넥트 커미션 수익 시작 확인
4. 유입 키워드 분석 → 유사 글 확장 작성
5. 2개월차 콘텐츠 캘린더 완성
