# 블로그 주제 5분만에 찾는 5가지 방법

블로그 주제 5분만에 찾는 5가지 방법 

교육생은 이미 이 방법으로 
수익화를 하고 있다

저장하고 댓글로

1. 네이버 크리에이터 어드바이저

1) 트렌드 -> 검색 유입 트렌드
2) 내 카테고리 선택
3) 현재 인기 검색어 TOP 20 확인

2. 네이버 연관 검색어

1) 네이버 검색창에 메인 키워드 입력
2) 자동완성 키워드 확인
3) 검색 결과 하단 연관검색어 확인
4) 각 키워드로 글 작성

3. 구글트렌드

1) Google Trends 사이트 접속
2) 관심 키워드 입력
3) 관련 검색어, 급상승 검색어 확인
4) 한국 + 최근 7일/30일 필터링
