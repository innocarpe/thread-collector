# 유튜브보다

유튜브보다 
훨씬 덜 일하고 
훨씬 더 쉽게 쌓이는 구조가 있다. 

<유튜브 vs 자동 확산 앱>

유튜브 : 콘텐츠 하나 만들려면 하루 소요
앱 : 콘텐츠 하나 만들려면 30분 소요
(나중엔 5분 컷 가능) 

유튜브 : 알고리즘 못 타면 끝 
앱 : 올리면 바로 유저에게 노출

유튜브 : 보고 떠나면 끝 
앱 :보고 공유하면서 확산됨 

유튜브 : 다음 콘텐츠 계속 고민 
앱 : 이미 검증된 콘텐츠 반복 

유튜브 : 할수록 더 바빠짐 
앱 : 할수록 할 일 줄어듦 

유튜브 : 계속 신경 서야 유지됨 
앱 : 올려놓고 일상 살면 됨 

결국 차이는 하나다 
내가 계속 해야 돌아가냐 
구조가 대신 돌아가냐 

근데 진짜 중요한 건 
이 앱이 다른 앱이란 
완전히 다르다는 점이다. 

그래서 더 늦기 전에 
알아두는 게 중요하다 

이 앱이 다른 앱이란 
완전히 다른 점을 공개하자면...

1. 누구나 좋아하는 콘텐츠만을 다룸 
2. 누구나 매일 필요한 콘텐츠만을 다룸
3. 누구나 공유하고 싶어하는 콘텐츠만을 다룸 
4. 누구나 댓글 달고 싶어하는 콘텐츠만을 다룸
5. 누구나 하트 달고 싶어하는 콘텐츠만을 다룸 
6. 이 모든 걸 무료로 사용자에게 제공함 
7. 결국 자동으로 앱이 성장하는 선순환 구조임

이 외에도 몇 가지 더 있는데 
다음글에서 계속 알려줌.
