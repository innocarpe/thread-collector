# 뉴스레터 시작전 Ghost로 SEO 최적화 연습

뉴스레터 시작전 Ghost로 SEO 최적화 연습

뉴스레터를 시작하기 위해 처음엔 

서브스택(Substack)과 고스트(Ghost)를 놓고
꽤 오래 고민했습니다.

결국 고스트를 선택하게 되었고 이유는 
월이용 금액을 지불하더라도

"뉴스레터 + 블로그 아카이브를 동시에 가져가고 싶다"

이런 이유로 Ghost를 선택하게 되었죠
물론 바이브 코딩으로 구현하는 것도 나쁘지 않겠지만

제가 바닥부터 쌓아올리는 것보다는 
해외 대형 서비스들이 이용하는
방법론을 택한 것이지만요

문제는 여기서 기본만을 따지는것이아니라 
몇몇가지 SEO 디테일을 잡는 중인데 죽겠네요

혹시 읽는 분들 중에 SEO 전문가분중 제가 빼먹었다 싶은
부분이 있다면, 알려주시면 감사할것 같습니다.

우선 전체적인 구조를 잡았습니다.

상단네비게이션을 잡은후
소개(/about/)
Home
제미나이 가이드 (/tag/gemini-guide/)
AI 가이드 (/tag/ai-guide/)
프롬프트 엔지니어링 (/tag/prompt-engineering/)

태그는 영어 slug로 정리하였죠 
URL을 짧고 명확하게가져가며 주요키워드를 태그로 설정했습니다.
물론 복합키워드를 고려한다면 몇몇가지 추가 작업이 들어가야하지만

우선은 이정도면 충분하다 판단했습니다.

타겟이 외국인이 아닌 SEO 불모지인 한국 이니까요

robots.txt, 세멘틱 HTML, XML Sitemap은 자동으로 잡아주는 걸로압니다.

저걸 설정하면서 발생했던 문제는 $15기준 
가장기본인 설정만을 사용할수있는데 

이 디자인의 경우 글을 안쓰면 404에러가 뜨더라구요
연결이 안된줄알고 40분을 날려먹었습니다.

어쩄든 여기까지는 확인했어요 두사이트에 접속해
크롤도 문제없고 빈페이지 없는것도 확인했고요

사이트 레벨(전역) 설정
사이트 메타 타이틀 / 설명을 잡으면서 전체적인 사이트의 구조와 무슨사이트 인지 설명 까지 넣었습니다.

마음같아선 영어로넣고 싶지만 우선 국문으로 넣었어요

다음으로 설정한건 OG 이미지죠

카톡, X, 슬랙, 디스코드에 도메인 링크 붙였을 때 
항상 썸네일이 뜨도록 설정한겁니다.

이건 조금더 수정이 필요한가 고민하고 있어요
