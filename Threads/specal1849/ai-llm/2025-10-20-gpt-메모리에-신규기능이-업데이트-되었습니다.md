# GPT 메모리에 신규기능이 업데이트 되었습니다

GPT 메모리에 신규기능이 업데이트 되었습니다.

이제는 자동으로 관리도 가능하고 세부적인 조절도 가능한데요

메모리는 잘쓰면 좋지만 잘못쓰면 할루시가 발생하는
양날의 검 이죠

저도 필요에 따라 키고 끄지만 이걸 잘쓰는 법을
알아봅시다.

간단하게 "AI가뭐야" 라는 질문을 한다고 해보죠
1번은 임시모드[튜닝아예없음]
2번은 제가 튜닝해논 메모리
3번은 프롬프트 생성기로 1차 편향성을 넣어준 녀석입니다.
꾸준히 말씀드리지만 이런 차이가 발생합니다.

일단 차이가 발생함은 명백히보이는데요 
쉽게말하면 시스템프롬-입력프롬 

이사이에 내 정보를 저장해 컨텍스트로 활용한다

더직관적으로 말하면 GPT가 기억하는 내정보라고 할수있겠죠
그럼 어떤종류가 있을까요?

Working Memory -체팅 기록참고
Persistent Memory -사용자 프로필 형태로 저장되는 설정값
이두가지가 존재하는데

Working Memory같은경우 아직관리가 불가능합니다 다만 이전대화에 참조를 하는 정도의 기능이고

이번에나온기능이Persistent Memory의 관리기능이에요
채팅기록참고-Working Memory 
저장된 메모리참고-Persistent Memory

그래서 이제 이걸 제어하는 기능이죠

사실 쉽게 관리할수 있었지만 이번에 이걸 추가적으로 편하게
관리할수 있는 기능을 추가한거죠

이중에서 필요한 메모리와 필요없는 메모리가 존재하는데

뭬예를 들면 아무필요없는 내용 ex)사용자는 예시로 고양이를 좋아함
이런것들이 녹아들어 있는 경우가 있거든요

그럼이제 원하지않는 예제가 나오곤 하니꼬박꼬박 관리해줘야하고
이를 잘해줘야한다 정도로 설명드릴수 있겠죠

이번에 나온건 이편의성을 올리는기능입니다.
