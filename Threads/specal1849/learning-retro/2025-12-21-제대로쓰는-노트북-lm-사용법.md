# [제대로쓰는 노트북 lm 사용법]

[제대로쓰는 노트북 lm 사용법]
제발 노트북 lm에 있는자료 다떄려박지 마세요...
노트북 lm을쓰는 이유가 뭔가요

공부하고 자료정리하고 핵심뽑아내려고 쓰시는거잔아요

좋다 나쁘다를 떠나 다 떄려박으면 
영상과같은 문제가 생깁니다.

겹치는 내용들로 넣으면 그나마 다행인데
그냥 다넣으면 어떤문제가 발생하는지 해결법에 대해이야기 해봅시다.

뭐 RAG DB 지루하고 복잡한 이론을 다떠나봅시다.

오늘제가 공부한논문은

From Code Foundation Models to Agents and Applications
A Comprehensive Survey and Practical Guide to Code Intelligence

란논문으로 AI코드작성이 어떤식으로 발전했는가 조망한논문입니다.

좌측은 303페이지를 한번에 처리한거고
우측은 마지막 컨클루전파트 60페이지만을 넣어둔거에요

무슨말씀인지 이해가 가시나요?

제일큰문제는 너무많은양을 처리하지 못한다는데 있습니다.

예를 들면 데이터에서 내가 조망하고 싶은부분은 
1-100페이지인데 결론부에 집중해야하고
뭔가 결과를 내야하는 LLM이가진 특성상

결국 후반부 자료가 섞여들어간다는 문제점이 존재하는것이지요

저또한 실제로 공부하는데 있어서는
선행으로 한번 쫙둘러보고

후행으로 PDF를 나눠서 넣습니다.
물론넣을때 챕터별로 끊어서 넣는게 가장좋지만

그게힘들다면 제발 30페이지씩이라도 짤라서 넣으세요...
그냥 다떄려박지말라고!!!!

다떄려박고 이상해요 라고하면 진짜 고양이가 뭅니다
캬아아아아악!!!!
