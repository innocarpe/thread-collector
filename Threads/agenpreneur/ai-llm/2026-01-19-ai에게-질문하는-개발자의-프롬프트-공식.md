# AI에게 질문하는 개발자의 프롬프트 공식

AI에게 질문하는 개발자의 프롬프트 공식

바이브코딩 망하는 지름길
"게시판 만들어줘" 

바이브코딩 성공하는 지름길
1.데이터 구조(Schema) 먼저 정의하기
2.필요한 API 명세서 AI에게 던져주기
3.UI 컴포넌트 단위로 쪼개서 요청하기

AI는 여러분의 손이지 뇌가 아닙니다. 뇌(설계)가 멈추면 손(코드)은 엉뚱한 곳을 긁습니다. 10년 차인 저도 설계 없이 AI 돌리면 100% 스파게티 코드 나옵니다.
