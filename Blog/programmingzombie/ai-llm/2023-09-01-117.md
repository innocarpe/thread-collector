특정인과의 카톡의 대화 내용을 가지고 학습을 시키거나, 내가 원하는 대화스타일의 내용을 주입하고 gpt에게 학습 시키면 그와 비슷한 느낌으로 대답하게 된다. StableDiffusion에서의 마치 LoRA와 같다고 할까 진작에 이미지 생성관련쪽은 Dreambooth도 있고 LoRA도 있으니 이제 대화형 모델에도 그와 비슷한것이 생긴거라 보면되겠다. 

 

이미지도 내가 원하는 이미지를 학습해서 특정 인물을 만들고, 대화 스타일도 학습시켜서 접목 시킨다면 정말 가상의 특정한 성격을 가진 인물을 누구나 쉽게 만들 수 있게 된 것이다. 

 

거기에 더할 수 있는 기술들이 너무 무궁무진하게 생겨나고 있는것 같다.

> 여기에 Whisper도 붙힌다면?

> 여기에 음성을 학습해서 특정 인물의 목소리로 대화할 수 있다면?

 

요즘 쏟아져 나오는 AI 소식들로 인해 만들고 싶은것들이 넘쳐나고 있는것 같다.

 

#### 조금더 생각에 꼬리를 물고 물어 생각해본 아이디어인데,  이런것도 만들 수 있지 않을까?

아래와 같은 데이터를 모두 저장하고 학습한다면?

> [ChatGPT Finetuning]  
- 나의 대화 내용  
- 채팅 내역  
  
[음성 데이터 별도 Finetuning]  
- 나의 음성 데이터  
  
[Vector DB]  
- 장기 기억 관련 데이터

위와 같이 데이터를 모아두고 LangChain 같은걸 활용해서 GPT에게 장기기억 관련 데이터는 벡터 DB에서 가져오고 적절한 내용으로 답변을 제공해준다면, 정말 하나의 인격체와 대화하는듯한 효과를 낼 수도 있을것 같다. 목소리로 학습된 데이터가 있다면 특정 사람의 목소리로 대화가 가능할지도 모르겠다.

 

위 내용을 바탕으로 GPT에게 아이디어를 가져오라고 했더니 아래와 같은 다양한 서비스 IDEA를 추천해주었다.

- **개인 비서 서비스**: 사용자의 대화 스타일, 목소리, 취향 등을 학습해 개인화된 비서를 제공할 수 있습니다.

- **커스텀 오디오북 제작**: 사용자의 목소리와 대화 스타일을 학습하여, 개인화된 오디오북을 생성합니다.

- **방송 라디오 DJ AI:** 특정 인물의 대화 스타일과 목소리를 학습시켜 라디오 DJ로 활용합니다. 특정 시간대마다 리퀘스트 노래와 소식을 전달합니다.

- **시나리오 작성 도우미**: 작가나 드라마 작성자가 원하는 대화 스타일에 맞춰 캐릭터 대사를 자동으로 생성해주는 서비스입니다.

- **메디컬 상담 AI**: 환자와 의사 간의 대화 데이터를 학습하여, 의료 상담을 하는 AI를 만듭니다.

- **친구 대화 모방 서비스**: 사용자의 친구와의 카톡 대화 내용을 학습하여, 해당 친구가 없을 때 대화 상대로 서비스를 제공합니다.

- **시장 분석 리포트 자동 생성기**: 사용자가 원하는 언어와 스타일로 주식이나 경제 리포트를 자동으로 작성해 주는 서비스입니다.

- **온라인 데이팅 어드바이저**: 사용자의 선호를 학습하여 이상적인 파트너에게 어떤 메시지를 보내면 좋을지 조언을 해주는 서비스입니다.

- **AI 언어 교육**: 사용자의 음성 데이터와 대화 스타일을 학습하여 맞춤형 언어 교육을 제공합니다.

- **가상 여행 가이드**: 사용자의 관심사와 대화 스타일에 맞춰 여행지를 추천하고 가상으로 동행하는 여행 가이드 서비스입니다

이제 글자 몇자만 적어도 쉽게 관련하여 좋은 정보와 아이디어를 얻을 수 있는 시대가 되었다.

확실한건 앞으로 더욱더 기술개발이 빠르게 진행될테고 재미있는것들도 더 많이 나올것이라는게 확실하다는것이다.

미리 준비를 하고 내것으로 만들어두는 경험과 노력이 필요한 시점이 아닐까 생각해본다.
