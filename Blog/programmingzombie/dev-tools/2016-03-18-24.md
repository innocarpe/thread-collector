### [UBUNTU SERVER] 노트북 덮개(LID) 닫을때 대기모드 진입 방지

 
 
 soulduse
  2016. 3. 18. 13:22
 
 
 

 

 /etc/systemd/logind.conf

  

이 파일을 바꾸면 된다. GUI에서는 쉽게 설정가능하지만, 서버에서는 sudo명령을 통해 권한을 받은 뒤 vim등으로 접근한다

  

```

#HandleLidSwitch=suspend -> HandleLidSwitch=ignore

```

  

주석을 나타내는 #을 제거뒤, 

```
poweroff
```

```
hibernate
```

```
ignore
```

```
lock
```

이 위의 네가지 속성이 있는데, 무시하려면 ignore가 맞고, 모니터만 꺼지게 하려면 lock의 속성으로 준다.
