### [안드로이드]시스템 영역에 APK 올리기

 
 
 soulduse
  2016. 5. 20. 09:15
 
 

 

 

아래와 같이 터미널에서 입력한다.

  

```
adb remount
adb push Launcher.apk /system/app/
adb shell chmod 644 /system/app/Launcher.apk
adb reboot
```
