### Ubuntu 22.04LTS Server 자동 로그인 하기

 
 
 soulduse
  2023. 9. 6. 09:15
 
 

 

 

최근 우분투 22.04LTS 버전으로 서버를 운영하려고 데스크탑을 한대 구매했다.

SSH로 붙어서 사용하려고 하는데 매번 서버를 reboot 할때마다 로그인을 해줘야해서 여간 귀찮은게 아니다.

 

다음과 같이 하면 로그인 없이 자동 로그인을 할 수 있다.

 

```
# 1. 터미널을 열고 다음 명령어를 입력합니다:
sudo mkdir -p /etc/systemd/system/getty@tty1.service.d/

# 2. 이어서 override.conf 파일을 생성하고 편집합니다:
sudo nano /etc/systemd/system/getty@tty1.service.d/override.conf

# 3. 편집창에서 아래와 같이 입력합니다:
[Service]
ExecStart=
ExecStart=-/sbin/agetty --autologin username --noclear %I $TERM

# 여기서 username은 자동으로 로그인하려는 사용자 계정 이름으로 변경해야 합니다.

# 4. 저장 후 시스템 데몬을 재로드하고, getty 서비스를 재시작합니다:
sudo systemctl daemon-reload
sudo systemctl restart getty@tty1.service
```

위 순서대로 따라하면 서버를 reboot 하더라도 자동으로 로그인되어 로그인 없이 바로 사용할 수 있게 된다.

 

이제 SSH로도 언제든 접근가능!
