### [안드로이드] adb에서 apk version 확인하기

 
 
 soulduse
  2016. 4. 26. 12:16
 
 

 

 

ADB로 상대방 디바이스에 붙어서 버전명을 알아봐야 하는 상황이 있었다.

  

아래의 명령어로 간단히 정보를 얻을 수 있다.

  

  

```
adb shell dumpsys package my.package | grep versionName
```
