## 핵심 주장 (≤5)
1. `String` 값은 `ByteArrayInputStream`을 사용해 `InputStream`으로 변환할 수 있다.
2. 변환한 `InputStream`은 `BufferedReader`와 `InputStreamReader`로 감싸서 읽을 수 있다.
3. 이렇게 만든 `BufferedReader`로 한 줄씩 읽어 콘솔에 출력할 수 있다.

## 공개된 숫자·지표
- 없음

## 언급된 도구·서비스
- Java: `String`을 `InputStream`으로 바꾸고 `BufferedReader`로 읽는 예제 코드의 실행 환경
- ByteArrayInputStream: `str.getBytes()`를 받아 `String`을 `InputStream`으로 변환하는 데 사용
- InputStream: 문자열을 바꿔 담는 중간 스트림 객체
- InputStreamReader: `InputStream`을 `BufferedReader`가 읽을 수 있게 감싸는 데 사용
- BufferedReader: 변환된 `InputStream`을 읽고 출력하는 데 사용

## 언급된 다른 creator·앱
- 없음

## 복제 가능한 전술 (≤3)
1. 외부 파일 없이 문자열 기반 입력을 테스트하는 인메모리 리더 경로를 만든다.
   - 구체적 스텝: 테스트용 `String`을 선언하고 `new ByteArrayInputStream(str.getBytes())`로 `InputStream`으로 변환한 뒤 `new BufferedReader(new InputStreamReader(is))`로 감싸서 `readLine()` 반복문으로 읽는다.
   - 예상 리소스: Java 개발 환경, 짧은 예제 코드 작성 시간
   - 예상 효과: 문자열 데이터를 `BufferedReader` 입력처럼 다루며 읽기 로직을 바로 확인할 수 있다.

## 원문 요약 (≤5문장)
이 글은 Java에서 `String` 값을 `InputStream`으로 바꾼 뒤 `BufferedReader`로 읽는 간단한 예제를 보여준다. 작성자는 `ByteArrayInputStream`에 `str.getBytes()`를 넣어 문자열을 스트림으로 변환한다. 이후 `InputStreamReader`와 `BufferedReader`를 순서대로 감싸서 읽기 객체를 만든다. 마지막으로 `readLine()` 반복문으로 내용을 출력하고 `BufferedReader`를 닫는다.

## 본문 포인트별 발췌
> A simple Java program to convert a **String to InputStream**, and use `BufferedReader` to read and display the converted`InputStream`.
> // convert String into InputStream
> InputStream is = new ByteArrayInputStream(str.getBytes());
> // read it with BufferedReader
> BufferedReader br = new BufferedReader(new InputStreamReader(is));
