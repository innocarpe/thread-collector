## 핵심 주장 (≤5)
1. 코틀린은 자바 플랫폼에서 동작하며 자바와의 상호운용성을 중시하는 언어다.
2. 코틀린은 기본값, 널 허용 타입, 람다식 같은 문법으로 더 간결하게 코드를 작성할 수 있다.
3. 코틀린은 정적 타입 지정 언어이지만 타입 추론을 지원해 모든 타입을 직접 선언할 필요는 없다.
4. 코틀린의 함수형 프로그래밍 특성은 코드를 간결하고 가독성 있게 만들며 다중스레드 환경에서도 안전성을 높인다.
5. 코틀린은 무료 오픈소스이며 아파치 2 라이선스로 제공된다.

## 공개된 숫자·지표
- 없음

## 복제 가능한 전술 (≤3)
1. 언어 학습 초반에는 짧은 예제를 직접 뜯어보며 문법 요소를 단계별로 분해해 이해한다.
   - 구체적 스텝: `data class`, 기본값, 널 허용 타입, `listOf`, `maxBy`, 람다식처럼 한 예제에 들어간 요소를 한 줄씩 분해하고 각 표현식의 역할을 한국어로 다시 적는다.
   - 예상 리소스: 짧은 예제 코드 1개, 코틀린 실행 환경, 학습 시간
   - 예상 효과: 당장 전체 코드가 이해되지 않아도 흐름을 먼저 파악하고 핵심 문법을 빠르게 익힐 수 있다.
2. 표준 라이브러리 함수의 내부 구현을 확인해 추상화 뒤의 동작을 이해한다.
   - 구체적 스텝: 예제에서 사용한 `maxBy` 같은 함수를 선택하고 라이브러리 구현을 열어 반복 구조, 반환 조건, 비교 방식이 어떻게 되어 있는지 확인한다.
   - 예상 리소스: 코틀린 표준 라이브러리 소스, IDE 또는 저장소 브라우저, 학습 시간
   - 예상 효과: 고수준 함수 호출이 실제로 어떻게 동작하는지 이해해 람다식과 컬렉션 API 사용 감각을 높일 수 있다.
3. 공식 채널에서 기술 채택 신호를 확인해 학습 우선순위를 정한다.
   - 구체적 스텝: 플랫폼 공식 홈페이지와 기술 발표 영상을 확인해 특정 언어나 기술이 메인 페이지, 예제, 발표 영상에서 얼마나 강조되는지 본다.
   - 예상 리소스: 공식 문서 사이트, 발표 영상, 확인 시간
   - 예상 효과: 어떤 기술이 실제로 밀리고 있는지 판단해 학습할 기술의 우선순위를 정할 수 있다.

## 원문 요약 (≤5문장)
글은 코틀린을 자바 플랫폼에서 동작하고 자바와의 상호운용성을 중시하는 언어로 소개한다.
작성자는 `Person` 예제를 통해 기본값, 널 허용 타입, `maxBy`, 람다식의 `it`, 엘비스 연산자 `?:`를 설명한다.
또한 코틀린이 정적 타입 지정 언어이면서도 타입 추론을 지원한다고 정리한다.
함수형 프로그래밍과 객체지향 프로그래밍을 함께 지원하며, 불변성과 순수 함수가 코드 가독성과 멀티스레드 안전성에 도움이 된다고 평가한다.
마지막으로 코틀린이 무료 오픈소스이며 구글 안드로이드 공식 채널에서도 적극적으로 밀리고 있다고 본다.

## 본문 포인트별 발췌
> 자바 플랫폼에서 돌아가는 새로운 프로그래밍 언어.
> 자바와의 상호운용성을 중시 한다.
> 코틀린이 알아서 Int 타입임을 자동으로 추론해준다.
> 소스를 간결하고 가독성있게 바꿀수 있다.
> 개발은 깃헙(https://github.com/jetbrains/kotlin) 코틀린 프로그래밍 커뮤니티 기여에 대해 열려 있다.
