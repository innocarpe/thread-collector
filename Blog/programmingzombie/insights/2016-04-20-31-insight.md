## 핵심 주장 (≤5)
1. Intellij에서 복붙한 코드에 `Found duplicated code in ...` 노란줄이 나타날 수 있다.
2. 이 경고가 눈에 거슬린다면 `Inspections`의 `Duplicated Code` 체크를 해제해 해결할 수 있다.
3. 해당 설정을 끄면 노란선이 사라진 코드를 볼 수 있다.

## 공개된 숫자·지표
- 없음

## 언급된 도구·서비스
- Intellij: 코딩 중 `Found duplicated code in ...` 경고가 나타나는 개발 도구로 언급됨

## 언급된 다른 creator·앱
- 없음

## 복제 가능한 전술 (≤3)
1. Intellij의 중복 코드 검사 경고를 비활성화해 에디터의 노란줄을 제거한다.
   - 구체적 스텝: `File -> Settings -> Editor -> Inspections`로 이동한 뒤 `General -> Duplicated Code` 체크를 해제한다.
   - 예상 리소스: 도구는 Intellij, 비용은 없음
   - 예상 효과: `Found duplicated code in ...` 관련 노란선이 사라져 더 깔끔한 코드 화면을 볼 수 있다.

## 원문 요약 (≤5문장)
글은 Intellij에서 코딩하다가 복붙한 코드에 `Found duplicated code in ...`라는 노란줄이 나타나는 상황을 다룬다. 작성자는 이 메시지가 상당히 거슬린다고 설명한다. 해결책으로 `File -> Settings -> Editor -> Inspections` 경로에서 `General -> Duplicated Code` 체크를 해제하라고 안내한다. 그렇게 하면 노란선이 사라진 깔끔한 코드를 볼 수 있다고 정리한다.

## 본문 포인트별 발췌
> **Intellij **에서 코딩을 하다보면 가끔식 복붙 신공을 할 때 다음과 같은 노란줄을 볼 수 있다.
> Found duplicated code in ...
> 라는 메세지가 출력되는데 여간 눈에 거슬리는게 아니다.
> 해결 방법은 다음과 같다.
> General –> Duplicated Code 체크를 해제 해준다.
