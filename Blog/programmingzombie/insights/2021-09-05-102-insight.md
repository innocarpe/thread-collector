## 핵심 주장 (≤5)
1. EditText에서 숫자를 입력할 때 콤마를 자동으로 붙이려면 Kotlin Extension으로 간단하게 구현할 수 있다.
2. `androidx.core` 라이브러리의 `doOnTextChanged` Extension을 활용하면 EditText 변경 내용을 바로 받아 처리할 수 있다.
3. EditText에 작성한 함수만 적용하면 사용법은 끝날 정도로 적용 방식이 단순하다.
4. `androidx.core` 내부에는 유용한 Extension 함수가 많아 직접 살펴보는 것도 도움이 된다.

## 공개된 숫자·지표
- `androidx.core:core-ktx` 버전: `1.6.0` (기간: 명시 없음, 출처: `implementation 'androidx.core:core-ktx:1.6.0'`)

## 언급된 도구·서비스
- Kotlin Extension: EditText에 콤마 formatting 기능을 확장 함수로 붙이는 데 사용됨
- EditText: 숫자 입력 시 콤마가 나오도록 처리하는 대상 UI 컴포넌트
- androidx.core: `doOnTextChanged` Extension을 사용하기 위해 추가한 라이브러리
- Gradle: app 단위 의존성에 `androidx.core:core-ktx:1.6.0`를 추가하는 데 사용됨
- `doOnTextChanged`: EditText 변경 시점의 텍스트를 바로 받아 콤마 가공 로직을 실행하는 Extension 함수

## 복제 가능한 전술 (≤3)
1. EditText 전용 콤마 formatting 확장 함수를 만들어 입력 중 실시간으로 숫자 포맷을 교정한다.
   - 구체적 스텝: `androidx.core:core-ktx` 의존성을 추가한 뒤 `EditText.initComma` 같은 확장 함수를 만든다; `doOnTextChanged`에서 빈 값과 중복 변경을 제외한다; 입력값에서 콤마를 제거한 문자열을 다시 콤마 포맷으로 변환한다; 필요하면 원본 숫자를 listener로 전달하고 `setText`와 `Selection.setSelection`으로 커서를 끝으로 맞춘다.
   - 예상 리소스: Android Kotlin 프로젝트, Gradle 수정, `androidx.core:core-ktx:1.6.0`
   - 예상 효과: EditText에서 숫자를 입력할 때 콤마가 자동으로 표시된다.
2. 라이브러리 내부 Extension 구현을 참고해 입력 처리 코드를 더 빠르게 작성한다.
   - 구체적 스텝: `androidx.core.widget`의 `doOnTextChanged` 구현을 확인한다; TextWatcher를 직접 구현할지 Extension으로 감쌀지 결정한다; 필요한 입력 처리 로직에 같은 패턴을 재사용한다.
   - 예상 리소스: `androidx.core` 소스 코드 열람
   - 예상 효과: EditText 변경 내용을 바로 가져오는 방식으로 간단한 입력 후처리 기능을 구현할 수 있다.

## 원문 요약 (≤5문장)
이 글은 Android Kotlin에서 EditText 숫자 입력값에 콤마를 자동으로 붙이는 방법을 소개한다. 작성자는 Kotlin Extension으로 `initComma` 함수를 만들어 `doOnTextChanged` 시점에 입력값을 가공하는 방식을 제안한다. 이를 위해 `androidx.core:core-ktx:1.6.0` 의존성을 추가하고, 텍스트 변경 후 콤마 제거 및 재포맷, 커서 위치 보정을 수행한다. 마지막으로 `androidx.core` 내부에 유용한 Extension 함수가 많으니 직접 살펴보는 것도 좋다고 덧붙인다.

## 본문 포인트별 발췌
> EditText에서 숫자를 입력하면, 콤마가 나오게 하고 싶을때가 있다.
> Kotlin Extension을 활용하여 간단하게 만들어 보았다.
> androidx.core 라이브러리를 사용하면 EditText에 변경된 내용만 바로 가져오는 Extension 코드가 있는데 이를 활용하였다.
> app 단위에 Gradle 에 다음과 같은 의존성을 추가해주도록 하자
> androidx.core 라이브러리 내부에는 다양한 유용한 Extension 함수들이 많으니 내부를 물고, 뜯고 맛봐도 좋은 경험이 된다.
