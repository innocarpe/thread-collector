## 핵심 주장 (≤5)
1. 안드로이드 버튼 터치/클릭 시 ripple effect는 뷰 XML에 속성을 추가하는 방식으로 줄 수 있다.
2. ripple effect는 `drawable` 폴더에 별도 XML을 만들어 색상, 모양, 타겟 버전 등을 커스텀해서 적용할 수 있다.
3. 커스텀 ripple XML을 만든 뒤 해당 뷰의 `android:background`에 연결하면 효과를 완성할 수 있다.
4. 더 자세한 정보는 Android 공식 문서의 `RippleDrawable`을 참고할 수 있다.

## 공개된 숫자·지표
- 없음

## 언급된 도구·서비스
- Android View XML: 뷰에 `android:clickable="true"`와 `android:background="?attr/selectableItemBackground"`를 넣어 ripple effect를 주는 데 사용
- `drawable` 폴더: `ripple_effect.xml` 파일을 만들어 effect를 커스텀하는 위치로 사용
- `RippleDrawable` 공식 문서: ripple effect 관련 상세 정보를 참고하는 자료로 언급

## 복제 가능한 전술 (≤3)
1. 기본 selectable background로 클릭 시 ripple effect를 즉시 적용한다.
   - 구체적 스텝: ripple 효과를 줄 View XML을 열고 `android:clickable="true"`와 `android:background="?attr/selectableItemBackground"`를 추가한다.
   - 예상 리소스: XML 편집기, 안드로이드 레이아웃 파일 수정 시간
   - 예상 효과: 버튼 터치/클릭 시 번짐효과를 빠르게 적용할 수 있다.
2. 커스텀 ripple drawable을 만들어 원하는 스타일의 효과를 적용한다.
   - 구체적 스텝: `drawable` 폴더에 `ripple_effect.xml`을 만들고 `<ripple>` 내부에 색상과 배경 shape를 정의한 뒤, 대상 뷰의 `android:background`를 `@drawable/ripple_effect`로 지정한다.
   - 예상 리소스: `drawable` XML 파일 1개, 색상 리소스, 안드로이드 XML 편집기
   - 예상 효과: 원하는 색상과 모양, 타겟 버전 등을 커스텀한 ripple effect를 뷰에 적용할 수 있다.

## 원문 요약 (≤5문장)
글은 안드로이드에서 버튼 터치나 클릭 시 ripple effect를 주는 방법을 정리한다. 첫 번째 방법은 View XML에 `clickable`과 `selectableItemBackground`를 지정하는 것이다. 두 번째 방법은 `drawable` 폴더에 ripple XML을 만들어 색상과 배경 모양을 커스텀하는 방식이다. 이후 해당 뷰의 background에 커스텀 drawable을 연결하면 효과를 적용할 수 있다고 설명한다. 추가 정보는 Android 공식 문서에서 확인할 수 있다고 덧붙인다.
