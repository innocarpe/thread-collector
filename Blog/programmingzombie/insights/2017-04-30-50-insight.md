## 핵심 주장 (≤5)
1. 개인 앱 개발 중 네트워크 연결상태에 따라 구분 처리해야 하는 상황이 있었고, 이를 `ConnectivityManager` 기반 메소드로 해결할 수 있다.
2. 모바일 인터넷 연결 여부는 `cm.getActiveNetworkInfo() != null` 결과를 boolean 값으로 반환하는 방식으로 체크할 수 있다.
3. 네트워크 상태 또는 와이파이 연결 체크를 위해 Manifest에 `ACCESS_WIFI_STATE`와 `ACCESS_NETWORK_STATE` 퍼미션을 추가해야 한다.
4. `InetAddress.getByName("google.com")`를 활용해 디바이스가 인터넷에 연결되어 있을 때 네트워크 상태를 체크하는 다른 방식도 있다.

## 공개된 숫자·지표
없음

## 언급된 도구·서비스
- Android `ConnectivityManager`: 모바일 인터넷 연결 여부를 체크하는 메소드 구현에 사용됨
- Android Manifest: 네트워크 상태 또는 와이파이 연결 체크를 위한 퍼미션 추가 위치로 사용됨
- `InetAddress`: 디바이스의 인터넷 연결 상태를 다른 방식으로 체크하는 코드에 사용됨
- Google.com: `InetAddress.getByName("google.com")` 호출 대상으로 사용됨
- Stack Overflow: 글의 해결 방법 출처로 언급됨

- 없음

## 복제 가능한 전술 (≤3)
1. 앱에서 네트워크 연결 여부를 사전 확인하는 유틸 메소드를 추가한다.
   - 구체적 스텝: `ConnectivityManager`를 `getSystemService(Context.CONNECTIVITY_SERVICE)`로 가져온 뒤 `getActiveNetworkInfo() != null` 결과를 반환하는 메소드를 작성하고, 네트워크 상태에 따라 분기해야 하는 화면이나 기능 진입 전에 호출한다.
   - 예상 리소스: Android 앱 코드 수정, Manifest 접근 권한, 짧은 구현 시간
   - 예상 효과: 원문처럼 네트워크 연결상태에 따라 구분하여 처리할 수 있다.
2. 네트워크/와이파이 상태 체크용 퍼미션을 Manifest에 명시한다.
   - 구체적 스텝: `AndroidManifest.xml`에 `android.permission.ACCESS_WIFI_STATE`와 `android.permission.ACCESS_NETWORK_STATE`를 추가한 뒤, 관련 네트워크 체크 코드가 정상 동작하는지 확인한다.
   - 예상 리소스: Manifest 수정, Android 프로젝트 권한 설정
   - 예상 효과: 원문에서 제시한 네트워크 상태 또는 와이파이 연결 체크가 가능해진다.
3. 보조적으로 호스트 조회 기반 인터넷 연결 확인 방식을 적용한다.
   - 구체적 스텝: `InetAddress.getByName("google.com")`를 호출하는 `isInternetAvailable()` 형태의 메소드를 추가하고, 예외 발생 시 `false`를 반환하도록 처리한다.
   - 예상 리소스: Android/Java 네트워크 코드 작성
   - 예상 효과: 원문에서 제시한 아래의 방식으로도 디바이스 인터넷 연결 시 네트워크 상태를 체크할 수 있다.

## 원문 요약 (≤5문장)
글쓴이는 개인 앱을 만들면서 네트워크 연결상태에 따라 다르게 처리해야 하는 상황을 겪었다. 이를 해결하는 방법으로 `ConnectivityManager`를 사용해 활성 네트워크 정보 존재 여부를 boolean으로 반환하는 메소드를 소개한다. 또한 네트워크 상태나 와이파이 연결 체크를 위해 Manifest에 두 가지 퍼미션을 추가해야 한다고 설명한다. 추가로 `InetAddress.getByName("google.com")`를 이용해 인터넷 연결 여부를 확인하는 대안도 제시한다. 해당 방법의 출처는 Stack Overflow라고 밝힌다.

## 본문 포인트별 발췌
> 개인 앱을 만들고 있는 도중 네트워크 연결상태에 따라 구분하여 처리해야하는 상황이 있었다.
> 이를 어떻게 해결할까 찾아보던 도중 다음과 같이 해결 할 수 있었다.
> 이 메소드는 모바일 인터넷이 연결 되었는지 체크하고 난 뒤 값을 연결됬는지 값을 boolean 값으로 반환하는 메소드 이다.
> 네트워크 상태 또는 와이파이 연결 체크를 위해 Manifest또한 아래 퍼미션을 추가해줘야 한다.
> 또한 아래의 방식으로도 디바이스가 인터넷에 연결 되어있을때 네트워크 상태를 체크할 수 있다.
