## 핵심 주장 (≤5)
1. 카카오링크 공유 버튼에 `setAndroidExecutionParams`로 파라미터를 붙이면 앱 실행 시 원하는 상세 페이지로 바로 이동시킬 수 있다.
2. KakaoLink의 FeedTemplate를 사용하면 콘텐츠 정보와 소셜 지표, 앱 실행 버튼을 함께 담아 공유 메시지를 구성할 수 있다.
3. 공유 링크로 앱이 열릴 때 `MainActivity`에서 `Intent.ACTION_VIEW`와 URI query parameter를 읽어 상세 화면으로 라우팅하면 된다.

## 공개된 숫자·지표
- 없음

## 언급된 도구·서비스
- KakaoLink: 앱 공유와 공유 링크 기반 딥링크 진입 구현에 사용
- Kakao Developers 문서: KakaoLink 시작 전 설정값과 Android 연동 방법 참고용으로 사용
- FeedTemplate: 카카오링크 공유 메시지 템플릿으로 사용
- AndroidManifest.xml: 딥링크 수신용 `intent-filter` 설정에 사용
- MainActivity: 공유 링크 진입 시 파라미터를 받아 상세 페이지로 보내는 진입점으로 사용
- DetailActivity: 전달받은 게시판 번호에 따라 바로 보여줄 상세 화면으로 사용

## 언급된 다른 creator·앱
- KakaoTalk: 카카오링크 공유가 실제로 전달되는 앱으로 언급됨

## 복제 가능한 전술 (≤3)
1. 카카오링크 버튼에 딥링크 파라미터를 실어 특정 콘텐츠 상세로 바로 진입시키기
   - 구체적 스텝: `FeedTemplate`의 버튼 `LinkObject`에 `.setAndroidExecutionParams("board_id=${board.id}")`를 추가하고, 공유 버튼 클릭 시 `KakaoLinkService.getInstance().sendDefault(...)`로 전송한다.
   - 예상 리소스: Android 앱 코드 수정, KakaoLink 기본 설정 완료 상태, Kakao Developers 문서 참고
   - 예상 효과: 공유받은 사용자가 앱에서 바로 확인을 눌렀을 때 메인 화면이 아니라 특정 게시물 상세 페이지로 이동할 수 있다.
2. Android 딥링크 intent-filter로 카카오링크 실행을 수신하기
   - 구체적 스텝: `AndroidManifest.xml`의 대상 액티비티에 `VIEW`, `DEFAULT`, `BROWSABLE` 카테고리를 포함한 `intent-filter`와 `scheme`, `host` 데이터를 추가한다.
   - 예상 리소스: `AndroidManifest.xml` 수정, KakaoLink 시작 전 설정값
   - 예상 효과: 카카오링크를 통해 앱이 열릴 때 `MainActivity`가 `ACTION_VIEW` 형태로 실행되어 후속 라우팅 처리가 가능해진다.
3. 앱 진입 시 query parameter를 읽어 상세 화면으로 라우팅하기
   - 구체적 스텝: `MainActivity`에서 `if(intent.action == Intent.ACTION_VIEW)`를 검사하고, `intent.data.getQueryParameter("board_id")`로 값을 꺼낸 뒤 해당 ID를 사용해 `DetailActivity`를 실행한다.
   - 예상 리소스: `MainActivity` 라우팅 코드, 상세 화면 이동 함수
   - 예상 효과: 사용자가 공유 링크를 눌렀을 때 메인 화면에서 멈추지 않고 바로 원하는 상세 페이지를 볼 수 있다.

## 원문 요약 (≤5문장)
글은 Android 앱에서 KakaoLink를 이용해 앱을 공유하고, 공유된 링크를 통해 앱 내부의 특정 페이지로 바로 이동시키는 방법을 설명한다. 작성자는 FeedTemplate 기반 공유 코드에서 `setAndroidExecutionParams`로 `board_id`를 전달하는 부분을 핵심으로 짚는다. 이후 `AndroidManifest.xml`의 `intent-filter` 설정으로 `MainActivity`가 `ACTION_VIEW` 형태의 실행을 받도록 구성한다. 마지막으로 `MainActivity`에서 URI query parameter를 읽어 `DetailActivity`로 이동시키면 딥링크가 완성된다고 설명한다.

## 본문 포인트별 발췌
> 1. 카카오링크를 사용해서 내 앱을 공유한다.
> 2. 공유된 링크를 타고 앱으로 접근하면 내가 보여주고자 하는 특정 페이지로 바로 이동시킨다.
> 데이터 형식은 "key1=value1" 로 작성하면된다.
> 아래에서는 파라미터를 board_id=id 값으로 넘겨주었다.
> Intent 값이 ACTION_VIEW 형태로 실행되었는지 체크하고, uri로부터 parameter를 가져온다.
