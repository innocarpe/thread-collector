## 핵심 주장 (≤5)
1. GPT 3.5 파인튜닝으로 원하는 대화 스타일을 주입해 비슷한 느낌의 응답을 만들 수 있다.
2. 이미지 학습과 대화 스타일 학습을 결합하면 가상의 특정 성격을 가진 인물을 쉽게 만들 수 있다.
3. Whisper, 음성 학습, 벡터 DB, LangChain을 결합하면 장기기억을 가진 하나의 인격체처럼 대화하는 시스템을 만들 가능성이 있다.
4. AI 발전으로 짧은 입력만으로도 좋은 정보와 서비스 아이디어를 쉽게 얻을 수 있는 시대가 되었다.
5. 앞으로 기술 발전 속도가 더 빨라질 것이므로 미리 준비하고 직접 만들어보는 경험이 중요하다고 본다.

## 공개된 숫자·지표
- 없음

## 언급된 도구·서비스
- ChatGPT: GPT 3.5 버전 파인튜닝 대상으로 언급되며 원하는 대화 스타일을 학습시키는 용도
- GPT 3.5: 파인튜닝 가능한 버전으로 언급됨
- StableDiffusion: 이미지 생성 쪽에서 원하는 이미지를 학습시키는 비교 사례로 언급됨
- LoRA: 이미지 생성에서 특정 스타일·인물 학습의 비유로 언급됨
- Dreambooth: 이미지 생성 관련 학습 도구 사례로 언급됨
- Whisper: 음성을 붙여 대화 시스템에 확장할 수 있는 기술로 언급됨
- Vector DB: 장기 기억 관련 데이터를 저장하고 가져오는 용도로 언급됨
- LangChain: GPT가 벡터 DB에서 장기기억 데이터를 가져오도록 연결하는 도구로 언급됨

## 언급된 다른 creator·앱
- OpenAI: GPT-3.5 Turbo 파인튜닝 및 가격 페이지 링크의 주체

## 복제 가능한 전술 (≤3)
1. 개인 대화 스타일을 반영한 대화형 모델을 만든다.
   - 구체적 스텝: 자신의 대화 내용과 채팅 내역을 모은 뒤, 원하는 말투와 스타일이 드러나는 데이터만 선별해 ChatGPT 파인튜닝 학습 데이터로 정리하고, 결과 모델이 비슷한 느낌으로 답변하는지 테스트한다.
   - 예상 리소스: ChatGPT GPT 3.5 파인튜닝, 대화 로그 정리 작업, API 사용 비용
   - 예상 효과: 내가 원하는 대화스타일의 내용을 주입하고 학습 시키면 그와 비슷한 느낌으로 대답하게 된다.
2. 장기 기억을 붙인 페르소나형 AI를 설계한다.
   - 구체적 스텝: 대화 데이터는 파인튜닝용으로, 장기 기억 관련 정보는 Vector DB용으로 분리 저장한 뒤, LangChain 같은 도구로 질의 시 관련 기억을 불러와 GPT 응답에 함께 반영되도록 구성한다.
   - 예상 리소스: 파인튜닝용 대화 데이터, Vector DB, LangChain, GPT API
   - 예상 효과: 장기기억 관련 데이터는 벡터 DB에서 가져오고 적절한 내용으로 답변을 제공해준다면, 정말 하나의 인격체와 대화하는듯한 효과를 낼 수도 있을것 같다.
3. 텍스트 페르소나에 음성과 STT를 결합해 특정 인물형 인터페이스를 확장한다.
   - 구체적 스텝: 대화 스타일 파인튜닝 결과에 음성 데이터를 별도로 학습시키고, Whisper 같은 음성 입력 처리를 붙여 사용자가 말하면 특정 인물의 목소리와 말투로 답하는 흐름을 설계한다.
   - 예상 리소스: 음성 데이터, Whisper, 음성 학습 도구, 대화형 모델
   - 예상 효과: 특정 사람의 목소리로 대화가 가능할지도 모르겠다.

## 원문 요약 (≤5문장)
글쓴이는 ChatGPT의 GPT 3.5 파인튜닝 공개를 계기로, 개인 대화 스타일을 학습한 대화형 AI가 빠르게 현실화되고 있다고 본다. 이미지 생성에서 LoRA나 Dreambooth가 해온 역할이 이제 대화형 모델에도 확장됐다고 해석한다. 여기에 Whisper, 음성 학습, Vector DB, LangChain을 결합하면 장기 기억과 특정 목소리를 가진 인격형 AI도 가능할 수 있다고 본다. 또한 GPT에게 아이디어를 요청해 개인 비서, 오디오북, 라디오 DJ AI 같은 다양한 서비스 구상을 얻었다고 말한다. 결론적으로 AI 발전 속도가 매우 빠르므로 미리 준비하고 직접 만들어보는 경험이 중요하다고 주장한다.

## 본문 포인트별 발췌
> 얼마전부터 ChatGPT에서 GPT 3.5 버전으로 파인튜닝을 할 수 있게 되었다.
> 특정인과의 카톡의 대화 내용을 가지고 학습을 시키거나, 내가 원하는 대화스타일의 내용을 주입하고 gpt에게 학습 시키면 그와 비슷한 느낌으로 대답하게 된다.
> 이미지도 내가 원하는 이미지를 학습해서 특정 인물을 만들고, 대화 스타일도 학습시켜서 접목 시킨다면 정말 가상의 특정한 성격을 가진 인물을 누구나 쉽게 만들 수 있게 된 것이다.
> 위와 같이 데이터를 모아두고 LangChain 같은걸 활용해서 GPT에게 장기기억 관련 데이터는 벡터 DB에서 가져오고 적절한 내용으로 답변을 제공해준다면, 정말 하나의 인격체와 대화하는듯한 효과를 낼 수도 있을것 같다.
> 미리 준비를 하고 내것으로 만들어두는 경험과 노력이 필요한 시점이 아닐까 생각해본다.
