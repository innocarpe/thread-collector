# 소재가 없을 땐 이렇게 하세요 - 꿀통 하나 풉니다

요즘 개인적으로 SaaS도 만들고 있고
디코에 말씀드렸다시피 B2B 쪽으로도 작업하고 있어서
많이 바쁩니다. 그래서 정보글을 많이 못 올렸네요.
​
암튼 각설하고,
​
"바이브 코딩으로 뭘 만들어야 하지?" 하시는 분들 계실거라 봅니다.
그래서 아이디어 제공해드리면 다음과 같습니다.

온비드(공매 경매 플랫폼) API 등
쓸 수 있는 것들 많습니다.
