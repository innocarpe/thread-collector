1. 국내 서비스에서 리눅스 서버 구매

2. 리눅스 서버에 접속해서 터미널로 클로드봇 설치
​
명령어는 Perplexity 에게 물어보면 잘 가르쳐줍니다.
어려우신 분들은 아래 참고하세요.
​
구매한 계정으로 로그인 후 아이피 확인하고
윈도우에서 윈도우 + R 해서 cmd 입력 후
                            
                        
                    
                    
                                
                    
                        
                            
                                
                                    ssh root@아이피 입력
                                
                            
                        
                    
                    
                                
                    
                        
                            
                                    
이어서 패스워드 입력창이 나오는데
거기서 가입 시 입력한 패스워드 입력하고 엔터하면 바로 접속 완료.
​
-서버 업데이트 및 node 설치 명령어
                            
                        
                    
                    
                                
                    
                        
                            
                                
                                    sudo apt update && sudo apt upgrade -y
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt install -y nodejs
node --version  # v22.x.x 확인

                                
                            
                        
                    
                    
                                
                    
                        
                            
                                    
-거기에 RAM 적을 경우 아래 옵션 진행(SSD를 메모리로 스왑하는 것)
                            
                        
                    
                    
                                
                    
                        
                            
                                
                                    sudo fallocate -l 2G /swapfile && sudo chmod 600 /swapfile && sudo mkswap /swapfile && sudo swapon /swapfile
echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab

                                
                            
                        
                    
                    
                                
                    
                        
                            
                                    
-이어서 설치
                            
                        
                    
                    
                                
                    
                        
                            
                                
                                    apt install -y wget
wget -qO- https://clawd.bot/install.sh | bash

                                
                            
                        
                    
                    
                                
                    
                        
                            
                                    
이어서 온보딩 진행.
​
그리고 AI 연결 나오면 URL 복사 후 로그인하시고
로그인 끝나면 나오는 URL를 다시 터미널에 복붙하면 auth0로 로그인 완료.
(API면 API키만 입력)
