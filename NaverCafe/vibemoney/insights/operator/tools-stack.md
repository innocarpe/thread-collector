# 운영자 사용/추천 기술 스택 정리

---

## AI 코딩 툴

### 안티그래비티 (Google IDX / Project IDX)
- **언급 맥락**: 현재 주력 코딩 툴. "원래 Cursor 사용했으나 제미나이 무료 이용권 때문에 안티그래비티로 전환" (2025-12-16). 결혼 박람회 사이트, 토스 미니앱, CPA 랜딩페이지 등 대부분의 프로젝트에 사용.
- **무료 여부**: 구글 계정 기반 무료 사용 가능. 학생 프로계정 레퍼럴로 2개 계정 무료 운영 가능.
- **실제 사용 난이도**: 낮음 (초보 가능). 단, 토스 미니앱처럼 특정 프레임워크 환경에서는 환경 설정 에러가 많이 발생한다고 직접 언급.
- **팁**: 제미나이 구독 후 부계 레퍼럴로 4개월 무료 이용권 발급 가능 ("4개월 전 계정 정리하면 무료").

### 오픈클로 (Open Claude / 클로드봇)
- **언급 맥락**: 스레드 자동화 및 서비스 수정/배포에 주로 사용 (2026-02-13). "맥미니 없이" 리눅스 서버에서 돌리는 방법 상세 공개.
- **무료 여부**: 자체는 유료 (리눅스 서버 비용 별도). GitHub Copilot(월 10달러) 연결 시 GPT-5-mini 무제한 사용 가능.
- **실제 사용 난이도**: 중간. 리눅스 서버 세팅, SSH 접속, 텔레그램 봇 연결 등 기본 CLI 지식 필요.
- **추천 대안 모델**: z.ai의 GLM-4.7(Light, 10달러) 또는 GLM-5(Pro) — "2억 토큰, 클로드 대비 200~300만원 수준의 양".

### Cursor
- **언급 맥락**: 안티그래비티 이전에 사용. 현재는 "제미나이 무료 이용권이 꿀이라 지움" (2025-12-16).
- **무료 여부**: 무료 플랜 있음, 프로는 유료.
- **실제 사용 난이도**: 낮음. 안티그래비티와 유사.

### Perplexity
- **언급 맥락**: "AI 하면 딱 Perplexity랑 안티그래비티 2개 씀". 외주 기획 문서 작성 시 데이터 수집 용도로 필수 사용. SNS 데이터까지 긁어오는 '소셜' 기능 사용 (2025-12-16).
- **무료 여부**: 페이팔 연동으로 1년 무료 혜택 있었음 (2025-12-16 기준, 현재 유효 여부 불명). 정상가 3만원/월.
- **실제 사용 난이도**: 낮음. 검색형 AI.

## 호스팅 / 배포

### Netlify
- **언급 맥락**: 배포 서비스로 주력 사용 (2025-12-16). GitHub 연동 시 자동 배포.
- **무료 여부**: 무료 플랜으로 충분. DB(Neon) 연동도 무료.
- **실제 사용 난이도**: 낮음. GitHub 연결 후 자동화되면 "자동 업데이트". 클라이언트 계정으로 GitHub 연결해서 납품하는 방식 사용.

### iwinv
- **언급 맥락**: 오픈클로 돌리기 위한 리눅스 서버. "한국 IP 중 가장 저렴" (2026-02-13). 월 2만원 초반대.
- **무료 여부**: 유료.
- **실제 사용 난이도**: 중간. SSH 접속 기본 지식 필요.
- **경쟁 비교**: 카페24(오픈클로용 상품 소진, 이후 성능·가성비 쓰레기 평가), ultahost(해외, 2만원 후반대), Contabo(일본 로케이션, 고성능·최저가).

### 카페24
- **언급 맥락**: 회원이 전화상담 예약 사이트를 카페24 템플릿으로 만들어 250만원에 매각한 사례 소개 (2026-03-07). 오픈클로 서버용으로는 비추천.
- **무료 여부**: 유료 (템플릿 기반 유료 서비스).

## DB

### Neon
- **언급 맥락**: Netlify와 함께 무료 DB 솔루션으로 사용 (2025-12-16). "DB 사용도 무료(Neon 계정 필)".
- **무료 여부**: 무료 플랜 있음.
- **실제 사용 난이도**: 낮음. Netlify와 자동 연동 가능.

## 결제

### 토스페이먼츠
- **언급 맥락**: 결제 모듈 외주의 핵심 상품. 공식 문서가 잘 되어 있어 AI에 복붙하면 빠르게 구현 가능하다고 설명 (2025-12-30).
- **무료 여부**: 가맹점 수수료 구조 (결제 건당 %).
- **실제 사용 난이도**: 낮음~중간. 공식 문서 기반 AI 프롬프트로 2시간 내 연동 가능하다는 것이 운영자 주장.

### 포트원 (아임포트)
- **언급 맥락**: 토스페이먼츠와 함께 국내 결제 모듈 외주 대상으로 언급 (2025-12-30).
- **무료 여부**: 가맹점 수수료 구조.

### 페이팔
- **언급 맥락**: 해외 대상 서비스 결제 모듈. "깃허브에서 코드 받아서 폴더에 넣고 분석 후 결제 모듈 붙여줘 하면 끝" (2025-12-30).
- **무료 여부**: 거래 수수료 구조.

## 외주 플랫폼

### 크몽
- **언급 맥락**: 수수료 비싸고 마케팅 시 손에 쥐는 게 적음. 후기/포트폴리오 축적 + 첫 고객 유치 용도로만 사용 (2025-12-16).
- **전략 팁**: 저렴하게 올려서 후기 쌓은 뒤 가격 올리고, 고객 연락처 확보 후 직접 거래로 전환.

### 위시켓
- **언급 맥락**: 수수료 낮고 큰 건 많음. 포트폴리오 충분히 쌓인 후 가격 절반으로 지원하는 전략 (2025-12-16).
- **전략 팁**: 위시켓에서 따낸 포트폴리오 누적 후 점진적으로 가격 인상.

## 자동화

### 매니챗
- **언급 맥락**: 릴스 댓글 → 자동 DM 발송 서비스. 국내의 소셜비즈와 비교 대상으로 언급 (2025-12-13).
- **무료 여부**: 유료 (소셜비즈보다 비쌈).
- **실제 사용 난이도**: 낮음. 템플릿 기반.

### 소셜비즈
- **언급 맥락**: 매니챗의 국내 후발주자. "조금 더 저렴하다" (2025-12-13). 주로 추천.
- **무료 여부**: 유료 (매니챗보다 저렴).
- **실제 사용 난이도**: 낮음.

### 오픈클로 (자동화 맥락)
- 별도 항목 참조. 스레드 계정 자동 포스팅 + 서비스 자동 배포에 사용.

### n8n
- **언급 맥락**: 직접 언급 없음. 그러나 운영자가 언급한 자동화 흐름(스레드 → 사이트 → DB → 결제)을 구성할 때 추가 자동화 레이어로 연결 가능한 툴. 카페 전체 글에서 언급되지는 않음.

## 기타

### 네임칩 (Namecheap)
- **언급 맥락**: 도메인 구매. "가비아 등 국내 서비스 대비 절반 가격. .ai 도메인도 저렴" (2025-12-16).
- **무료 여부**: 유료.
- **실제 사용 난이도**: 낮음. Netlify와 2~3분 설정으로 연결 가능 (배포 완전 적용은 24시간 소요).

### 공공데이터 API (data.go.kr)
- **언급 맥락**: 바이브코딩 아이디어 소재로 제안. 국토부 부동산 실거래가, 온비드(공매 경매) API 등 구체적 예시 (2026-02-04).
- **무료 여부**: 무료 (일부 키 발급 필요, 하루 호출량 제한).
- **실제 사용 난이도**: 중간. API 키 발급 1~2일, 호출량 제한 있음.

### GitHub / GitHub Copilot
- **언급 맥락**: 버전 관리 + 오픈클로 무제한 사용을 위한 Copilot 결제 (월 10달러). GPT-5-mini 무제한 사용 가능 (2026-02-13).
- **무료 여부**: GitHub 무료, Copilot은 월 10달러.

### 나노바나나
- **언급 맥락**: AI 이미지 생성 툴. 랜딩페이지 내 타로 카드 이미지 및 카툰 이미지 생성에 사용 (2025-12-12).
- **무료 여부**: 미공개.

### 타입캐스트 / AI 스튜디오
- **언급 맥락**: 릴스 제작 시 TTS(텍스트 음성 변환) 도구로 언급 (2025-12-13).
- **무료 여부**: 타입캐스트 유료, AI 스튜디오 유료.

### 텔레그램
- **언급 맥락**: 오픈클로 제어 인터페이스. 봇 만들어서 토큰 연결하면 리눅스 서버의 오픈클로를 텔레그램으로 조작 가능 (2026-02-13).
- **무료 여부**: 무료.

### Vite / React / TypeScript
- **언급 맥락**: 토스 미니앱 개발 스택. 안티그래비티 내에서 React+TypeScript+Vite로 프로젝트 초기화 (2026-01-19).
- **무료 여부**: 무료 (오픈소스).
