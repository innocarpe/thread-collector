### 4-1. 오픈클로 (Openclaw) 보안 이슈
가장 반복적인 불만. 최소 3개 이상 파일에서 언급.
- article_id 6338 (2026-03-24): "오픈클로 보안 이슈 있던데 궁금합니다"
- article_id 2286 (2026-02-08): "오픈클로 잘 쓰면 꽤 좋아보이던데 API 비용이 문제"
- article_id 2623 (2026-02-13): "오픈클로가 대세가 될까요? 보안 이슈가 많던데"
- QnA에서도: "오픈클로 뭐 시켜도 권한이 없다고 하는거" (article_id 4009)

오픈클로는 로컬 파일 접근 권한이 필요한 특성상 "보안 무섭다" vs "중요 자료 없는 PC에 설치하면 된다"로 의견 분리.

### 4-2. 안티그래비티 MCP 오류
- article_id 126 (2025-12-15): MCP 연결 시 채팅 자체가 중단되는 버그. 초기 입문자들의 첫 좌절 포인트.

### 4-3. Claude Code 토큰 한도 소진
- article_id 4014 (2026-03-01): "3만원짜리 결제해서 쓰는데 토큰 제한이 너무 빨리 온다"

- 토큰 소진 = Pro/Max 업그레이드 vs 다른 툴 병용의 분기점

### 4-4. 러버블(Lovable) 크레딧 소모
- article_id 2200 (2026-02-05): "러버블은 편의성 압도적이지만 크레딧이 엄청 쓰인다. 안티그래비티는 초보에게 오류가 많다"

## 5. 비용 관련

### 언급된 구독/비용 구조
| 항목 | 금액대 | 관련 글 |
|------|--------|---------|
| Claude Code Pro | ~30,000원/월 | article_id 4014 |
| ChatGPT Pro | ~300,000원/월 (풀가격) | article_id 6175 |
| ChatGPT Plus | ~20,000원/월 | 다수 |
| 안티그래비티 무료 플랜 | 무료 (제한 있음) | article_id 390 |
| 안티그래비티 가족 공유 | 사실상 무료 우회 | article_id 2276 |
| 오픈클로 | API 비용 종량제 | article_id 2286 |

### 핵심 비용 패턴
1. **"무료로 분위기 보다가 유료 전환"** 흐름이 지배적. 안티그래비티 무료 → 클로드 코드 유료 전환이 가장 일반적인 경로.
2. **가족 계정 공유 꿀팁** (article_id 2276): 구글 가족 계정으로 안티그래비티 Pro 공유 가능하다는 팁이 공유됨. 비용 절감 수요 높음.
3. **"API 직접 vs 유료 계정"** 고민 반복 (article_id 2658): 소량 사용자는 유료 계정 UI로, 대량 사용자는 API로 분기하는 구조 이해를 못하는 경우 많음.
4. **오픈클로 = 저비용 대안**으로 포지셔닝. z.ai Coding Plan Lite 같은 저렴한 모델 연결 시도 (article_id 6401).
5. GPT Pro 카카오 할인 종료 → 구독 지속 여부 고민층 상당수.

## 6. Temperstone 시사점

### 이 커뮤니티의 도구 사용 수준
- **전형적 프로필**: 코딩 비전공자, "스레드 보고 왔다", 안티그래비티나 클로드 코드 한 개 유료 구독 중, 배포는 Vercel 또는 토스 미니앱
- **기술 이해도**: Git/GitHub 개념 자체를 모르는 경우 다수 (article_id 227에서 GitHub 설명 필요성이 언급될 정도). MCP가 뭔지 모르면서 쓰는 사람 많음.
- **도구 선택 기준**: 커뮤니티에서 "추천받는 것" 우선. 합리적 비교보다는 "시바님이 쓰는 것", "여기서 많이 얘기되는 것" 위주.

### 기회 포인트
1. **도구 조합 가이드 수요 실재**: "커서 vs 클로드 vs 안티그래비티 뭐가 좋아요?" 류의 질문이 수십 건. 이 커뮤니티에 진짜 필요한 건 '내 상황에 맞는 조합 추천'이지 도구별 기능 설명이 아님. → **"내 수준/목적별 도구 조합 가이드"** 포맷의 콘텐츠 또는 기능.
2. **비용 최적화 니즈**: API vs 유료 계정 혼란, 가족 공유 꿀팁, 오픈클로 API 세팅 — 비용을 아끼면서도 성능을 유지하는 방법에 대한 명확한 가이드 없음.
3. **오픈클로 보안 불안 해소**: 오픈클로가 커뮤니티 내에서 점점 주목받고 있지만 보안 이슈로 진입 장벽 존재. 이 지점을 해소하는 콘텐츠/툴이 갭.
4. **토스 미니앱 출시 pipeline**: tools-ai 폴더에 토스 미니앱 홍보 글이 44건. "바이브코딩 → 토스 미니앱 출시"가 이 커뮤니티의 대표 성공 패턴으로 자리 잡음. 이 파이프라인을 지원하는 서비스(템플릿, 검증 도구, 체크리스트)에 수요 있음.
5. **초보자 온보딩 공백**: GitHub 사용법, MCP 개념, 토큰 한도 관리 등 "바이브코딩 시작 후 바로 맞닥뜨리는 기술 장벽"에 대한 체계적 설명이 없음. → 커뮤니티 자체보다 외부 콘텐츠/가이드 상품으로 포지셔닝 가능.
