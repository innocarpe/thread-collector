# shortsyoutuber 운영자가 사용/추천한 기술 스택

## 범위

- 운영자 글에서 실제 이름이 나온 도구와 플랫폼만 정리
- 개발 스택보다는 쇼츠 운영 스택에 가까움
- 요청 카테고리 중 언급이 전혀 없는 항목은 `언급 없음`으로 표기

## 총평

이 운영자의 실제 스택은 `YouTube Shorts + TikTok + SnapTik + 미리캠버스 + 네이버 카페 + AdSense`다. 개발 스택이나 SaaS 자동화 스택은 사실상 보이지 않고, 대부분이 콘텐츠 소싱, 편집, 배포, 브랜딩에 집중돼 있다. 이 중 실무적으로 참고할 만한 건 `YouTube Shorts`, `미리캠버스`, `카페 브랜딩`이고, 가장 경계해야 할 건 `SnapTik`을 통한 해외 짜집기 흐름이다.
