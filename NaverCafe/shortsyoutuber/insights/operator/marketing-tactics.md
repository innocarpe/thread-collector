# shortsyoutuber 운영자의 마케팅 전술 분석

## 1. 콘텐츠 → 판매 퍼널 구조

운영자의 구조는 노골적인 즉시 판매형보다 `무료 정보 연재형 예열 퍼널`에 가깝다.

1. 카페에 무료 정보 글을 연재
2. 글 제목을 `1번 강의`, `4번 강의`, `5번 강의`처럼 커리큘럼화
3. 수익 인증과 운영 사례로 권위 확보
4. 질문 페이지와 댓글로 반복 접점 생성
5. 개인과외/오프라인 컨설팅으로 선현금화
6. 개인 유튜브 채널 브랜딩 후 강의 출시 준비

근거 문구:

- `교육용 쇼츠 채널을 개설했습니다`
- `카페엔 Q/A 전용 게시글이랑 정보 글이 올라와있습니다`
- `강의 연락은 정말 많이오고 미팅도 했습니다`
- `개인 유튜브 채널로 브랜딩화를 끝내고 강의를 진행하려던 계획`
- `나중에 교육할때 신뢰성과 기대를 높히기 위함입니다`

## 2. 후킹 전술 목록

### 수익 숫자 후킹

- `한달 수익 4~6000정도 벌고 있습니다`
- `한달 유효조회수 1000만`
- `달에 6~700정도`
- `월100만원 200만원은 정말 쉽습니다`
- `월 400이상은 가능합니다`

효과:

- 숫자를 먼저 제시해 초보자의 관심을 빠르게 잡음
- 검증보다 기대수익 상상을 먼저 유도함

평가:

- 가장 강한 후킹이지만 과장 가능성도 가장 큼

### 쉬운 공식 프레이밍

- `쇼츠는 처음부터 떡상하는 공식이 정해져있습니다`
- `양산형 템플릿들은 쇼츠에서 필수입니다`
- `주제 확장은 필수입니다`

효과:

- 복잡한 알고리즘 문제를 간단한 공식처럼 보이게 만듦
- 초보자는 `이 사람만 따라가면 된다`는 느낌을 받기 쉬움

평가:

- 콘텐츠 소비율은 높이지만 현실 난이도를 숨김

### 적대 대상 설정

- `인터넷 강의팔이나 채널팔이`
- `결론:강의팔이는 몇몇빼면 대부분 사기다`
- `이득 아닙니다. 하지마세요`

효과:

- 자신을 `사기꾼 반대편`에 위치시켜 신뢰를 빠르게 획득
- 독자가 방어심리를 풀고 내부자처럼 받아들이게 만듦

평가:

- 매우 잘 먹히는 방식이지만, 동시에 자기 판매를 위한 정화 장치 역할도 함

### 내부자 포지셔닝

- `저한테도 강의팔이 연락 많이옵니다`
- `문제있는 업체는 거르게 알려주시면 감사하겠습니다`
- `오프라인 시간당 20받습니다`
- `월 200이였습니다만 이제는 정리중입니다`

효과:

- 단순 정보 제공자가 아니라 실제 시장 안에서 거래되는 사람처럼 보이게 함

평가:

- 신뢰 형성에는 유효, 동시에 고가 서비스 가격 정당화 장치로도 작동

### 시리즈화

- `관련강의 4번 양산형 템플릿`
- `5번 강의에서 말한 케이스`
- `오늘 강의는 법적으로 민감한 소재`

효과:

- 게시글을 개별 팁이 아니라 코스처럼 소비하게 만듦
- 다음 글, 이전 글 클릭을 유도함

평가:

- 무료 콘텐츠를 교육 상품 전 단계로 묶는 데 효과적

## 3. 신뢰 구축 방법

### 수익 인증

- `최근 애드센스 내역` 이미지 삽입
- `어느정도 신뢰성은 가질수 있다고 판단해서 현재 수익을 보여드렸습니다`

평가:

- 가장 직접적인 신뢰 장치
- 다만 원본 검증이 안 돼서 절대 증거는 아님

### 실전 사례 제시

- `3일차`, `5일차`, `6일차`
- `8월경 만들었던 다른채널`
- `평균 조회수 100만`

평가:

- 이론보다 현장 경험 있는 사람처럼 보이게 만듦
- 숫자와 스크린샷은 강하지만 재현 근거는 약함

### 댓글/Q&A 개방

- `질문있으시면 Q/A 게시글에 댓글 달아주세요`
- 질문 페이지와 공지 페이지 운영

평가:

- 커뮤니티 체류를 늘리고 문의 전환 가능성을 높임

### 가격 공개

- `오프라인 시간당 20받습니다`
- `월 200이였습니다`

평가:

- 가격을 숨기지 않아 오히려 진짜처럼 보이는 효과가 있음
- 동시에 `고가여도 팔리는 사람` 이미지가 강화됨

## 4. SNS 트래픽 → 카페 → 강의 연결 구조

현재 자료 기준으로 보이는 연결 구조는 아래와 같다.

1. 외부 SNS/유튜브에서 관심층 확보
   - 운영자는 직접 `개인 유튜브 채널` 브랜딩을 언급
   - 사기꾼 사례 설명에서도 `쓰레드 글, 광고, 유튜브 컨텐츠`를 신뢰 확보 경로로 제시
2. 카페에서 무료 정보와 강의형 게시글 제공
3. 질문 페이지, 댓글, 개인문의로 리드 축적
4. 오프라인 컨설팅/개인과외로 고관여 고객 선별
5. 브랜딩이 충분해지면 강의로 전환

즉, `SNS/유튜브에서 호기심 생성 → 카페에서 신뢰 강화 → 문의/상담 → 유료 교육` 흐름이다.

## 5. 참고할 부분 / 피해야 할 부분

### 참고할 부분

- 무료 콘텐츠를 시리즈로 묶어 권위를 만드는 방식
- 숫자, 사례, Q&A를 섞어 신뢰를 만드는 구조
- 질문 창구를 별도로 열어 커뮤니티 허브를 만드는 운영법
- 경쟁 강의를 비판하며 자신의 포지션을 선명하게 만드는 전략

### 피해야 할 부분

- `월 100~200은 정말 쉽다` 같은 쉬운 돈 프레임
- 검증이 어려운 고수익 수치를 전면 배치하는 방식
- 해외 짜집기, 공정사용 과신처럼 리스크를 축소하는 메시지
- 공식 근거 없는 `채널점수론`을 확정적 사실처럼 말하는 방식

## 총평

운영자의 마케팅은 거칠지만 구조는 분명하다. 핵심은 `무료 정보 + 수익 인증 + 사기 비판 + Q&A 유도 + 고가 상담 + 향후 강의`다. 실전 마케팅 관점에서는 잘 짜여 있지만, 독자 입장에서는 신뢰 형성과 과장 카피가 섞여 있다는 점을 분리해서 봐야 한다.
